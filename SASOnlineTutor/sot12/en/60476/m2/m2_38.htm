Submit a SAS program that contains an invalid option, and resolve the error.

  1. Clear the Editor, Log, and Output windows.

  2. Copy and paste or type the program below into the code editing window. 
    data work.billing;
   set sasuser.insure;
run;
proc print data=work.billing keylabel;
   label total='Total Balance'
         balancedue= 'Balance Due';
run;

  3. Submit the program and check for errors.

  4. Change the KEYLABEL option in the PROC PRINT statement to LABEL.

  5. Clear the Log window and resubmit the program.

  6. Scroll through the output and verify that the revised program ran correctly.
