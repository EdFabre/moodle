Edit and resubmit a SAS program. Then, view the results.

  1. If necessary, clear the Editor, Log, and Output windows. Copy and paste the following PROC FREQ step into the code editing window: 
    proc frq data=sasuser.heart order=freq;
   table sex*survive*shock / 
         nopercent list;
run;
  2. Submit the program.

  3. Check for errors.

  4. Correct any errors.

  5. Resubmit the program.

  6. View the output created by the program.

  7. Verify that the program ran correctly.
