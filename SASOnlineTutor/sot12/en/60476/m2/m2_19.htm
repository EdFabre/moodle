Edit and submit two SAS programs.

You must complete this practice in order to do the next practice in this lesson.

  1. Open the Program Editor window (or activate it if it is already open). If necessary, clear it by selecting EditClear All.

    If you are using SAS in the Windows operating environment and the Program Editor window is not specified as the SAS editor, set your preferences to make it the editor.

  2. Copy the following program and paste it into the Program Editor window: 
    data sasuser.admit2;
set sasuser.admit;run;
    Indent the statements as necessary to enhance the program's readability. Then submit the program.

  3. Copy the following program and paste it into the Program Editor window: 
    proc print data=sasuser.admit2;
run;
  4. Add the following statement between the PROC PRINT statement and the RUN statement:
    var id name actlevel fee;
  5. Submit the program.
