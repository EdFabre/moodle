Edit the SAS statements that you submitted previously. Then save, submit, and include the revised program.

  1. View both of the programs you submitted during the previous practice for the Editor window or the Program Editor window.

  2. Delete the VAR statement that precedes the final RUN statement.

  3. Save the program as program1.sas in the My SAS Files directory (or in your default storage location). In the CMS operating environment, specify program sas in quotation marks as the filename and filetype.

    Remember where you stored this file.

  4. Submit the revised program.

  5. Clear the Editor or Program Editor window.

  6. Use the Open window to include the stored program into the Editor or Program Editor window.
