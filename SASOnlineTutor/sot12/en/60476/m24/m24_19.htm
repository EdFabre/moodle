View the value of the YEARCUTOFF= system option.

  1. Write and submit a PROC OPTIONS statement to check the value of the YEARCUTOFF= system option.

  2. View the current value of the option.
  1. Open the SAS System Options window.

  2. Find the current value for the YEARCUTOFF= system option.
  1. Write and submit the following OPTIONS procedure: 
    proc options option=yearcutoff;
run;
  2. In the Project window, double-click the log. The value of the YEARCUTOFF= option is written in the log.
  1. From the SAS menu, select ToolsOptionsSystem.

  2. Open the Input Control folder and double-click Data Processing. View the YEARCUTOFF= settings, and close the Options window when done.

    NOTE: If you change the YEARCUTOFF= option at this time, be sure to reset it to the original value before proceeding.
