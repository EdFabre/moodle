Create a user-defined format.

You must complete this practice in order to do the other practices in this lesson.

  1. Write a LIBNAME statement to assign the libref Library to the SAS library of your choice. 
    libname library 'SAS-library';
  2. Submit the LIBNAME statement and check the log to verify that the libref was assigned.

  3. Write a PROC FORMAT step that creates a permanent format named $ITEMFMT, whose values are: 
         C   Cassette
     R   Radio
     T   Television
  4. Submit the PROC step and check the log to verify that the format was created.
