Associating User-Defined Formats with Variables (continued)
Add a statement to the DATA step to associate the variable Sex with the format that is defined in the PROC FORMAT step. 

proc format lib=library;
   value sexfmt
         1='Female'
         2='Male';
run;
data perm.survey;
   infile perm.clinic1;
   input LastName $ 1-10 FirstName $ 12-22 Sex 24;
   
run;
