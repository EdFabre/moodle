Creating Multidimensional Arrays (continued)

Which element in the qtr array does qtr{i} reference when i equals 1?

     data finance.quarters(drop=i j);
        set finance.monthly;
        array m{4,3} month1-month12;
        array Qtr{4};
        do i=1 to 4;
          qtr{i}=0;
           do j=1 to 3;
              qtr{i}+m{i,j};
           end;
        end;
     run;
