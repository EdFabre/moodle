Creating Multidimensional Arrays (continued)

When the nested DO loop executes for the first time, the index variable j is set to 1. Which array element is referenced by m{i,j}?

data finance.quarters(drop=i j);   
   set finance.monthly;
   array m{4,3} month1-month12;
   array Qtr{4};
   do i=1 to 4;
     qtr{i}=0;
      do j=1 to 3;
         qtr{i}+m{i,j};
      end;
   end;
run;
