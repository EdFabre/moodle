Sequential Processing of Multiple Records in the DATA Step

This animation shows the sequential processing of multiple records in the raw data file Memdata.

  1. The following program is submitted to SAS:
    data perm.members; 
       infile memdata;
       input Fname $ Lname $ /
             Address $ 1-20 /
             City & $10. State $ Zip $;
    run;
    During compilation, the program data vector is created for the Perm.Member data set. When the DATA step executes, the values in the first record are read, and the / line pointer control moves the input pointer to the second record.

  2. The values for Address are read, and the second / line pointer control advances the input pointer to the third record.

  3. The values for City, State, and Zip are read, and the INPUT statement is completely executed.

  4. The values in the program data vector are written to the data set as the first observation.

  5. Control returns to the top of the DATA step, and the variable values are reinitialized to missing.

  6. During the second iteration, values for Fname and Lname are read beginning in column 1 of the fourth record.

  7. The values for Address are read and the / line pointer control advances the input pointer to the fifth record.

  8. The values for City, State, and Zip are read, and the INPUT statement is completely executed.

  9. The values in the program data vector are written to the data set as the second observation.
