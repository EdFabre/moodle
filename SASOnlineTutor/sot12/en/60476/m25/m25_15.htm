Read multiple records non-sequentially to create one observation, using the raw data file to which the fileref Personel has been assigned.

  1. Create Sasuser.Emplist2 from the raw data file that is referenced by the fileref Personel. Use the #n line pointer to read the variables in this order: Department, ID, Name, JobCode, and Salary.

  2. Use the PRINT procedure to display the data set.
