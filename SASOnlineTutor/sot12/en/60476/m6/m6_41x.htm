1.  

 Which SAS statement below invokes a SAS/GRAPH procedure to produce a plot using a SAS data set named Year3 in a SAS data library to which the libref Stats has been assigned?

a. 
proc gchart data=stats.year3;
b. 
proc plot data=stats.year3;
c. 
proc gchart data=year3.stats;
d. 
proc gplot data=stats.year3;


The PROC GPLOT statement invokes the GPLOT procedure, which you can use to produce either single plots or overlaid plots. To reference a file, you use a two-level name. The first part of the name is the libref of the SAS data library in which the file is stored, and the second part is the name of the file itself. 

2.  

 Select the statement below that produces a single plot that has the values of SysBP on the vertical axis and the values of Month on the horizontal axis. The vertical axis should be ordered from 40 to 240 by 20, and the horizontal axis should display only JAN, APRIL, JULY, and OCT.

a. 
plot sysbp*month / vaxis=40 to 240 by 20
     haxis=jan april july october;
b. 
gplot sysbp*month / vaxis=40 to 240 by 20
      haxis='JAN' 'APRIL' 'JULY' 'OCT';
c. 
plot month*sysbp / haxis='JAN' 'APRIL' 'JULY' 'OCT'
     vaxis=40 60 80 100 120 140 160 180 200 220 240;
d. 
plot sysbp*month / vaxis=40 to 240 by 20
     haxis='JAN' 'APRIL' 'JULY' 'OCT';


The PLOT statement specifies the variables to be plotted (with the variable on the vertical axis listed first), followed by the VAXIS= and HAXIS= options. You can specify tick-mark values for the axes either by listing the values (character values must be enclosed in quotation marks) or by specifying a range of values. 


3.  

 Select the statement below that produces two plots on a single pair of axes. The variables Sales and Rentals should be plotted against Year, and the vertical axis should be ordered from 0 to 10000 by 10.

a. 
plot (sales rentals)*year /
     vaxis=0 to 10000 by 10;
b. 
plot sales*year rentals*year /
     vaxis =0 to 10000 by 10 overlay;
c. 
plot sales rentals*year /
     vaxis=0 to 10000 by 10 overlay;
d. 
plot sales*year rentals*year /
     vaxis=0 to 10000 by 10 / overlay;


The PLOT statement specifies the variables to be plotted, followed by the options OVERLAY to specify multiple plots and VAXIS= to specify how the vertical axis is ordered.

5.  

 Which of the following values of the INTERPOL= option connects the points on a plot with a smoothed line?

a. SPLINE
b. NEEDLE
c. JOIN
d. CONNECT


The SPLINE value connects the points on a plot with a smoothed line. 

6.  

 Assume that the current color list for your device consists of five colors. Select the description of the plot line for DiasBP that would result from the program below. 
symbol1 interpol=spline value=square width=2;
symbol2 interpol=join value=triangle width=1.5;
proc gplot data=clinic.admit;
   plot sysbp*week diasbp*week / overlay;
run;

a. Straight lines with a width of 1.5 connecting triangular plotting symbols.
b. A curved line with a width of 1.5 connecting triangular plotting symbols.
c. A curved line with a width of 2 connecting square plotting symbols.
d. Vertical lines with a width of 1.5 drawn from each triangular plotting symbol to the Y axis.


If you don't specify the COLOR= option in a SYMBOL statement, SAS/GRAPH software cycles that SYMBOL statement through the current list of colors for your device before using the next SYMBOL statement. In this case the SYMBOL1 statement was cycled through the first five colors in the list. The SYMBOL2 statement was not needed and was not used. 


7.  

Suppose you originally submitted the following statement:

symbol1 interpol=spline color=blue width=2 value=star;

Select the statement you would use to change only the plotting symbol for the first plot line in subsequent plots.

a. 
symbol2 interpol=spline color=blue
        width=2 value=square;
b. 
symbol2 value=square;
c. 
symbol1;
d. 
symbol1 value=square;


Once defined, SYMBOL statements remain in effect until you change them, cancel them, or end the SAS session. Within one SYMBOL statement, you can change the value of one option value without affecting the values of other options. 


8.  

 If you are using RUN-group processing in interactive mode, which of the following methods would NOT end the PROC PLOT procedure?

a. submitting a DATA step
b. submitting another RUN statement
c. submitting a QUIT statement
d. submitting another PROC step


Submitting another RUN statement would simply submit the code between the previous RUN statement and before the current RUN statement. In this case, there is no code other than the RUN statement to submit. Only a DATA step, a PROC step, or a QUIT statement ends the current RUN group.  


9.  

 Assume that the libref Shoes has already been assigned. Which statement stores the plot in a permanent catalog to which the libref Shoes has been assigned?

a. 
proc gplot data=tryon.shoes gout=shoes.boots;
b. 
proc gplot data=tryon.shoes out=shoes.boots;
c. 
proc gplot data=tryon shoes gout=boots.shoes;
d. both a and b


You use the GOUT= option with a two-level name to store a plot in a permanent catalog. 


10.  

 To view a permanently stored graph in a subsequent SAS session, which of the following windows can you use?

a. the GRAPH window
b. the Results window
c. the Explorer window
d. both a and c


You can use all of these windows to view graphs that have been created during your SAS session. However, you must use the Explorer window or the GRAPH window to view graphs that were stored in previous SAS sessions.
