Specifying Tick-Mark Values for the Axes (continued)

Range of Values

You can also specify a range of tick-mark values. Notice the default tick-mark values on the vertical axis in the plot below.


To specify tick-mark values from 0 to 100 in increments of 50, for example, you can use the VAXIS= option that is shown in the PROC GPLOT step below. 
     proc gplot data=clinic.therapy1999;
        plot aerclass*month / haxis='01' '06' '12'
             vaxis=0 to 100 by 50;
     run;

The modified vertical axis appears in the plot below.


When you specify a range of values, be sure to use increments that accommodate the smallest and largest data values to be plotted.
