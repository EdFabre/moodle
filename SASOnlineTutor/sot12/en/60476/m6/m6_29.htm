Selecting Observations

In this lesson so far, you've used all of the observations in a data set in your plots. Alternatively, you can use a WHERE statement in your PROC GPLOT step to select a subset of the observations.

If you completed the lesson on PROC PRINT or PROC TABULATE, you learned to use the WHERE statement to select observations that meet certain criteria. 


     proc print data=clinic.admit;
        where actlevel in ('HIGH','LOW'); 
     run;
     proc tabulate data=clinic.stress;
        var maxhr rechr resthr;
        table max*(maxhr rechr resthr);
        where maxhr-resthr<=70; 
     run;
The WHERE statement works the same way with PROC GPLOT. For example, the following plot displays only observations for which the values of both Swim and AerClass exceed 35. To select these observations, you use this WHERE statement: 

     symbol1 interpol=spline value=none color=red;
     symbol2 interpol=spline value=none color=blue;
     proc gplot data=clinic.therapy1999;
        plot swim*month aerclass*month/overlay;
        where swim>35 and aerclass>35;
     run;
