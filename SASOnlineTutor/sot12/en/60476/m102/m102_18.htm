Create a list report.

If you have not already added the Marchflights data to your project, you must do so before you begin this practice.

Use the Marchflights data to create a list report that does the following:

  • groups data by the Destination variable

  • lists and displays totals for the variables PassengerCapacity, Boarded, Transferred, Nonrevenue, and EmptySeats

  • displays a custom title.

Click to view the report


  1. In the Project window, select Marchflights to make it the active data. Select AnalysisDescriptiveList Data.

  2. On the Columns tab, assign PassengerCapacity, Boarded, Transferred, Nonrevenue, and EmptySeats to the List variables role and to the Total by role.

  3. Assign Destination to the Group table by and Identifying label roles.

  4. Click the Options tab. Clear the Print the row number check box.

  5. Click the Titles tab. In the Section box, click Report Titles. In the Text box, delete the default title and type Passenger Counts for March.

  6. Click Finish to create the report.
