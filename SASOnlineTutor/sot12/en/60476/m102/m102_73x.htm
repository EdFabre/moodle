1.  

 To show the total and average sales for a region, which report would you use?

a. frequency report
b. summary report
c. list report
d. none of the above


Of the types of reports listed, only a summary report enables you to compute descriptive statistics such as averages (means). Frequency reports can calculate frequencies and percentages, and list reports can calculate totals (sums). 

2.  

 To which of the following result formats can you apply custom styles?

a. HTML
b. PDF
c. Text Output
d. all of the above


You can apply styles to various report formats, but custom styles can only be applied to HTML results.

8.  

 How are custom formats useful when you are creating reports?

a. Custom formats enable you to convert character strings to numeric values.
b. Using custom formats, you can control the written appearance of data values, as in converting a numeric code to a character string.
c. Custom formats permanently replace data values in a column with the formatted values.
d. Custom formats enable you to perform arithmetic operations on user-formatted numeric values.


Custom formats enable you to replace data values with character strings in any data set that contains the appropriate data values. Formats don't replace data values in a data set; they affect only the way that the values are displayed. User-defined formats always associate variable values with character values. If you attempt to perform arithmetic operations on user-formatted numeric values, the unformatted values are used.
