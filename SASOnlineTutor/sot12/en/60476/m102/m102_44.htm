Combining Reports (continued) Create RTF output for the Summary Tables task.
  1. In the Project window, right-click the first Summary Tables task under the ProductSales data (the summary report that summarizes data by category) and select Properties.

  2. In the Properties window, click the Results tab.


  3. Select Override the preferences set in Tools -> Options.

  4. In the Result formats area, select RTF. Leave HTML selected. You will produce both types of output when you run the task.

  5. Click the Report style drop-down box and select Rtf.


Click to view the full window.

  1. Click OK.

  2. In the Project window, right-click the Summary Tables task again and select Run this task. In the message window, click Yes.

  3. When the RTF file is created, a File Download window might open, prompting you to either open or save the file. Click Save. In the Save As window, name the file BoardReport.rtf and select a location where you want to save the file. Click Save.

  4. Close the window that has the action cancelled message.

Next, you open the RTF file in a word processing program.



  1. In the Project window, right-click the first Summary Tables task under the ProductSales data (the summary report that summarizes data by category) and select Properties.

  2. In the selection pane, click Results.


  3. Select Override the preferences set in Tools -> Options.

  4. In the Results Formats area, select RTF. Leave HTML selected. You will produce both types of output when you run the task.

  5. Notice that the default style for RTF results is Rtf.


Click to view the full window.

  1. Click OK.

  2. In the Project window, right-click the Summary Tables task again and select Run this task. In the message window, click Yes.

  3. In the Project window, right-click the RTF file and select Export. In the Export ProductSales window, select Local Computer and select a location where you want to save the file. Name the file BoardReport.rtf and click Save.

    If you have Microsoft Word installed, you could also open the results directly by right-clicking the RTF icon and selecting Open with Microsoft Word.
