Creating and Applying Custom Formats (continued) Create a format.
  1. Select ToolsCreate Format.

  2. On the Options tab, you define the general attributes of the new format. In the Format name box, type origin. Leave the Use default check box selected.

    Format names for numeric formats are limited to 8 characters. Format names for character formats are limited to 7 characters because they are automatically appended with a dollar sign ($).

  3. In the Scope area, select Permanent. This makes the format available to data in other projects, and it remains available after you have ended your SAS Enterprise Guide session.

  4. In the Format type area, select Character.

Click to view the full window.

Next, you assign labels and values for the format.
