Create a frequency report.

Use the Marchflights data to create a frequency report that shows the distribution of flights by destination. Add a custom title to the report.



Click to view the report

  1. In the Project window, select Marchflights to make it the active data. Select AnalysisDescriptiveOne-Way Frequencies.

  2. On the Columns tab, assign Destination to the Analysis variables role.

  3. Click the Statistics tab. Select Frequencies and Percentages.

  4. Click the Titles tab. In the Section box, select One-Way Frequencies. In the Text box, delete the default title and type Count of Flights by Destination.

  5. Click Finish to create the report.

Use the Marchflights data to create a frequency report that shows the distribution of flights by destination. Add a custom title to the report.



Click to view the report

  1. In the Project window, select Marchflights to make it the active data. Select DescribeOne-Way Frequencies.

  2. Click Task Roles in the selection pane. Assign Destination to the Analysis variables role.

  3. Click Statistics in the selection pane. Select Frequencies and Percentages.

  4. Click Titles in the selection pane. In the Section box, select Analysis. Clear the Use default text check box and delete the default title. Type Count of Flights by Destination.

  5. Click Run to create the report.
