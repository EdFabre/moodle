Create and apply a custom format.

Create a permanent character format named $dest that can be applied to the Destination variable. The format should apply the labels to discrete values as follows:

Label
Values
Domestic DFW, LAX, ORD, HNL
International CPH, FRA, LHR, HND

After you create the format, reopen the bar chart task for the Marchflights data and apply the format to the Destination variable. Do not overwrite the previous bar chart task when you run the task. You will create a bar chart that shows the total number of empty seats for formatted values of Destination.


Click to view the report


  1. Select ToolsCreate Format.

  2. In the Format name box, type dest. In the Scope area, click Permanent. In the Format type area, click Character.

  3. Click the Define Format tab.

  4. Click New Label. In the Label box, type Domestic. Click New Range. Under Type, select Discrete. In the Values box, type DFW. Click New Range. In the Values box, type LAX. Click New Range. In the Values box, type ORD. Click New Range. In the Values box, type HNL.

  5. Click New Label. In the Label box, type International. Click New Range. Click the Values drop-down list and select All other values.

  6. Click Finish to create the new format.

  7. In the Project window, double-click the Bar Chart for the Marchsales data to open the task.

  8. Click the Columns tab. Under the Column to Chart role, right-click Destination and select Properties.

  9. In the Properties window, click the ellipsis button next to the Format box. In the Categories box, select User Defined. In the Formats box, select $DEST. Click OK. Click OK in the Properties window.

  10. Click Finish to create the bar chart. In the message window, click No.

Click to view the report

  1. Select DataCreate Format.

  2. In the Format name box, type dest. In the Location to store format area, select the drop-down list under Library and select Sasuser. In the Format type area, click Character.

  3. Click Define formats in the selection pane.

  4. Click New Label. In the Label box, type Domestic. Click New Range. Under Type, select Discrete. In the Values box, type DFW. Click New Range. In the Values box, type LAX. Click New Range. In the Values box, type ORD. Click New Range. In the Values box, type HNL.

  5. Click New Label. In the Label box, type International. Click New Range. Click the Values drop-down list and select All other values.

  6. Click Run to create the new format.

  7. In the Project window, double-click the Bar Chart for the Marchsales data to open the task.

  8. Click Task Roles in the selection pane. Under the Column to Chart role, right-click Destination and select Properties.

  9. In the Properties window, click the Change button next to Format. In the Categories box, select User Defined. In the Formats box, select $DEST. Click OK. Click OK in the Properties window.

  10. Click Run to create the bar chart. In the message window, click No.
