Using Styles to Customize HTML Reports (continued) Apply the new style to a report.
  1. In the Project window, right-click the first Summary Tables task under ProductSales (the one that summarizes profit by category) and select Properties from the pop-up menu.

  2. Click the Results tab. The last time you ran this task you created RTF output. In the Properties window, clear the RTF check box. Click the Report styles drop-down list and select MyEditedStyle. Click OK.

Click to view the full window.

  1. In the Project window, right-click the Summary Tables task again and select Run this task. In the message window, click Yes.

    Your results are created using the MyEditedStyle style.

Click to view the full window.

Now try saving, editing, and applying a style on your own.
