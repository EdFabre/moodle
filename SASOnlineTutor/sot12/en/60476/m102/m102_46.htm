Combining Results in an HTML Document

So far you've created a Microsoft Word document that combines SAS Enterprise Guide results. You can also use the Document Builder to create one HTML file from several HTML results that are produced in SAS Enterprise Guide. To customize the appearance of your HTML file, you can select a style to apply to the document.

After you have the results that you want in one HTML file, you can edit the HTML file to add titles, text, and so on.





Let's see how you create the HTML file.
