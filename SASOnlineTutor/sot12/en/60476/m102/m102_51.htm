Using Styles to Customize HTML Reports (continued) Modify text attributes in a style.
  1. Let's change the font style and size first. In the preview area of the Edit Style window, click the SAS System Title text.

  2. The Text tab is selected by default. For Font, type Tahoma or select Tahoma from the list. Notice that the preview area is updated as you make changes.

  3. For Font style, select Bold. For Size, select 14.

Click to view the full window.

  1. Now let's change the font color. You need to specify a custom color for the titles. Click the Foreground arrow under Colors. At the bottom of the pop-up palette, click Custom Colors.

  2. At the bottom right, type these values for Red, Green, and Blue:

    Red: 54
    Green: 98
    Blue: 174

    Click Add to Custom Colors. Then click OK.

Click to view the full window.

  1. In the preview area, click the SAS Procedure Title text. For Font, type Tahoma or select Tahoma from the list.

  2. Click the Foreground arrow under Colors. Click the new custom color (the last color in the palette).

  3. Finally, let's change the footnote text color. Click the SAS System Footnote text. Click the Foreground arrow under Colors and click black (the first color in the palette).

  1. Let's change the font style and size first. In the preview area of the Style Editor window, click the SAS System Title text.

  2. The Text tab is selected by default. For Font, type Tahoma. You can also click Browse next to the Selected fonts box. This opens the Browse available fonts window. In the Selected fonts box, select arial and click the arrow pointing to the left to remove this font. Repeat this with any remaining selected fonts. Then, in the Available fonts list, scroll down and select Tahoma and click the arrow pointing to the right. This adds Tahoma to your selected fonts list. Click OK. Notice that the preview area is updated as you make changes.
