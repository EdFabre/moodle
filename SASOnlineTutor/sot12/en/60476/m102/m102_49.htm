Using Styles to Customize HTML Reports

So far in this lesson, you've seen how you can create different types of reports. You've also learned to select a default style for your results by using the Options window and the Style Manager window.

To further customize the look of your HTML reports, you can also do the following:

  • modify the predefined styles that SAS Enterprise Guide provides for you
  • create your own new styles from existing styles
  • add an external style that resides on a Web server or elsewhere on your network.


Customizing Styles in the Edit Style Window

In SAS Enterprise Guide, you use the Edit Style window to modify and create styles. In the Edit Style window, you can select style elements graphically or from a drop-down list. To customize styles, you can define text and border attributes, specify background and border images, and create your own custom properties. As you edit styles, the Edit Style window displays your changes graphically.
