Setting Options for Graphs (continued) Work with the ActiveX graph.
  1. Select ToolsOptions. Click the Results tab. Look at the default image format. Is it set to ActiveX?


    • Yes: The bar chart in your project is already an ActiveX object. Click OK to close the Options window.


    • No: Click the Image format drop-down box and select ActiveX. Click OK to close the Options window. In the Project window, right-click Bar Chart and select Run this task. In the message window, click Yes.

  2. In the Project window, double-click HTML under the Bar chart task if the chart is not already open. Right-click the bar chart and select Chart TypePie. The bar chart changes to a pie chart.

  3. Right-click any segment of the pie chart and select Pie Options.

  4. In the Labels area, click the Percent drop-down box and select Outside. This adds the percentage to the chart. Click the Slice drop-down box and select Inside. This positions the category names on the pie slices.


  1. Click the Font button. In the Font style box, select Bold. Click OK to close the Font window. Click OK to close the Options window.

  2. Right-click any segment of the pie chart and select View2D Chart.

  3. You have worked with the interactive features of the ActiveX graph to create a pie chart. If you close the HTML window now, these changes are not saved. Let's save this pie chart as an image file in JPG format.
