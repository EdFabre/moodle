Setting Options for Graphs
Earlier in this lesson, you learned that you can create reports in various formats. You can also create graphs in various image formats. If you haven't changed your graph settings, the bar chart that you just created is an ActiveX object. ActiveX is the default image format for SAS Enterprise Guide.

In the Options window, you can choose from six image formats to set as the default for the graphs that you generate. You can also specify a default size for all graphs.




Click to view the full window.


The format that you select depends on how you plan to use the graph. The following table has information about the image formats that are available in SAS Enterprise Guide.


Image Format
Description
Is Graph Interactive?
ActiveX Generates the graph as an ActiveX control. This is the default format and can be viewed on Microsoft Windows machines by using Microsoft Internet Explorer.
yes
Java Generates the graph as a Java applet. The graph can be viewed in any Web browser that supports Java.
yes
GIF Generates the graph as a GIF file.
no
JPG Generates the graph as a JPG file.
no
ActiveX image
(SAS 9)		 Generates the graph as a JPG file by using ActiveX technology. This format can be generated only by the SAS System for Windows server, Version 9 or later.
no
Java image
(SAS 9)		 Generates the graph as a JPG file by using Java technology. This format can be generated by any SAS System server, Version 9 or later.
no


There are additional options that you can set for various image formats. For example, if you are generating GIF or JPG format graphs, you can set options to include titles and footnotes as part of the image. To learn more about these additional options, click Help in the Options window.


Overriding the Default Image Format for an Individual Task

After you have chosen a default image format, you can change it for an individual task by right-clicking the task in the Project window and selecting Properties. You can select a different image format and then rerun the task to produce a graph in the selected format. You cannot override the image size that is specified in the Options window.
