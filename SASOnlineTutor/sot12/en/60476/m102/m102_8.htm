Setting Options for Results (continued) View default result formats and set a default report style.
  1. Select ToolsOptions.

  2. In the Options window, click the Results tab.

  3. Notice that HTML is the only result format that is selected by default. Let's leave this setting as it is for now. Later in this lesson, you'll generate other result formats for specific reports by changing the task properties.

Click to view the full window.

  1. Click Cancel to close the Options window.

  2. Select ToolsStyle Manager.

  3. If you have not changed the style in the past, it will be set to EGDefault. Select various styles in the Available styles list and view the style in the Preview area.

  4. Select the sasweb style.

    You can select any style that you want to use, but the results that are shown in this lesson use the sasweb style.

Click to view the full window.

  1. Click Set as Default. Then click OK.
