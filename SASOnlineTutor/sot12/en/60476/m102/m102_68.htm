Using Traffic Lighting in a Summary Table (continued) Create a summary report with the data that you want to view.
  1. In the Project window, right-click the first ProductSales Summary Table task and select Properties.

  2. Click the Results tab and clear the Override the preferences set in Tools->Options check box. Click OK.

Click to view the full window.

  1. Double-click the first summary report to open the task. You will modify this report because much of it is already created for you.

  2. Click the Table tab.

  3. In the Available variables list, select ProductName and drag and drop it in the Preview area just to the right of CategoryName. The new summary table will compute information for individual products within each category.

Click to view the full window.

  1. Click Finish. In the message window, click No.

  2. View the report. Summary information for each product is listed. Notice that at a glance it is difficult to get an idea of which products are doing well.

Click to view the full window.
