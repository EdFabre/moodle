Creating Graphs (continued) Select the chart type and assign variables to roles.
  1. In the Project window, select ProductSales to make it the active data. Select GraphBar from the main menu.

  2. On the Chart Gallery tab, select Horizontal Colored Bars.

Click to view the full window.

  1. Click the Columns tab.

  2. Assign CategoryName to the Column to chart role. The values of CategoryName will define the bars in the chart.

  3. Assign OrderProfit to the Sum by role. The values of OrderProfit will determine the length of the bars in the chart.

Click to view the full window.

  1. In the Sum by role, right-click OrderProfit and select Properties.

  2. In the Label box, type Profit.

  3. Click the ellipsis button next to the Format box. In the Formats window, change the value in the Decimal places box to 0. Then click OK.

Click to view the full window.

  1. Click OK in the Properties window.

Next, you specify appearance options for the chart.
