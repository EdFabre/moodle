Creating Summary Reports (continued) Assign variables to roles in the Summary Tables task.
  1. In the Project window, select ProductSales to make it the active data. Select AnalysisDescriptiveSummary Tables.

  2. On the Columns tab, assign OrderProfit and RetailOrders to the Analysis variables role. The table will have statistics calculated for the variables in this role.

  3. Assign CategoryName, CompanyNameLong, and ProductName to the Classification variables role. These variables will be the rows of the table.
