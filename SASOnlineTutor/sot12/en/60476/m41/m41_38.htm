Using the Sasuser.Heart data set, control the content of crosstabulation output.

  1. Write a program to create a crosstabulation of the variables Shock and Survive in the data set Sasuser.Heart.

  2. Submit the program and examine its output.

  3. Edit the TABLES statement, adding any combination of the NOFREQ, NOPERCENT, NOROW, and NOCOL options.

  4. Submit the program. Compare its output to the previous version.

  5. Experiment with the other options.
