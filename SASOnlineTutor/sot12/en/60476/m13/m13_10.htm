Referencing a DBMS Using the Explorer Window

You've seen that you can reference a DBMS using a SAS/ACCESS LIBNAME statement. Another way to associate a libref with a DBMS is to define a new library from the Explorer window. This method is useful when you don't need to store a SAS/ACCESS LIBNAME statement with your SAS program.

To open the window, select FileNew from the Explorer window. Then choose Library. In the New Library window, type a libref in the Name box and type or select an engine in the Engine box. Select Enable at startup if you'd like the libref to be assigned automatically whenever you start a SAS session.



When you specify an engine for a DBMS, under Library Information the New Library window displays
  • boxes for SAS/ACCESS engine connection options that are available for the engine. If appropriate, enter options to connect to your database, schema, or server.

  • an Options box for SAS/ACCESS LIBNAME options. Use blanks to separate multiple options.
