Referencing DBMS Objects in SAS Programs (continued)
Suppose you want to create a listing of the data in the DB2 table STAFF. Complete the PROC PRINT statement below by specifying the two-level name of the STAFF table.

SSID= is a DBMS-specific LIBNAME engine option that is not required. For details about this and other LIBNAME engine options, see the documentation for SAS/ACCESS. 

libname mydb2lib db2 ssid=db2;
proc print data=;
   where state = 'NJ';
run;
