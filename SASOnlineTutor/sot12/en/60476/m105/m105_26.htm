Simple Linear Regression (continued) Assign variables to roles.

In this task, you start the Linear Regression task and assign variables to roles.

  1. If the Concrete data is already in your project, select it to make it the active data set. If it is not in your project, select InsertData. In the Insert Data window, navigate to the location where you saved the practice data for this lesson. Double-click Concrete to add the data to the project.

  2. From the menu bar, select Analysis Regression Linear.

  3. On the Columns tab, assign Strength1 to the Dependent variables role. Assign Humidity to the Explanatory variables role.
