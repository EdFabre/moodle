Analysis of Covariance (continued) Start the task and assign variables to roles.

In this task, you start the Linear Models task and assign variables to roles.

  1. Select InsertData. In the Insert Data window, navigate to the location where you saved the practice data for this lesson. Click the Files of Type drop-down arrow and select Microsoft Excel Files (*.xls). Double-click Boards.xls, and then double-click boards$ to add the file to the project.

  2. Scroll the Tasks by Category list to the ANOVA category. Double-click Linear Models.

  3. Assign Strength to the Dependent variables role. Assign Density to the Quantitative variables role, and assign Type to the Classification variables role.
