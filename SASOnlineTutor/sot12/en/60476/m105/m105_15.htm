t-Tests (continued) Select the type of t-test and assign variables to roles.

In this task, you select the type of t-test that you want to run, then you assign variables to roles.

  1. If the Concrete data is already in your project, select it to make it the active data set. If it is not in your project, select InsertData. In the Insert Data window, navigate to the location where you saved the practice data for this lesson. Double-click Concrete to add the data to the project.

  2. Scroll the Tasks by Category list to the ANOVA category. Click t Test.

  3. On the t Test Type tab, select Paired.


Click to view the full window.


  1. Click the Columns tab. Assign the variables Strength1 and Strength2 to the Paired variables role.


Click to view the full window.


You have the option of setting the test statistic to be calculated for Strength1 - Strength2 or for Strength2 - Strength1. Either analysis produces the same p-value. Leave the analysis in the order in which you entered the variables.
