Recall that there is an additional variable in this table: the level of training of the employees. It is not reasonable to expect that the number of defective parts produced by a novice and the number of defective parts produced by an expert can realistically be used in the same control chart: they represent two distinct processes.

There are two options for dealing with grouping variables in SAS Enterprise Guide:

  • To produce separate charts for each level of a grouping variable, assign the grouping variable to the Group analysis by role.

  • To have both groups in the same chart and to label each group separately within the chart, assign the grouping variable to the Block variables role.

In this case, you want separate control limits for each group, so it is appropriate to assign the variable training to the Group analysis by role.
