In this task, you use the Distribution Analysis task to evaluate whether the residuals are approximately normally distributed.

  1. In the Project window, select the output data set you just created to make it the active data.

  2. From the menu bar, select Analysis Descriptive Distribution Analysis.

  3. On the Columns tab, assign residual_Strength1 to the Analysis variables role.
