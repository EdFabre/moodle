Two-Way Analysis of Variance (continued) Check the distribution of the residuals.

In this task you run the Distribution Analysis task to examine whether the residuals are normally and independently distributed.

  1. In the Project window, select the output data set you just created to make it active.

  2. From the menu bar, select Analysis Descriptive Distribution Analysis.

  3. On the Columns tab, assign residual_Strength1_ to the Analysis variables role.


Click to view the full window.


  1. Click the Distributions tab. Select Normal for the fitted distribution.

  2. Click the Plots tab. Select Histogram and Probability plot.


Click to view the full window.


  1. Click Finish.
