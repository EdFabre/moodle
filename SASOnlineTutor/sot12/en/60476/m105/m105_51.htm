Control Charts for Measurement Data (continued) Start the task and assign variables to roles.

In this task, you start the Mean and Range Chart task and assign variables to roles for the analysis.

  1. Select InsertData. In the Insert Data window, navigate to the location where you saved the practice data for this lesson. Double-click IceCream to add the data to the project.

  2. From the menu bar, select AnalysisControl ChartsMean and Range Chart.

  3. On the Columns tab, assign Temperature to the Process measurement role. Assign Sampleid to the Subgroup identifier role.
