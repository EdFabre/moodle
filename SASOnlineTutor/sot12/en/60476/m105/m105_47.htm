Analysis of Covariance (continued) Plot regression lines.

In this task, you plot the regression lines for each category. This allows you to visualize the relationship among the variables in this ANCOVA.

  1. In the Project window, select the Boards data to make it active.

  2. From the menu bar, select GraphLine.

  3. On the Chart Gallery tab, select Multiple line plots by group column.
