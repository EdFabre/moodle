Control Charts for Attribute Data (continued) Start the task and assign variables to roles.

In this task, you start the NP Chart task and assign variables to roles.

  1. Select InsertData. In the Insert Data window, navigate to the location where you saved the practice data for this lesson. Double-click Training to add the data to the project.

  2. From the menu bar, select AnalysisControl Chartsnp Chart.

  3. On the Columns tab, assign defects to the Number nonconforming role, assign hour to the Subgroup identifier role, and assign nsampled to the Subgroup sample size role.
