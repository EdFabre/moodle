Summary Statistics

Overview

Statistics can serve two purposes. They can describe the characteristics of a sample (descriptive statistics), and they can also be used to draw conclusions about the population from which the sample is drawn (inferential statistics). In this example, descriptive statistics help an airline company track the number of lost bags.

The baggage department needs a monthly report of the average number of lost bags arranged by Location and Operator. Data is collected in the Cargo data set, which has the following variables:


Variable Description
Obs unique identifier for the measurement occasion
Date date of the measurement
Location gate of the arriving flight
Operator primary employee on duty in charge of baggage handling
Time amount of time for the first bag to reach customers at baggage claim
Lostbags total number of baggage items that did not arrive with the flight


The Summary Statistics task allows you to create this type of report and select the statistics you want to generate.
