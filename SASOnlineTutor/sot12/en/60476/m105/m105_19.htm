In this task, you select plot options and run the analysis.

  1. Click the Plots tab.

  2. Under Types, select Means.

  3. Under Variance bar type, select Standard error of mean. Notice that the default height of the error bars is two standard error units. Leave the height option as it is.


Click to view the full window.


  1. Click Finish.
