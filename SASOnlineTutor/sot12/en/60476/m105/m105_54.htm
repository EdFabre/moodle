Control Charts for Attribute Data

Overview

Quality characteristics cannot easily be described by variable data, so these items are classified as

  • conforming if they operate within normal parameters
  • nonconforming if they do not operate within normal parameters

Attribute control charts attempt to detect when the number or proportion of nonconforming items is higher or lower than expected. P and NP charts use the binomial, or count, model. C and U charts use the Poisson, or event, model.

This example uses the Training data set, which contains information about the number of defective parts produced by each of 30 employees in a factory. Each employee produced 200 parts in an hour; the parts were measured and the number of defects recorded. Half the employees are experts who have completed a rigorous training regimen. The other half are novices who have not undergone the training regimen. The variables in the Training data set are as follows:


Variable Description
training operator skill level (Expert or Novice)
nsampled number of parts sampled
defects number of defective parts
hour hour of sampling in a sequence of 30 hours


The number of defects measured refers to the number of defective parts produced by the employee. Because this is count data and the sample sizes are equal, the NP and the P charts are both appropriate. In this example, you generate the NP chart.
