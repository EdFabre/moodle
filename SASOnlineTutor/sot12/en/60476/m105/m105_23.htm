One-Way Analysis of Variance (continued) Set options for the test and run the analysis.

In this task, you select the tests you want to run in the analysis.

  1. Click the Tests tab. This tab includes tests for the assumption of equal variances in ANOVA. The Welch ANOVA provides a nonparametric analysis that can be used in the event of unequal group variances. Select Welch's variance-weighted ANOVA and all three tests for equal variance.


Click to view the full window.


  1. Click the Means tab. The Comparisons button is selected. Under Methods to use, select Bonferroni t test and Tukey’s studentized range test (HSD).


Click to view the full window.


  1. On the Means tab, click Breakdown. Select Mean, Standard deviation, Standard error, and Number of observations.


Click to view the full window.


  1. Click Finish.
