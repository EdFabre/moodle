Create customized layouts with subtotals using Sasuser.Admit.

  1. Copy the program below and paste it into the code editing window. 
    proc print data=sasuser.admit;
   var age height weight fee;
   where age>30;
   sum fee;
run;
  2. Insert statements to

    • sort Sasuser.Admit by the variable ActLevel and create the temporary output data set Work.Activity

    • use the data in Work.Activity in the PRINT procedure

    • request subtotals for the variable Fee within each BY group of ActLevel

    • remove redundant BY variable headings

    • print BY groups on separate pages.

  3. Submit the program and view the output.
