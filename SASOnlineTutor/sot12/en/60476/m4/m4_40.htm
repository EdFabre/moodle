Before you complete this practice, be sure that you have Text output turned on.
Before you complete this practice, be sure that you have Text output turned on.

  1. Copy the program below and paste it into the code editing window: 
    proc print data=sasuser.admit;
   var age height weight fee;
   where age>30;
   sum fee;
run;
  2. Insert a statement to specify the title March Admissions on the first title line, and a statement to specify the title For Clinic 231 on the third title line.

  3. Submit the program and view both the HTML and text output.
