Using the data set Sasuser.Admit, label a variable and double space the output.

  1. Copy the program below and paste it into the code editing window: 
    proc print data=sasuser.admit;
   var age height weight fee;
   where age>30;
   sum fee;
run;
  2. Add one option to the PROC PRINT statement to display the label and another to double space the output. Insert a statement to assign the label Admission Fee to the variable Fee.
