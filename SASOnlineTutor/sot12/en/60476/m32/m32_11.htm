Using the Sasuser.Talent data set, reference an automatic macro variable.

Be sure to complete this practice. You'll use the output data set in later practices.

  1. Copy the program below and paste it into the code editing window. Submit the program and view the footnote that displays the date on which the SAS session began. 
    title;
footnote "Date: &sysdate9";
data sasuser.talent99;
   set sasuser.talent;
   if year(lasthired)=1999;
   format birthdate lasthired date.;
run;
proc print data=sasuser.talent99;
run; 

  2. Rewrite the FOOTNOTE statement to also display the day of the week on which the SAS session began. Submit the code and view the footnote again.

  3. Cancel the FOOTNOTE statement.
