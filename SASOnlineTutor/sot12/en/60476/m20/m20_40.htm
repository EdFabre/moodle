Excluding Unmatched Observations (continued)
Write a subsetting IF statement that selects observations for subsequent processing only if all three input data sets contributed to the current observation. 

     data mergdata.emppay;
        merge sales.reps(rename=(office=OfficeNumber) in=inreps)
              empinfo.sales(in=insales) 
              empinfo.bonuses(in=inbonus);
        by id;
        
     run;
