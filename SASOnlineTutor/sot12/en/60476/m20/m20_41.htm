Merge the sorted data sets Work.Adsort and Work.Strsort, and exclude unmatched observations.

Work.Adsort and Work.Strsort were created in a previous practice. If you haven't created these data sets, return to the practice and create them before continuing here.

  1. Copy the program below and paste it into the code editing window. Then submit the program. In the output, notice that two observations were unmatched in the input data sets and contain missing values for several variables. 
    data sasuser.merged;
   merge work.adsort
         (rename=(date=AdmitDate))
         work.strsort
         (rename=(date=VisitDate));
   by id;
run;
proc print data=sasuser.merged;
run;
