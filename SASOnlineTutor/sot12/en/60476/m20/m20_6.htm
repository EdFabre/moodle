Sort the input data sets Sasuser.Admitjune and Sasuser.Stresstest and perform one-to-one reading.

Be sure to complete this practice. You'll use the output data sets in later practices.

  1. Write two PROC SORT steps. In both, sort by the variable ID.

    In the first step, read the data set Sasuser.Admitjune and create the output data set Work.Adsort.

    In the second step, read the data set Sasuser.Stresstest and create the output data set Work.Strsort.

  2. Submit the program and view log messages about the PROC SORT steps. Were the sorted data sets created successfully? How many observations does each sorted data set contain?

  3. Print Work.Adsort and Work.Strsort and scroll through the output. Note the variables and the number of observations in each data set.

  4. Write a DATA step to create the data set Work.One2one by performing one-to-one reading of Work.Adsort and Work.Strsort. From Work.Adsort, read only the variables ID, Name, Sex, and Age. From Work.Strsort, read only the variables RestHR, MaxHR, and Tolerance.

  5. Submit the DATA step and view the log. Did the DATA step execute successfully? How many observations and variables does Work.One2one contain?

  6. Print the Work.One2one data set and view the output.
