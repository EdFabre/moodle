SAS Data Set Frequentflyers

The Frequentflyers data set contains information about all members of a frequent flyers program.


Data Set Description
Variable Type Length Description
FFID char 6 frequent flyer ID number
MemberType char 6 member type (GOLD, SILVER, BRONZE)
Name char 25 name
Address char 20 address
PhoneNumber char 12 phone number
City char 20 city
State char 2 state
ZipCode char 5 zip code
MilesTraveled num 8 miles traveled
PointsEarned num 8 points earned
PointsUsed num 8 points used
