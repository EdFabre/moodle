How SAS Updates Data Using the MODIFY and BY Statements

This animation shows how SAS updates values in the master data set (Capacity) based on values in a transaction data set (Sasuser.Newrtnum).

  1. The following program is submitted to SAS: 
    data capacity; 
   modify capacity sasuser.newrtnum;
   by flightid; 
run;
  2. SAS reads the first observation from the transaction data set (Sasuser.Newrtnum) into the transaction PDV, which is a memory buffer.

  3. Based on the value of the BY variable (FlightID) in the transaction data set, SAS builds a dynamic WHERE statement (where flightid = 'IA00100';).

  4. SAS then applies the dynamic WHERE statement to the master data set (Capacity) and reads the selected observation into the master PDV.

  5. To update the master data set, SAS overlays common variables from the transaction PDV onto variables in the master PDV. Only RouteID changes.

  6. Finally, SAS writes the updated observation from the master PDV to the same location in the master data set. Then SAS processes the remaining transactions in the same way.
