SAS Data Set Pilotemp

The Pilotemp data set contains information on pilots for an airline.

Data Set Description

Variable Label Type Length Description
Country Employee Country of Residence char 25 home country of employee
Division Division char 30 employee's division
EmpID Employee Identification Number char 6 ID number of employee
FirstName Employee First Name char 32 first name of employee
HireDate Employee Hire Date num 8 date of hire
JobCode Job Code char 6 abbreviated job description
LastName Employee Last Name char 32 last name of employee
Location Employee Office Location char 16 city location of employee
Phone Employee Extension Number char 8 phone extension of employee
Salary Employee Salary num 8 salary
