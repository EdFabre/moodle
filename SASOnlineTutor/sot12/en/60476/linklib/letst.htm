The %LET Statement

The %LET statement enables you to define a macro variable and to assign a value to it.


General form, %LET statement:
%LET variable = value;

where

  • variable is any name that follows the SAS naming convention.
  • value can be any string from 0 to 64K characters.
  • if either variable or value contains a reference to another macro variable, the reference is evaluated before the assignment is made.
If variable already exists, value replaces the current value.


For example, the following statement creates a macro variable named Time and assigns a value of afternoon to it. 

%let time=afternoon;
When you use the %LET statement to define macro variables, you should keep in mind the following rules:
  • Numeric tokens are stored as character strings.
  • Mathematical expressions are not evaluated.
  • The case of value is preserved.
  • Quotation marks bounding literals are stored as part of value.
  • Leading and trailing blanks are removed from value before the assignment is made.

The following table provides examples of macro variable assignment statements to illustrate these rules.

 
%LET Statement Variable Name Variable Value
%let name= Ed Norton ; Name Ed Norton
%let name2=' Ed Norton '; Name2 ' Ed Norton '
%let title="Joan's Report" Title "Joan's Report"
%let start=; Start  
%let total=0; Total 0
%let sum=4+3; Sum 4+3
%let total=&total+&sum; Total 0+4+3
%let x=varlist; X varlist
%let &x=name age height; Varlist name age height
