SAS Data Set Talent99

The Talent99 data set records information on performers, their talents and the agencies that represent them.


Data Set Description

Variable Type Length Description

FirstName Character 8 employee's first name
Address1 Character 21 street address
Address2 Character 16 city, state
Rate Numeric 8 pay rate
Agency Character 20 agency name
Comment Character 30 comments
ID Character 4 identification number
Phone Numeric 8 phone number
LastHired Numeric 8 date last hired
BirthDate Numeric 8 birthdate
Height Character 2 height, in inches
Month Numeric 8 month
Day Numeric 8 day
