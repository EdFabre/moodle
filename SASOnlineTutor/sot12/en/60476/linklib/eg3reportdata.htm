Set Up for Creating Reports Using SAS Enterprise Guide

Before you begin working through this lesson, perform the following tasks:

Check your version of SAS Enterprise Guide and apply a hotfix if necessary.

If you have already applied the hotfix for SAS Enterprise Guide 3.0, you can skip this section and go on to the next section.

  1. Start SAS Enterprise Guide.

  2. Select HelpAbout Enterprise Guide.

  3. Click System Information. Under Current Process, look at the value for ApplicationVersion.



    • If the value for ApplicationVersion is 3.0.1.nnn, the hotfix has been applied and you can go on to the next section.


    • If the value for ApplicationVersion is 3.0.0.nnn, you need to download and apply the hotfix by following these instructions:


      • Close SAS Enterprise Guide.

      • Download the hotfix from the SAS Technical Support Hot Fix Web site. Once you have applied the hotfix, you can verify that it is in place by repeating steps 1-3 above. Then, go on to the next section.

Download data and add it to your project

Open EGReportData.zip and extract the data files to your Enterprise Guide\sample\data directory.

You can save the data files to any location as long as you remember where they are so that you can find them when you add them to your project in SAS Enterprise Guide.
