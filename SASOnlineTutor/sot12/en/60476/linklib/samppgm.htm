Sample Program for z/OS (OS/390)

The following program shows examples of platform-specific statements for creating ODS output in the z/OS operating environment. For more details, see the SAS documentation.  


/* Allocate a PDSE for the HTML output.              */

filename pdsehtml '.example.htm' 
                  dsntype=library dsorg=po
                  disp=(new, catlg, delete);

/* Prevent error messages related to graph names.    */

options fileext=ignore;

/* Create variables to contain link locations.       */

data work.activity;
   set sasuser.diabetes;
   length DrillVar $ 30;
   if sex='F' then 
      drillvar='HREF="females.htm"';
   if sex='M' then 
      drillvar='HREF="males.htm"';
run;

/* Specify the files to create for the HTML output.  */
/* These files are PDSE members.                     */
/* The PATH= option specifies the location for the   */
/* HTML files. The URL= suboption on PATH= prevents  */
/* information from PATH= from appearing in the      */
/* links and references that ODS creates. The URLs   */
/* will be the same as the URL= specifications.      */
/* The RS= option creates HTML that you can modify   */
/* in an editor and use on a z/OS (OS/390) web server.     */

ods listing close;
ods html path=pdsehtml(url=none)
         frame='frame'(url='frame.htm')
         contents='toc'(url='toc.htm')
         body='htmlpage'(url='graph.htm')
         rs=none;

/* Specify graphics options.                         */

goptions device=gif xpixels=400 ypixels=300;

/* Create the graph and HTML file.                   */

title 'Age Categories by Sex';
proc gchart data=work.activity;
   vbar age / subgroup=sex html=drillvar ;
run;
quit;

/* Close HTML output and open listing output.        */

ods html close;
ods listing;
 
Remember that using the GPATH= option in addition to the PATH= option allows you to store HTML files in one place and GIF files in another. This separation may make it easier to distinguish which files need to be renamed to .gif and which need to be renamed to .html when you FTP them to a PC or UNIX system.
