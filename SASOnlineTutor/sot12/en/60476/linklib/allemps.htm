SAS Data Set Allemps

The Allemps data set contains information about employees of an airline.

Data Set Description

Variable Label Type Length Description
EmpID   char 6 employee's id number
LastName   char 32 last name of employee
Phone   char 8 phone extension of employee
Location   char 16 city location of office
Division   char 30 division of employment
