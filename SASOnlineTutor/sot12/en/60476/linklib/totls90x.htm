SAS Data Set Totals2000

The Totals2000 data set contains information on exercise therapy and treadmill tests in a hypothetical wellness clinic in a twelve-month period.


Data Set Description

Variable Type Length Description
Month num 2 month in which data is collected
Therapy num 3 patients in exercise therapy
NewAdmit num 3 number of new admissions
Treadmill num 3 number of treadmill tests administered
