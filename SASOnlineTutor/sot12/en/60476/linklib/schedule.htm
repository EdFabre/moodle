SAS Data Set Schedule

The Schedule data set contains schedule information about computer training courses.


Data Set Description

Variable Label Type Length Description
Begin_Date Begin num 8 start date for course
Course_Code Course Code char 4 code for course
Course_Number Course Number num 8 unique course number
Location Location char 15 city where course is taught
Teacher Instructor char 20 name of course instructor
