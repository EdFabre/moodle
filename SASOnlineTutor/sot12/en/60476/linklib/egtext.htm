Creating SAS Listing Output in SAS Enterprise Guide

Output in SAS Enterprise Guide is in HTML format by default. To create SAS listing output, do the following:

  1. Select ToolsOptions.

  2. Click the Results tab, then select the Text output check box. Leave the HTML check box selected.

  3. Click OK to close the Options window.

Once you have selected Text output, the output from your SAS procedures will be created in both HTML and SAS listing output (text). HTML output opens by default.

You can open the text output by double-clicking the Output item for your code in the Project window.
