Resetting SAS Enterprise Guide to Automatically Generate HTML Output

After you have learned to use ODS programming statements, you should reset SAS Enterprise Guide so that it automatically creates HTML output. Follow these steps:

  1. Select Tools Options.

  2. In the Options window, click the SAS Programs tab.

  3. In the Code area, select the Automatically direct results back to Enterprise Guide check box.

  4. Click the Results tab.

  5. In the Result Formats area, clear the Text output check box.

  6. Click OK to close the Options window and apply the changes.
