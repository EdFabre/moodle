SAS Data Set Diabetes

The Diabetes data set contains medical information for diabetic patients in a wellness clinic.


Data Set Description

Variable Type Length Description
ID char 4 patient ID number
Sex char 1 sex of patient
Age num 8 age of patient
Height num 8 height of patient
Weight num 8 weight of patient
Pulse num 8 pulse rate
FastGluc num 8 fasting glucose level
PostGluc num 8 postprandial glucose level
