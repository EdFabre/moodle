SAS Data Set Wchill

The Wchill data set contains data values in a coordinate chart structure that helps to calculate wind chill values. Wind chill is based upon temperature (in degrees Fahrenheit) and wind speed (in miles per hour). The columns represent temperatures and the rows represent wind speeds.
