MEANS Procedure

The MEANS procedure provides an easy way to compute descriptive statistics. These data summarization tools can answer basic questions about numeric variables.


General form, basic MEANS procedure:
PROC MEANS <DATA=SAS-data-set>
<statistic-keyword(s)> <option(s)>;
RUN;

where

  • SAS-data-set identifies the data set to process
  • statistic-keyword(s) specifies the statistics to compute
  • option(s) controls the content, analysis, and appearance of output.


In its simplest form, PROC MEANS prints the n-count (number of non-missing values), mean, standard deviation, and minimum and maximum values of every numeric variable in a data set.


Example

The following example prints the n-count, mean, standard deviation, and minimum and maximum values of every numeric variable in the Sasuser.Sale2000 data set. 

     proc means data=sasuser.sale2000;
     run;
