How SAS Updates Data Using the MODIFY Statement

This animation shows how the DATA step updates values in the SAS data set that is specified in the MODIFY statement.

  1. The following program is submitted to SAS: 
    data capacity;
   modify capacity;
   CapEcon = int(CapEcon * .95);
   CapBusiness = int(CapBusiness * .90);
run;
  2. SAS reads the first observation from Sasuser.Capacity into the program data vector.

  3. Then SAS updates values for CapBusiness and CapEcon in the program data vector, based on the two assignment statements in the DATA step.

  4. Finally, SAS writes the updated observation to Sasuser.Capacity and then repeats the process for all observations in the data set.
