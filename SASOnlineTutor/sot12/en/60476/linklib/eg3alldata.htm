Setting up Practice Data for SAS Enterprise Guide 3.0

Before you begin working through lessons in Using SAS Enterprise Guide, perform the following tasks:

Check Your Version of SAS Enterprise Guide and Apply a Hot Fix if Necessary

  1. Start SAS Enterprise Guide.

  2. In the Welcome window, click New Project.

  3. Select HelpAbout Enterprise Guide.

  4. Click System Information. Under Current Process, look at the value for ApplicationVersion.



    • If the value for ApplicationVersion is 3.0.1.nnn or higher, your version is fine and you can go on to the next section.


    • If the value for ApplicationVersion is 3.0.0.nnn, you need to download and apply the latest hot fix by following these instructions:


      • Close SAS Enterprise Guide.

      • Download the hot fix from the SAS Technical Support Hot Fix Web site. Once you have applied the hot fix, you can verify that it is in place by repeating steps 1-3 above. Then, go on to the next section.

Download Practice Data to Your Sample Data Directory

Open egalldata.zip and extract the data files to your SAS\Enterprise Guide\Sample\Data directory.

Some of the data sets in this ZIP file are already included in the sample data directory for this version of SAS Enterprise Guide. You can overwrite the files.

You can save the data files to any location as long as you remember where they are so that you can find them when you add them to your project in SAS Enterprise Guide.
