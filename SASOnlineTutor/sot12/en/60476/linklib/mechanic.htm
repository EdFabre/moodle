SAS Data Set Mechanics

The Mechanics data set contains information about the mechanics employed at an airline.


Data Set Description

Variable Type Length Description
ID char 4 ID number
LastName char 9 last name
FirstName char 9 first name
City char 12 city
State char 2 state
Gender char 1 gender
Salary num 8 current salary
Birth num 8 birth date
Hired num 8 date hired
HomePhone char 12 home phone number
SSN num 8 Social Security number
JobCategory char 2 job category
JobLevel char 2 job level
