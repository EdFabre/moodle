SAS Data Set Empdata

The Empdata data set contains information about employees of an airline.

Data Set Description

Variable Label Type Length Description
Country Employee Country of Residence char 25 employee's home country
Division Division char 30 division of employment
EmpID Employee Identification Number char 6 employee's id number
FirstName Employee First Name char 32 first name of employee
HireDate Employee Hire Date num 8 date of hire
JobCode Job Code char 6 abbreviated job description
LastName Employee Last Name char 32 last name of employee
Location Employee Office Location char 16 city location of office
Phone Employee Extension Number char 8 phone extension of employee
Salary Employee Salary num 8 salary of employee
