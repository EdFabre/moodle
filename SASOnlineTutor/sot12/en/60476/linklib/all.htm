SAS Data Set All

The All data set contains information combined from the Courses, Schedule, Students, and Register data sets. This data set provides an overview of information about computer training courses and students.


Data Set Description

Variable Type Length Description
Student_Name char 25 name of student
Course_Number num 8 unique course number
Paid char 1 whether the student has paid
Course_Code char 4 code for course
Location char 15 city where course is taught
Begin_Date num 8 start date for course
Teacher char 20 name of course instructor
Course_Title char 25 title of course
Days num 8 length of course, in days
Fee num 8 fee for course
Student_Company char 40 company that student represents
City_State char 20 city and state for student
