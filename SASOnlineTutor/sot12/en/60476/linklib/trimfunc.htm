TRIM Function

The TRIM function enables you to remove trailing blanks from character values.


General form, TRIM function:
TRIM(argument)

where argument can be any character expression, such as

  • a character variable: trim(address)
  • another character function: trim(left(id)).



Example

The variable Address in the data set Work.Temp contains embedded blanks.


SAS Data Set Work.Temp

Name Address Phone
CICHOCK, ELIZABETH MARIE 65 ELM DR,        CARY,       27513 6224549
BENINCASA, HANNAH LEE 11 SUN DR,        CARY,       27513 6223251


The assignment statement in the following program creates the variable NewAddress from Address. The TRIM function is included in the assignment statement to remove the embedded blanks. 
     data work.newtemp(drop=address city state zip);
        set work.temp;
        NewAddress=trim(address);
     run;

SAS Data Set Work.NewTemp
Name NewAddress Phone
CICHOCK, ELIZABETH MARIE 65 ELM DR, CARY, 27513 6224549
BENINCASA, HANNAH LEE 11 SUN DR, CARY, 27513 6223251


Points to Remember

The TRIM function does not affect how a variable is stored. Suppose you trim the values of a variable and then assign these values to a new variable. The trimmed values are padded with trailing blanks again if the values are shorter than the length of the new variable.

Here's an example. In the DATA step below, the trimmed value of Address is assigned to the new variable Street. When the trimmed value is assigned to Street, trailing blanks are added to the value to match the length of 20. 

     data temp;
        set work.temp;
        length Street $ 20;
        Street=trim(address);
     run;
