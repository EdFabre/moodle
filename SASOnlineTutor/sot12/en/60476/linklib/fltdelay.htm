SAS Data Set Flightdelays

The Flightdelays data set contains information about flights that have had delays.

Data Set Description
Variable Type Length Description
Date num 8 date of flight
FlightNumber char 3 flight number
Origin char 3 flight origin
Destination char 3 flight destination
DelayCategory char 15 delay category
DestinationType char 15 destination type (Domestic, International)
DayOfWeek num 8 day of the week
Delay num 8 length of delay
