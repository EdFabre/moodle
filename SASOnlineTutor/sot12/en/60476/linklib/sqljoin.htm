Using PROC SQL to Create a Table from the Results of an Inner Join

You can use PROC SQL to combine data from multiple data sets and write the results to a new data set.


General form, PROC SQL step to create a table from the results of an inner join:
PROC SQL;
      CREATE TABLE SAS-data-set AS
            SELECT column-1<, column-2...,column-n>
                  FROM table-1<, table-2...,table-n>
                  WHERE joining criteria;
where
  • SAS-data-set names the table that is created to hold the results
  • column-1<..., column-n> specifies the columns that are contained in the new table
  • table-1<..., table-n> specifies the input tables
  • joining criteria refers to one or more expressions that specify the column or columns on which the tables are to be joined.


Example

The following program combines data from the Sasuser.Empdata and the Sasuser.Jcodedat data sets into a new data set named Sasuser.Combined: 
     proc sql;
        create table sasuser.combined as
           select empdata.lastname, empdata.jobcode, 
		          jcodedat.descript
              from sasuser.empdata, sasuser.jcodedat
              where empdata.jobcode=jcodedat.jobcode;
     quit;
