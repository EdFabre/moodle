Reserved Words in the Macro Facility

The macro facility has reserved words that cannot be used as macro names:


  • ABEND
  • ABORT
  • ACT
  • ACTIVATE
  • BQUOTE
  • BY
  • CLEAR
  • CLOSE
  • CMS
  • COMANDR
  • COPY
  • DEACT
  • DEL
  • DELETE
  • DISPLAY
  • DMIDSPLY
  • DMISPLIT
  • DO
  • EDIT
  • ELSE
  • END
  • EVAL
  • FILE
  • GLOBAL
  • GO
  • GOTO
  • IF
  • INC
  • INCLUDE
  • INDEX
  • INFILE
  • INPUT
  • KEYDEF
  • LENGTH
  • LET
  • LIST
  • LISTM
  • LOCAL
  • MACRO
  • MEND
  • METASYM
  • NRBQUOTE
  • NRQUOTE
  • NRSTR
  • ON
  • OPEN
  • PAUSE
  • PUT
  • QSCAN
  • QSUBSTR
  • QSYSFUNC
  • QUOTE
  • QUPCASE
  • RESOLVE
  • RETURN
  • RUN
  • SAVE
  • SCAN
  • STOP
  • STR
  • SUBSTR
  • SUPERQ
  • SYSCALL
  • SYSEVALF
  • SYSEXEC
  • SYSFUNC
  • SYSGET
  • SYSRPUT
  • THEN
  • TO
  • TSO
  • UNQUOTE
  • UNSTR
  • UNTIL
  • UPCASE
  • WHILE
  • WINDOW


Example

If you attempt to define a macro named PUT, SAS will issue the folowing messages to the SAS log when you submit the macro definition for compilation.


Partial SAS Log 
ERROR: Macro PUT has been given a reserved name.
ERROR: A dummy macro will be compiled.
73   %macro put;
74       proc print data=&syslast(obs=5);
75       title "Listing of &syslast data set";
76       run;
77   %mend;
