Macro Variables Used By SAS Enterprise Guide

SAS Enterprise Guide uses the following macro variables:

 

Macro Variable Description
_EGUSERID User ID of the client user.
_EGUSERNAME Full user name, if that information is available. The full user name is available only on Microsoft Windows NT, 2000, and XP. On other operating systems, this variable is the same as _EGUSERID.
_EGMACHINE Client machine node name.
_EGVERSIONL EG product version, including build number.
_EGSERVERNAME Name of the logical server.
_EGHOSTNAME Server node name (IP address, DNS name, or DCOM name).
_EGTASKLABEL Label for the current task. This is the text label that is displayed in the project window.
