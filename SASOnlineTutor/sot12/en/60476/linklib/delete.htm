DELETE Statement

The DELETE statement in a PROC SQL step removes one or more rows from a table or view that is specified in the FROM clause.


General form, DELETE statement for deleting rows in a table:

DELETE FROM table-name | proc-sql-view
      <WHERE expression>;

where

  • table-name specifies the name of the table in which rows will be deleted.
  • proc-sql-view specifies a PROC SQL view that you are deleting rows from. proc-sql-view can be a one-level name, a two-level libref.view name, or a physical pathname that is enclosed in single quotation marks.
  • WHERE is optionally used to specify an expression that subsets the rows to be deleted.
If you want to delete only a subset of rows in the table, you must specify a WHERE clause or all rows of the table will be deleted.
