Using the Sasuser.Admit data set, add flyover text to cells.

  1. Copy the following program and paste it into the code editing window.
    ods listing close;
ods html path='?' body='?';
proc format;
   value $colsex 'F'='lipk'
                 'M'='vpab';
   value agealert low-29='lio'
                  other='vpag';
run;
proc tabulate data=sasuser.admit format=4.1;
   var age / style=<parent>;
   class sex actlevel / style={background=vpag};
   classlev sex / style={background=$colsex.};
   classlev actlevel / style=<parent>;
   table sex*actlevel*age*{style={
         background=agealert.
         font_weight=bold foreground=black}},
         mean / box={style={background=vpag}};
   keyword mean / style={background=vpag};
run;
ods html close;
ods listing;
  2. Replace the question marks (?) with PATH= and BODY= values that conform to your operating system's file structure. Remember that you can use the URL= suboption in PATH=, BODY=, or both.

  3. Submit the program and view its output in your browser.

  4. Add a statement to create a user-defined format for flyover text. Name the format AGEFLYOV and repeat the ranges that you used for AGEALERT:

    • low-29=' Could be a problem '

    • other=' ' (Leave blank.)

  5. Apply the format by setting the Age variable's flyover attribute to AGEFLYOV. in the TABLE statement.

  6. Submit the program and view its output in your browser. Hold your mouse pointer over the orange data cell.
