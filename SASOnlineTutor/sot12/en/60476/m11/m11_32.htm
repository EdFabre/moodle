Adding Flyover Text (continued)
The program below creates flyover text by using the format AGEFLYOV. Complete the program by adding the missing attribute and value to the STYLE= option that is attached to Age. 

ods listing close;
ods html path='c:\data' body='table.html';
proc format;
   value $sexcolr 'F'='cx00DDDD'
                  'M'='cx00BBBB';
   value agealert low-29='cxFF7700'
                  other='cxFFDDBB';
   value ageflyov low-29='Too low: Need to attract seniors.'
                  other=' ';
run;
proc tabulate data=clinic.admit format=4.1
     style={background=cxFFDDBB};
   var fee;
   class sex actlevel / style={background=cxFFDDBB};
   classlev sex / style={foreground=cxFFFFFF
            font_style=italic background=$sexcolr.};
   classlev actlevel / style=<parent>;
   var age / style=<parent>;
   table sex*actlevel*
         age*{style={background=agealert.
         }},
         mean / box={style={background=cxFFDDBB}};
   keyword mean / style={background=cxFFDDBB};
run;
