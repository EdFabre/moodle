Using the Sasuser.Admit data set, add trafficlighting to cells.

  1. Copy the following program and paste it into the code editing window.
    ods listing close;
ods html path='?' body='?';
proc format;
   value $colsex 'F'='lipk'
                 'M'='vpab';
run;
proc tabulate data=sasuser.admit format=4.1;
   var age / style=<parent>;
   class sex actlevel / style={background=vpag};
   classlev sex / style={background=$colsex.};
   classlev actlevel / style=<parent>;
   table sex*actlevel*age*{style={font_weight=bold
         foreground=black}},mean /
         box={style={background=vpag}};
   keyword mean / style={background=vpag};
run;
ods html close;
ods listing;
  2. Replace the question marks (?) with PATH= and BODY= values that conform to your operating system's file structure. Remember that you can use the URL= suboption in PATH=, BODY=, or both.

  3. Submit the program and examine its output in your browser.

  4. Create a user-defined format for trafficlighting. Name the format AGEALERT and give it these values:

    • low-29='lio' (SAS/GRAPH color name light orange)

    • other='vpag' (SAS/GRAPH color name very pale green)

  5. Apply the format by adding it to the STYLE= option that is attached to the Age variable in the TABLE statement. Use AGEALERT as the value of the background attribute.

  6. Submit the program and examine its output in your browser.
