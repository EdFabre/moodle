Adding Color to Table Cells (continued)
Edit the following program to change the background of the box cell to cyan. Use the color name CYAN or the RGB value 00FFFF. 

ods listing close;
ods html path='c:\data' body='table.html';
proc tabulate data=clinic.admit format=4.1;
   var age;
   class sex actlevel;
   table sex*actlevel*age,mean / 
        
   keyword mean / style={background=cyan};
run;
ods html close;
