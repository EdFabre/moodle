Using the Sasuser.Admit data set, apply styles to heading cells.

Have you set options in Enterprise Guide so you can submit your own ODS statements? You must do this before you can do any of the practices in this lesson.
Have you set options in Enterprise Guide so you can submit your own ODS statements? You must do this before you can do any of the practices in this lesson.

  1. Copy the following program and paste it into the code editing window. 
    ods listing close;
ods html path='?' body='?';
proc tabulate data=sasuser.admit 
     format=4.1;
   var age;
   class sex actlevel;
   table sex*actlevel*age,mean;
run;
ods html close;
ods listing;
  2. Replace the question marks (?) with PATH= and BODY= values that conform to your operating system's file structure. Remember that you can use the URL= suboption in PATH=, BODY=, or both.

  3. Submit the program and view the HTML table in your browser.

  4. Add the STYLE= option to the program's CLASS statement to change the background color of the Sex and ActLevel headings to white. Remember to add a slash (/) before the option.

  5. Insert a KEYWORD statement to change the background color of the MEAN statistic to white. Remember to add a slash before the STYLE= option.

  6. Add the BOX= option to the TABLE statement to change the background color of the upper-left cell to ltgray (you could also use CCCCCC). Remember to add a slash before the BOX= option and to enclose the entire STYLE= option in braces.

  7. Submit the program and view the results in your browser.
The following program is an example. Many variations are possible. 
ods listing close;
ods html path='c:\data' body='active.html';
proc tabulate data=sasuser.admit format=4.1;
   var age;
   class sex actlevel / style={background=white};
   table sex*actlevel*age,mean /
         box={style={background=ltgray}};
   keyword mean / style={background=white};
run;
ods html close;
ods listing;
Using Windows Explorer, locate the HTML file you created and open it in your browser.
