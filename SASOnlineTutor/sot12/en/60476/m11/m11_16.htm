Using the Sasuser.Admit data set, apply styles to class level headings.

  1. Copy the following program and paste it into the code editing window. 
    ods listing close;
ods html path='?' body='?';
proc tabulate data=sasuser.admit 
     format=4.1;
   var age;
   class sex actlevel / 
         style={background=vpab};
   table sex*actlevel*age,mean /
         box={style={background=papk}};
   keyword mean / 
           style={background=papk};
run;
ods html close;
ods listing;
  2. Replace the question marks (?) with PATH= and BODY= values that conform to your operating system's file structure. Remember that you can use the URL= suboption in PATH=, BODY=, or both.

  3. Submit the program and view its output in your browser.

  4. Add a CLASSLEV statement and the STYLE= option to change the background color of the levels of Sex. Use either SAS/GRAPH color names or RGB values.

  5. Submit the program and view its output in your browser.

  6. Create a user-defined format to specify different colors for the values of Sex. Change the STYLE= option in the CLASSLEV statement to apply the new format.

  7. Submit the program and view its output in your browser.
