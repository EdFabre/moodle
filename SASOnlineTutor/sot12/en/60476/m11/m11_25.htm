Using the Sasuser.Admit data set, specify style attributes that are inherited by other table cells.

  1. Copy the following program and paste it into the code editing window.
    ods listing close;
ods html path='?' body='?';
proc format;
   value $colsex 'F'='lipk'
                 'M'='vpab';
run;
proc tabulate data=sasuser.admit 
     format=4.1;
   var age;
   class sex actlevel / 
         style={background=vpag};
   classlev sex / 
            style={background=$colsex.};
   table sex*actlevel*age,mean /
         box={style={background=vpag}};
   keyword mean / 
           style={background=vpag};
run;
ods html close;
ods listing;
  2. Replace the question marks (?) with PATH= and BODY= values that conform to your operating system's file structure. Remember that you can use the URL= suboption in PATH=, BODY=, or both.

  3. Submit the program and view its output in your browser.

  4. Add a CLASSLEV statement with a STYLE= option to make the ActLevel class level headings inherit styles from the levels of Sex.

  6. Edit the VAR statement to make the Age headings inherit styles from the ActLevel headings.
