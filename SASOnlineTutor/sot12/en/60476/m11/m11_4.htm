Adding Color to Table Cells (continued)

Changing Class Headings

When you customize tables, it's best to begin with the outer cells. To change the background color of the Dest class heading, add a slash (/) and the appropriate STYLE= option to the CLASS statement. 

     class dest / style={background=lib};
You can specify any SAS/GRAPH color name. The color name LIB, for example, corresponds to light blue. For even greater control, supply an RGB color value in the form cxRRGGBB, where red, green, and blue are represented by hexadecimal values from 00 (darkest) to FF (lightest). For example, the value cxCCEEDD is a pale bluish-green. 

     ods listing close;
     ods html path='c:\data' body='laguard.html';
     title1 'LaGuardia Flights by Destination';
     title2 'and Revenue, March 4-10';
     proc format;
        value $desname 'CPH'='Copenhagen'
                       'FRA'='Frankfurt'
                       'LON'='London'
                       'PAR'='Paris';
     run;
     proc tabulate data=flights.laguardia format=comma9.;
        class dest / style={background=cxCCEEDD};
        var boarded transferred deplaned revenue;
        table dest*(boarded transferred deplaned revenue),
              min max mean sum;
        label dest='Destination';
        format dest $desname.;
     run;
     ods html close;
     ods listing;
