Understanding How Data Sets Are Read (continued)

Execution Phase (continued)

  1. At the bottom of the DATA step, the values in the program data vector are written to the data set as the second observation.

  1. The value of _N_ is set to 3, and control returns to the top of the DATA step. SAS retains the values of variables that were read from a SAS data set with the SET statement, or that were created by a sum statement. All other variable values, such as the values of the variable Interest, are set to missing.


This process continues until all of the observations are read.
