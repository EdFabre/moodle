Practice grouping variables using the Sasuser.Pilots data set.

  1. Copy the program below and paste it into the code editing window. 
    data work.pilotjob(drop=salary);
   set sasuser.pilots(keep=state salary);
   TotalPay=0;
   totalpay+salary;
run;
proc print data=work.pilotjob noobs;
   sum totalpay;
   format totalpay dollar12.0;
run;
  2. Submit the program and view the output. The sum of TotalPay should be $1,637,513.

  3. Modify the program to group observations by State and to select only the last observation for each state. (Sort the data set if necessary.)

  4. Submit the program and view the log messages about the DATA step. Work.Pilotjob should contain 3 observations and 2 variables. If not, correct the program and resubmit it.

  5. Scroll through the output. The sum of TotalPay should be the same as before.
