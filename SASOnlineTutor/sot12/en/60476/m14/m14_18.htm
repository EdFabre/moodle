Detecting the End of a Data Set (continued)
Suppose that the Payroll.Divs data set is sorted by the variable Division and that Division contains three unique values: ADM, CAM, and LAM. Which of the following programs does not correctly select only the last observation in the data set? 
data work.lastobs;
   set payroll.divs;
   by division;
   if division='LAM' and last.division;
run;

data work.lastobs;
   pickobs=last;
   set payroll.divs point=pickobs;
run;

data work.lastobs;
   set payroll.divs end=last;
   if last;
run;
