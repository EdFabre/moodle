Practice detecting the end of the data set using the Sasuser.Stress2 data set.

  1. Copy the program below and paste it into the code editing window. 
    data work.testtime;
   set sasuser.stress2;
run;
proc print data=work.testtime;
run;

  3. Modify the DATA step to select only the last observation.

  4. Submit the program and view the log messages about the DATA step. Work.TestTime should contain 1 observation.
