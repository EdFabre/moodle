Creating One- and Three-Dimensional Tables

Throughout this lesson, you've created and enhanced a two-dimensional table. However, with a simple modification, you can easily produce a one- or three-dimensional table.

Think again about how the TABLE statement specifies table dimensions. As shown below, a TABLE statement that has two dimension expressions requests a table that displays rows and columns. 

     proc tabulate data=clinic.admit;
        class sex;
        var height weight;
        table sex,height*min weight*min;
     run;

Here is the two-dimensional table that the PROC TABULATE step produces.


  Height Weight
Min Min
Sex 61.00 118.00
F
M 69.00 147.00


Three-Dimensional Tables

To create a three-dimensional table, you add a third dimension expression on the left.

In the TABLE statement below, ActLevel is specified in the page expression, Sex is specified in the row expression, and Height and Weight are specified in the column expression. This PROC TABULATE step produces separate pages for each value of ActLevel. 

     proc tabulate data=clinic.admit;
        class sex actlevel;
        var height weight;
        table actlevel,sex,height*min weight*min;
     run;
