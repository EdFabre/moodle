Creating One- and Three-Dimensional Tables (continued)

One-Dimensional Tables

To create a one-dimensional table, you can delete or combine dimension expressions, leaving only the column dimension.

The TABLE statement below specifies columns that display the minimum values of Height and Weight. 

     proc tabulate data=clinic.admit;
        var height weight;
        table height*min weight*min;
     run;
