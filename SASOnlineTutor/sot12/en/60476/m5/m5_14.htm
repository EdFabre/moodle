Using the Sasuser.Admit data set, specify variables in a PROC TABULATE step.

  1. Copy the program below and paste it into the code editing window. 
    proc tabulate data=sasuser.admit;
   class ?
   var ?
   table sex,(height weight)*mean;
run;
  2. Edit the program, completing the CLASS and VAR statements with variables that are specified in the TABLE statement. End each statement with a semicolon. Note that MEAN is a statistic, not a variable name, so it does not belong in a CLASS or VAR statement.

  3. Submit the step and view the output.
