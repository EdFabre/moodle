Creating One- and Three-Dimensional Tables (continued)
The following PROC TABULATE step creates a one-dimensional table and a two-dimensional table. Add a TABLE statement to create a three-dimensional table that displays separate pages for tolerance levels (Tolerance), rows for maximum heart rate (MaxHR) and recovery heart rate (RecHR), and a column for the statistic MEAN. 

proc tabulate data=clinic.stress format=8.;
   class tolerance;
   var resthr maxhr rechr timemin timesec;
   table mean*(resthr maxhr rechr);
   table tolerance,timemin timesec;
   
run;
