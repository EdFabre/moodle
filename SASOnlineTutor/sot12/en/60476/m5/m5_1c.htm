Three-Dimensional Tabular Report

You can create a three-dimensional table, such as the one below, that displays a separate page for each value of ActLevel. 

     proc tabulate data=clinic.admit;
        class sex actlevel;
        var height weight;
        table actlevel,sex,height*min weight*min;
     run;
