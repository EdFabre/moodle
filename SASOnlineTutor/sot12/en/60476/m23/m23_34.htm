Use list output to create a raw data file using the SAS data set Sasuser.Insure.

  1. Create a comma-delimited raw data file named Insurdat from the SAS data set Sasuser.Insure. Save the file to a location in your operating environment. Notice that the values for Name contain commas. Format the values for Total and BalanceDue with the DOLLARw.d. format.

  2. Submit the program.

  3. Check the SAS log to verify that the program ran correctly. The Log window should indicate that 21 records were written from Sasuser.Insure to Insurdat.

  4. Using Windows Explorer, open Insurdat in a text editor such as Notepad.

    To browse this file using SAS Enterprise Guide, select FileOpen. Navigate to the location where you saved the file. In the Open As drop-down list, select Note.

    Notice that the values for Name and the values for Total that contain commas are enclosed in quotation marks.

  1. Create a comma-delimited raw data file named Insurdat from the SAS data set Sasuser.Insure. Save the file to a location in your operating environment. Notice that the values for Name contain commas. Format the values for Total and BalanceDue with the DOLLARw.d. format.

  3. Check the SAS log to verify that the program ran correctly. The Log window should indicate that 21 records were written from Sasuser.Insure to Insurdat.

  4. Use one of the following methods to browse the contents of Insurdat.

    • Assign a File Shortcut (or fileref) to the file. Then open it from the SAS Explorer.

    • Drag and drop the file into the code editing window.

    • Open the file using a text editor.

    Notice that the values for Name and the values for Total that contain commas are enclosed in quotation marks.

Submit the following program: 
 data _null_;
    set sasuser.insure;
    file 'c:\mydata\insurdat' dsd;
    put id name policy company pctinsured
       total : dollar9.2 balancedue : dollar9.2;
 run;

The path specified in the FILE statement will vary based on your operating environment and the storage location you specified.
