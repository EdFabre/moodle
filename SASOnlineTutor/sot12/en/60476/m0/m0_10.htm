Interactive Fill-in-the-Blank Questions

Each lesson contains many questions that test your skills as you go. Fill-in-the-blank questions provide a field in which you enter an answer to a question, such as code that is missing from a program.

After you have entered your answer, click Check My Answer for feedback. The answer checker provides provides specific feedback for common errors. Not sure about the answer? Click Show Correct Answer for the solution and an explanation.
