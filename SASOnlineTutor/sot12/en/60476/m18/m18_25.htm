Add an option to specify a destination for the HTML files you create.

  1. Copy the following program and paste it into the code editing window:
    ods listing close;
ods html body='myoutput.htm'
    (url='myoutput.htm')
    contents='mytoc.htm'
    (url='mytoc.htm')
    frame='myframe.htm';
proc print data=sasuser.admit;
run;
proc print data=sasuser.insure;
run;
ods html close;
ods listing;
  2. Add an option to save myoutput.htm, mytoc.htm, and myframe.htm to any location on your computer. Add a suboption to specify that no path information be used in the links that are generated to body and contents files.

  3. Submit the program and open the frame file.
  1. Copy the following program and paste it into the code editing window:
    ods listing close;
ods html body='myoutput.htm'
    (url='myoutput.htm')
    contents='mytoc.htm'
    (url='mytoc.htm')
    frame='myframe.htm';
proc print data=sasuser.admit;
run;
proc print data=sasuser.insure;
run;
ods html close;
ods listing;
  2. Add an option to save myoutput.htm, mytoc.htm, and myframe.htm to your My SAS Files directory (or your default storage location). Add a suboption to specify that no path information be used in the links that are generated to body and contents files.

    In the CMS operating environment: specify myoutput htm as the HTML filename and filetype for the body file.

  3. Submit the program and open the frame file.
  1. Submit the following program, replacing <my-location> with your directory path or storage location: 
    ods listing close;
 ods html path='<my-location>'
         (url=none)
          body='myoutput.htm'
         (url='myoutput.htm')
          contents='mytoc.htm'
         (url='mytoc.htm')
          frame='myframe.htm';
 proc print data=sasuser.admit;
 run;
 proc print data=sasuser.insure;
 run;
 ods html close;
 ods listing;
  2. In your browser, open myframe.htm to view both the body file and the contents file. Click the links in the contents to view the output for each procedure.
