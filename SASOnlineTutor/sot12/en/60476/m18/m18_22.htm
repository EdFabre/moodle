Use an option to specify link paths in ODS HTML output.

  1. Copy the following program and paste it into the code editing window:
    ods listing close;
ods html 
    body='<my-location>myoutput.htm'
    contents='<my-location>mytoc.htm'
    frame='<my-location>myframe.htm';
proc print data=sasuser.admit;
run;
proc print data=sasuser.insure;
run;
ods html close;
ods listing;
  2. Replace <my-location> with the path to any location on your computer.

  3. Add options to specify the URL myoutput.htm for the body file and the URL mytoc.htm for the contents file.

  4. Submit the program and open the frame file myframe.htm in your browser. Use the links in the contents file to view the procedure output in the body file.

  2. Replace <my-location> with your My SAS Files directory (or your default storage location).

    In the CMS operating environment: specify myoutput htm as the HTML filename and filetype for the body file.

  3. Add options to specify the URL myoutput.htm for the body file and the URL mytoc.htm for the contents file.

  4. Submit the program and open the frame file myframe.htm in your browser. Use the links in the contents file to view the procedure output in the body file.
  1. Submit the following program, replacing <my-location> with your directory path or storage location: 
    ods listing close;
ods html body='<my-location>myoutput.htm'
         (url='myoutput.htm')
         contents='<my-location>mytoc.htm'
         (url='mytoc.htm')
         frame='<my-location>myframe.htm';
proc print data=sasuser.admit;
run;
proc print data=sasuser.insure;
run;
ods html close;
ods listing;
  2. In your browser, open the frame file, myframe.htm, to view both the body file and the contents file. Select the links in the contents to view the output for each procedure.
