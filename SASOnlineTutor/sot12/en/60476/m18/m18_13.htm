  1. Write a SAS program to

    • close the Listing destination.

    • create HTML output using the PRINT procedure and the SAS data set Sasuser.Insure.

    • save the HTML output as billing1.htm in your My SAS Files directory (or your default storage location).

      In the CMS operating environment: specify billing1 htm as the HTML filename and filetype.

    • close the HTML destination and reopen the Listing destination.

  2. Submit the program and view the output.
  1. Submit the following program, replacing <my-location> with your directory path: 
    ods listing close;
ods html body='<my-location>billing1.htm';
proc print data=sasuser.insure;
run;
ods html close;
ods listing;
  2. Using Windows Explorer, navigate to the location where you stored billing1.htm and double-click the file to open it in your browser.

run;

  2. Select the link to the output in the Results window. Or, from the internal browser or your preferred Web browser, select FileOpen. Then open the body file, billing1.htm, to view the output.

    If you are running SAS software in an operating environment that doesn\'t have a Web browser, you can transfer the HTML file to the operating environment where your browser is located. Then select FileOpen in your browser and select the body file to view it.
