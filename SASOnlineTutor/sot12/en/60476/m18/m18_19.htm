Create HTML output with a linked table of contents using the SAS data sets Sasuser.Admit and Sasuser.Insure.

  1. Copy the following program and paste it into the code editing window:
    ods listing close;
ods html 
    body='<my-location>myoutput.htm';
proc print data=sasuser.admit;
run;
proc print data=sasuser.insure;
run;
ods html close;
ods listing;
  2. Replace <my-location> with the path to any location on your computer.

  3. Add specifications to the ODS HTML statement to save a contents file as mytoc.htm and a frame file as myframe.htm. Save these files in the same location as myoutput.htm.

  4. Submit the program and open the frame file, myframe.htm. Use the links in the contents file to view the procedure output in the body file.

  2. Replace <my-location> with your My SAS Files directory (or your default storage location).

  3. Add specifications to the ODS HTML statement to save a contents file as mytoc.htm and a frame file as myframe.htm. Save these files in the same location as myoutput.htm.

    In the CMS operating environment: specify myoutput htm, mytoc htm and myframe htm as the HTML filenames and filetypes.

    In the z/OS (OS/390) operating environment: if you store HTML files in a PDSE, specify myoutput, mytoc, and myframe as the member names.

  4. Submit the program and open the frame file, myframe.htm. Use the links in the contents file to view the procedure output in the body file.
  1. Submit the following program, replacing <my-location> with your directory path or storage location: 
    ods listing close;
ods html body='<my-location>myoutput.htm'
         contents='<my-location>mytoc.htm' 
         frame='<my-location>myframe.htm';
proc print data=sasuser.admit;
run;
proc print data=sasuser.insure;
run;
ods html close;
ods listing;
  2. In your browser, open myframe.htm, which includes both the body file and the contents file. Click the links in the contents to view the output for each procedure.
