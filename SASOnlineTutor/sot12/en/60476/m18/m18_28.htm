Apply a new style to HTML output.

After you complete this practice, you must reset options so that SAS Enterprise Guide automatically creates HTML output.

  1. Copy the following program and paste it into the code editing window:
    ods listing close;
ods html body='<my-location>mystyle.htm'; 
proc print data=sasuser.admit;
run;
proc print data=sasuser.insure;
run;
ods html close;
ods listing;
  2. Replace <my-location> with the path to a location on your computer.

  3. Add an option to apply the Beige style to the HTML output.

  4. Submit the program and view the results.

run;

  2. Replace <my-location> with your My SAS Files directory (or your default storage location).

    In the CMS operating environment: specify mystyle htm as the HTML filename and filetype for the body file.

  3. Add an option to apply the Beige style to the HTML output.

  4. Submit the program and view the results.
  1. Submit the following program, replacing <my-location> with your directory path or storage location: 
     ods listing close;
 ods html body='<my-location>mystyle.htm' 
    style=beige;
 proc print data=sasuser.admit;
 run;
 proc print data=sasuser.insure;
 run;
 ods html close;
 ods listing;
  2. Open mystyle.htm to view the output.
