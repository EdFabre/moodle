1.  

 Which statement below adds a second title line to a graphics report? The second title line should display the text Houses by Style and the text should be green.

a. title 'Houses by Style' color=green;
b. title2 'Houses by Style' color=green;
c. title2 c=green 'Houses by Style';
d. either b or c


You can specify options for color, height, and font with TITLE statements. In order to specify color, you must place the C= (or COLOR=) option before the text to be modified.  

2.  

 Which statement is false regarding the use of TITLE or FOOTNOTE statements in SAS/GRAPH output?

a. Space for the title and footnotes is allocated first, which reduces the area that is available for the chart or plot.
b. Options that specify the color, font, and height of TITLE and FOOTNOTE text are placed before the text.
c. The default color, font, and height is different for the text in the first title line than for the text in all other title and footnote lines.
d. TITLE and FOOTNOTE statements that are placed within a PROC step affect only the titles and footnotes for that particular report.


TITLE and FOOTNOTE statements remain in effect until you cancel them, re-specify them, or end the SAS session. 


3.  

 Which statement below enables you to replace variable names with descriptive text on your graph?

a. TITLE
b. LABEL
c. FORMAT
d. VALUE


A LABEL statement is used to assign descriptive text to the variables in your output. 


4.  

 Which HBAR statement produces the graph shown here?


a. 
hbar sex / sumvar=sysbp type=mean
     meanlabel='Average Systolic Blood Pressure'; 
 
b. 
hbar sex / group=sysbp patternid=group mean
     meanlabel='Average Systolic Blood Pressure';
 
c. 
hbar sex / sumvar=sysbp type=mean mean
     meanlabel='Average Systolic Blood Pressure';
 
d. 
hbar sex / var=sysbp mean
     meanlabel="Average Systolic Blood Pressure';


Use the SUMVAR= option and the TYPE= option to specify the statistic and variable that you want the statistic to be applied to. Use the MEAN and MEANLABEL= options to specify that you want only the MEAN statistic and the label for that statistic to be displayed. Note that if you specify the TYPE=MEAN option but do not include the MEAN statistic option, the default FREQ statistic will also be displayed along with the MEAN statistic on the right side of the bars. 


5.  

 Which statement would you use to display the value 10000 as $10,000.00 in a graph?

a. LABEL
b. FORMAT
c. DOLLAR9.2
d. INFORMAT


The FORMAT statement has the same effect in SAS/GRAPH output as in other procedure output. The FORMAT statement enables you to display a numeric value in a variety of formatted styles.  

6.  

 Which of the following statements is true for both SAS/GRAPH horizontal bar charts and vertical bar charts?

a. The default output shows only the statistic FREQ.
b. Statistics can be displayed both within and outside the bars.
c. A user-defined label can be applied to a statistic.
d. The TYPE= option must be used to specify the statistic to be calculated.


You can display a user-defined text label instead of the default statistical label by using the statistic-label= option for a horizontal bar chart and by using the AXIS statement and the RAXIS= option for a vertical bar chart. 


7.  

 It would be a good choice to export your SAS/GRAPH output interactively if which of the following were true?

a. You had a large number of graphs to export with a large amount of output.
b. You had only two graphs and you wanted to export them quickly.
c. You wanted to specify exactly the device driver that you needed.
d. You had several graphs that you needed to export to a variety of file formats.


The interactive method for exporting graphs is most useful for quickly exporting a few graphs. It is best to use programming statements to export graphs if you have large quantities of output. 


8.  

 Which of the following statements is true about the SAS/GRAPH Control for ActiveX?

a. You must specify DEVICE=ACTIVEX in a GOPTIONS statement to create interactive graphs with the control.
b. You can use the control to create bar charts, pie charts, plots, and maps.
c. The control is available in the Windows NT operating environment, or in Windows 95 or later.
d. all of the above


Creating interactive graphs is similar to creating graphs that are embedded in HTML pages, except that you specify the ACTIVEX device driver. You can create interactive graphs with the GCHART, GCONTOUR, GMAP, GPLOT, and G3D procedures in SAS/GRAPH. The control is available only in Windows operating environments. 


9.  

 Which of the following is false regarding the sizing of graphs?

a. You can specify the size of the graph.
b. When you size a graph, the display area and the graphics output area might not be the same size.
c. The way you create your graph determines which options you use for sizing your graph.
d. You can specify the width of the border that encloses the graph.


The BORDER option is used to create a border that encloses the graph, but you cannot specify a border width. 


10.  

 Which of the following can you do with SAS/GRAPH software?

a. create custom fonts and legends
b. display data or geographic maps
c. angle title and footnotes
d. all of the above


With SAS/GRAPH software, you can create custom fonts and legends, rotate title and footnotes, display data or geographic maps using GMAP, and much more.
