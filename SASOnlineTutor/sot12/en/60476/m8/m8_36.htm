Create and work with an interactive graph.

  1. Copy the program below and paste it into the code editing window. Then submit the program and view the interactive graph. 
    ods listing close;
ods html body='practice.htm';
goptions reset=goptions device=activex;
title 'Flights by Destination';
proc gchart data=sasuser.europe;
   hbar dest / sumvar=revenue;
run;
quit;
ods html close;
  2. Change the bar chart to a vertical bar chart.

    If you see your browser's pop-up menu when you right-click, your cursor is positioned outside the invisible boundary that separates the graph area from the rest of the page. Reposition your cursor closer to the graph and try again.

  3. Change the bar shape to a cylinder. Then change the color scheme for the graph to Terra, and change the graph depth to 3D Perspective.

  4. Display Help for the interactive graph. Select the link for information about using the Graph Toolbar. Scroll to find information about using the Subset Data tool and the Rotate tool.

  5. Using the Graph Toolbar, subset the graph to display only the bars for London (LON) and Paris (PAR). Then rotate the graph.

  6. Reset the graph to its original state (removing the subset and the rotation).

  7. Save the graph as an image named SASGraphSaved.jpg.
