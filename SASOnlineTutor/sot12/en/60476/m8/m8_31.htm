Creating Interactive Graphs

So far, you've created charts and plots that are static: once you create the graph, you must submit additional programming statements in order to change it. But suppose you want to explore a graph by interactively changing features such as

  • the chart type
  • the variables and statistics that are displayed
  • properties such as color, axes, labels, and legends.

The SAS/GRAPH control for ActiveX enables you to embed interactive graphs in Web pages and in OLE documents (in Microsoft Office products). When the graph is displayed, you can point and click to rotate, change, or further investigate the graph.





Using the SAS/GRAPH control for ActiveX, you can create bar charts, pie charts, surface plots, box-and-whisker plots, bubble plots, contour plots, high-low plots, line plots, regression plots, scatter plots, standard deviation plots, surface plots, and maps.

To view graphs that use the SAS/GRAPH control for ActiveX on the Web, you need an ActiveX-enabled browser such as Internet Explorer 4.0+.

The SAS/GRAPH control for ActiveX is available in the Windows NT operating environment, or in Windows 95 or later.
