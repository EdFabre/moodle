Assigning Descriptive Labels
You can use the LABEL statement to assign descriptive text to the variables in your SAS/GRAPH output.

Formatting Data Values
You can control how values are displayed by using a FORMAT statement.

Specifying Titles and Footnotes
To specify titles and footnotes for your SAS/GRAPH output, you can use TITLE and FOOTNOTE statements. You can control the appearance of titles and footnotes by using the COLOR=, FONT=, and HEIGHT= options to specify the color, height, and font for text. Titles and footnotes have default values for the COLOR=, FONT=, and HEIGHT= options.

Controlling HBAR Statistics
You can control which statistics are displayed on the right side of the bars on a horizontal bar chart. You can also use the FREQLABEL=, CFREQLABEL=, PERCENTLABEL=, CPERCENTLABEL=, SUMLABEL=, and MEANLABEL= options to assign descriptive labels to the statistics.

Controlling VBAR Statistics
By default, only one statistic is displayed for vertical bar charts. The OUTSIDE= option enables you to display a statistic above the bars in a vertical bar chart. The INSIDE= option enables you to display a statistic within the bars.

Exporting Graphs
You can export graphs that you have created in SAS to a wide variety of graphics formats, either interactively or by using program statements.

Sizing Graphs
You can size graphs by using either the HSIZE= and VSIZE= options or the XPIXELS= and YPIXELS= options, depending on whether you create your graph in the GRAPH window or export your graphs directly to external files by using program statements. You can also add a border to graphs by using the BORDER option.

Creating Interactive Graphs
In the Windows NT operating environment, or in Windows 95 or later, you can use the SAS/GRAPH control for ActiveX to embed interactive graphs in Web pages and in OLE documents (in Microsoft Office products). You can use the ActiveX control with the GCHART, GCONTOUR, GMAP, GPLOT, and G3D procedures in SAS/GRAPH. To create interactive graphs, you can submit ODS statements, a GOPTIONS statement that specifies DEVICE=ACTIVEX as the device driver for interactive graphs, and appropriate SAS/GRAPH procedure statements. When the graph is displayed, you can point and click to rotate, change, or further investigate the graph.

Additional Features
You can use several options within the GCHART and GPLOT procedures to further enhance your graphs.

                        YPIXELS=height-in-pixels>;
TITLE<n>  <COLOR=color> 
                    <HEIGHT=height-value <units>>   
                    <FONT=font-name>
                     'title-text' | "title-text";
FOOTNOTE<n>  <COLOR=color>   
                              <HEIGHT=height-value <units>>  
                              <FONT=font-name>  
                              'footnote-text' | "footnote-text";
PROC GCHART <DATA=SAS-data-set>;
         HBAR chart-variable </ statistic
                     FREQLABEL= | SUMLABEL= |
                     PERCENTLABEL= | MEANLABEL= 'label-text'>;
         VBAR chart-variable </
                     OUTSIDE=statistic INSIDE=statistic>;
         FORMAT variable(s) format;
         LABEL variable='label-text' <...variable<n>='label-text<n>';
RUN;
QUIT;
ODS HTML CLOSE;


Sample Programs 
     filename bargif1 'g:\images\barchart1.gif';
     goptions gsfname=bargif1 device=gif
              xpixels=500 ypixels=400;
     title1 color=red height=0.5 cm
            'Total Fees by Activity Level, Males';
     footnote1 height=0.5 cm color=blue
               font=centx 'Current Month';
     proc gchart data=clinic.admit;
        hbar actlevel / sumvar=fee patternid=midpoint
             sum sumlabel='Fees by Level';
        where sex='M';
        label actlevel='Activity Level';
        format fee dollar5.;
     run;
     goptions hsize=4in vsize=4in;
     title1 color=red height=0.5 cm
            'No. of Clients by Activity Level';
     footnote1 height=0.5 cm color=blue font=centx 
               'YTD FY1998';
     proc gchart data=clinic.admit;
        vbar sex / group=actlevel patternid=midpoint
             outside=cfreq inside=freq;
     run;

     ods listing close;
     ods html body='barchart.htm';
     goptions device=activex xpixels=400 ypixels=300;
     proc gchart data=clinic.heart;
        hbar3d shock / sumvar=arterial type=mean;
     run;
     quit;
     ods html close;


Points to Remember
  • FILENAME, GOPTIONS, TITLE, and FOOTNOTE statements are global and remain in effect until you cancel them, modify them, or end your SAS session.

  • Be sure to match quotation marks that enclose text in TITLE, FOOTNOTE, LABEL, and FORMAT statements.

  • If the statistics that are displayed for VBAR charts are wider than the bars, the statistics might not display properly.

  • To end a RUN-group procedure, you must submit another PROC step, a DATA step, or a QUIT statement.

  • The SAS/GRAPH control for ActiveX is available in the Windows NT operating environment, or in Windows 95 or later. To view graphs that use the SAS/GRAPH control for ActiveX on the Web, you need an ActiveX-enabled browser such as Internet Explorer 4.0+.
