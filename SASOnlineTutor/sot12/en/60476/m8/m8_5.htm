Formatting Data Values (continued)
Write a statement to display values of the variable Total using the format COMMA10.2 and to display the variable BalanceDue using the format DOLLAR10.2. 

      proc gchart data=clinic.insure;
         vbar company / type=sum sumvar=balancedue 
              patternid=midpoint;
         hbar company / type=sum sumvar=total 
              patternid=midpoint;
         label balancedue='BALANCE DUE' 
               total='TOTAL BALANCE';
         
      run;
