Sizing Graphs

When you create charts and plots, you might want to specify the size of the graph and even add a border around the graph. To do so, you specify options for controlling the height, width, and border in the GOPTIONS statement.


Sizing the Graph

The way you create the graph determines how you size it:

  • If you create your graph in the GRAPH window, use the HSIZE= and VSIZE= options.

  • If you export your graphs directly to external files by using program statements, use the XPIXELS= and YPIXELS= options.
