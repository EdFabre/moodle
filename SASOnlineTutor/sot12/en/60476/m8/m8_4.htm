Formatting Data Values

As with LABEL statements, FORMAT statements have the same effect in SAS/GRAPH output as in other procedure output. The FORMAT statement below specifies the DOLLAR5. format, which displays values of BalanceDue with a dollar sign ($), a comma (,), and no decimal places to a width of five positions.

     proc gchart data=clinic.insure;
        vbar company / type=sum sumvar=balancedue
             patternid=midpoint;
        where company in ('A&R','ACME','RURITAN');
        label balancedue='Balance Due'
              company='Insurance Co. Billed';
        format balancedue dollar5.; 
     run;
