Creating Interactive Graphs (continued)

Changing the Form of the Graph

In addition to modifying the appearance of interactive graphs, you can alter the graph by changing

  • the chart type
  • the data that is displayed in the chart.

For example, to change the bar chart to a pie chart that displays the same data, you can right-click the bar chart and select Chart TypePie from the menu. Now each bar from the bar chart becomes a slice in the pie.






To change the variables and statistics that are displayed in the graph, you can right-click the chart and select Data Options from the menu.

For example, you can remove the response variable Arterial from the chart and show only a frequency for categories of Shock. Now the pie chart looks like this:





To subset the data in the graph by displaying only selected pie slices, you can right-click the chart and select Graph Toolbar from the menu. Then you use the Subset Data tool to select the pie slices that you want to display. (When your mouse pointer hovers over the slices, "tool tips" identify the slices.)

To learn more about working with interactive graphs, you can right-click the graph and select HelpGraph Control Help from the menu.
