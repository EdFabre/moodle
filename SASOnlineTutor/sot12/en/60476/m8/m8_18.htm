Using Sasuser.Insure, use RUN-group processing to create charts.

  1. Copy the following program and paste it into the code editing window. 
    proc gchart data=sasuser.insure;
   vbar company / sumvar=total
        patternid=midpoint;
   where total>570;
run;
  2. Submit the program and view the output.

  3. Submit these statements and view the additional charts:

    • an HBAR statement that requests only the statistic PERCENT for the chart variable Company, with PERCENT labeled Percentage of Total

    • a VBAR statement that requests the statistic CFREQ outside the bars for the chart variable Company

    • a RUN statement.

  4. View the output. Then submit a statement to end the GCHART procedure.
