Export SAS/GRAPH output interactively.

Complete this practice only if you are using SAS in a desktop operating environment.

  1. Copy the program below and paste it into the code editing window. Then submit the program and view the plot. 
    proc gplot data=sasuser.totals2000;
   plot treadmill*month / areas=2;
   symbol i=spline;
run;
quit;
  2. Export the plot interactively to a GIF file named Myplot.gif in your default storage location.

  3. Verify that the GIF file was created.
