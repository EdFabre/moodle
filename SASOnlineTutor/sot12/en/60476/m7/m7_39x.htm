1.  

 Which SAS statement below invokes a SAS/GRAPH procedure to produce a chart, using a SAS data set named Sales1999 in a SAS data library to which the libref Stats has been assigned?

a. proc gchart data=stats.sales1999;
b. proc chart data=stats.sales1999;
c. proc chart data=sales1999.stats;
d. proc gchart data=sales1999.stats;


The PROC GCHART statement invokes the GCHART procedure, which you can use to produce horizontal bar charts, vertical bar charts, or pie charts. To reference a file, you use a two-level name. The first part of the name is the libref of the SAS data library in which the file is stored, and the second part is the name of the file itself. 

2.  

 Which statement would you add to the following program to produce a horizontal bar chart that calculates the average of the variable Profits for each value of the variable Style? 
proc chart data=houses.sales;
   ???
footnote1 '1999 New House Profits';
run;

a. hbar profits / var=style stat=mean;
b. hbar style / sumvar=profits type=mean;
c. hbar style / sum=profits type=avg;
d. hbar style / var=profits stat=mean;


In the HBAR statement, the chart variable is Style. To specify the average of the values for the variable Profits for each value of Style, you use the options SUMVAR=PROFITS and TYPE=MEAN. 


3.  

 Which statement in the following program contains an error? 
proc gchart data=crimes.misdmnr;
   pie region / type=sum;
   where robbery gt 300;
title2 'Crimes by Region';
run;

a. PROC GCHART
b. PIE
c. WHERE
d. TITLE


The SUMVAR= option is missing in the PIE statement. In order to use the TYPE= option with a value of SUM, you must specify an analysis variable. 


4.  

 Which statement would you add to the program below to create a vertical bar chart that displays the sum of the variable Balance for each value of the variable Account? 
proc gchart data=store1.ledger;
   ???
   where status='PAST DUE';
run;

a. vbar account / sumvar=balance;
b. vbar account / type=sum;
c. vbar account / sumvar=balance type=sum;
d. either a or c


In the VBAR statement, the chart variable is Account, the analysis variable is Balance, and the statistic is SUM. In this case, specifying the statistic is optional because SUM is the default. 


5.  

 Which statistic is displayed in the bar chart that the following program produces? 
proc gchart data=houses.rentals;
   vbar style;
run;

a. FREQ
b. SUM
c. PCT
d. There is no default statistic.


If you specify only a chart variable and not an analysis variable, the default statistic is FREQ (frequency).  

6.  

 If you are using RUN-group processing in interactive mode, which of the following methods would NOT end the PROC GCHART procedure?

a. submitting a DATA step
b. submitting another RUN statement
c. submitting a QUIT statement
d. submitting another PROC step


Submitting another RUN statement would simply submit the code between the previous RUN statement and before the current RUN statement. In this case, there is no code other than the RUN statement to submit. Only a DATA step, a PROC step, or a QUIT statement ends the current RUN group.  


7.  

 In the PROC GCHART step below, which statement or statements contain an error? 
proc gchart data=sales.houses;
   pie style / type=cfreq;
   vbar style / sumvar=saleprice type=percent;
   hbar style / sumvar=saleprice type=mean;
run;

a. PIE statement
b. VBAR statement
c. HBAR statement
d. both a and b


The values CFREQ and CPERCENT are not available for pie charts. The VBAR statement specifies SUMVAR= but requests a statistic other than SUM or MEAN. 


8.  

 To view a permanently stored chart in a subsequent SAS session, which of the following windows can you use?

a. the GRAPH window
b. the Results window
c. the Explorer window
d. both a and c


You can use all of these windows to view graphs that have been created during your SAS session. However, you must use the Explorer window or the GRAPH window to view graphs that were stored in previous SAS sessions.  


9.  

 Which of the following statements is false?

a. The PATTERNID=MIDPOINT option is used to vary the colors and patterns for the bars in bar charts.
b. The FILL= option is used to change the fill pattern in PIE charts.
c. If you use the FILL=S option and there are fewer colors in the color list than there are pie slices, the cross-hatch pattern will be used for some slices.
d. The default fill pattern for horizontal and vertical bar charts specifies that all bars will be the same solid color.


If you use the FILL=S option and run out of colors in the available color list, the fill pattern for the remaining slices remains solid, but the color cycles through the color list again. 


10.  

 Which statement is true for both pie charts and bar charts?

a. Character chart variables create one slice or bar for each variable value.
b. Numeric chart variables create one slice or bar for each variable value.
c. You can use the FILL= option to specify the fill pattern for bars and slices.
d. You must use the TYPE= option if the chart variable is numeric.


For character chart variables, SAS/GRAPH software creates one bar or slice for each unique value.
