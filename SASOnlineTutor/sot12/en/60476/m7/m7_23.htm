For the GCHART step below, write a statement that selects observations for which the value of MaxHR (maximum heart rate) minus the value of RestHR (resting heart rate) is greater than 100. 

     proc gchart data=clinic.stress2;
        hbar timemin / sumvar=rechr type=mean;
        pie tolerance / sumvar=timemin;
        
     run;
