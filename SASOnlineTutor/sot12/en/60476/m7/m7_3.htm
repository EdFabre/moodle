Create three charts, using the Sasuser.Admit data set.

  1. Copy the program below and paste it into the code editing window. 
    proc gchart data=sasuser.admit;
   hbar sex;
   vbar age;
   pie actlevel;
run;
  2. Submit the program.

  3. View the output, noticing the variables that are charted and the statistics that are calculated. Notice that the program requested three different charts from the same data, although the procedure was submitted only once.

    The title bar of the code editing window displays the message PROC GCHART running. Unlike most other procedures, the GCHART procedure continues to run until you submit another PROC step, a DATA step, or a QUIT statement. This feature is called RUN-group processing. RUN-group processing is discussed in the lesson . You can submit a QUIT statement now, or you can let PROC GCHART continue running.
