Specifying Statistics (continued)

Additional TYPE= Examples

The GCHART procedure below uses the TYPE= option to specify the statistic PERCENT. The bars now display the percentage for Company. Notice that the label on the vertical axis is now PERCENT. 

     proc gchart data=clinic.insure;
        vbar company / type=percent;
     run;

This GCHART procedure specifies TYPE=CPERCENT to display the cumulative percentage for Company. The label on the vertical axis is now CUMULATIVE PERCENT. 
     proc gchart data=clinic.insure;
        vbar company / type=cpercent;
     run;
