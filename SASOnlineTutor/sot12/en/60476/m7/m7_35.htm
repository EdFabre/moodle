Specify an output catalog and view graphs.

  1. Copy the program below and paste it into the code editing window. 
    proc gchart data=sasuser.stress2; 
   pie tolerance / fill=s; 
   vbar tolerance / sumvar=resthr
        type=mean;
run;
quit;
  2. Add an option to specify Sasuser.Mygraphs as the output catalog for your graphs.

  3. Submit the program and view the pie chart. Notice the title of the GRAPH window.

  4. In the GRAPH window, scroll down to view the vertical bar chart. Notice the title of the GRAPH window.

  5. Using the Results window, redisplay the pie chart.

  6. Open the Sasuser.Mygraphs catalog and display the Gchart1 entry.
