Working with Columns in a Query (continued) Add a column to compute total dollar amount of each product on order.
  1. On the Select and Sort tab, click New.


  2. In the Alias box, type Total Ordered.

  3. Click the Expression tab. For the Total Ordered column, you want the product of the UnitPrice and UnitsOnOrder columns. Click the Values tab below the row of operator buttons.

  4. In the Data Source box, be sure that Products is selected. In the Value box, double-click UnitPrice.

  5. Click the multiplication operator .

  6. In the Value box, double-click UnitsOnOrder.


Click to view the full window.


  1. In the New window, click the Format tab.

  2. In the Categories box, select Currency.

  3. In the Formats box, select DOLLARw.d.

  4. Click OK to close the New window and create the computed column.
