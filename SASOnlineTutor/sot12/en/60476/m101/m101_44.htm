Creating Queries That Accept Parameters (continued) Set options to prompt the user each time the query runs.
  1. Select ToolsOptions. Click the Data and Query tab.


  2. In the Query preferences area, select For queries with parameters, always prompt for values.


Click to view the full window.


  1. Click OK.

  2. In the Project window, right-click the query and select Run this Query. Because you set the option, you are prompted to select a category. Select any category from the parameter drop-down list and click OK.

  3. When prompted to replace results, click Yes. View the new results.
