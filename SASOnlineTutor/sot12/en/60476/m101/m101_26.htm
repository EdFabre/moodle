Join the Listings and ListingAgent tables so that the listing agent is included in the query results.

This practice uses the Practice1 query that was created in a previous practice. If you haven't created this query, return to the practice and create it before continuing here.

  1. Reopen Practice1.

    • In the Project window, double-click Practice1 to open the query.
  2. Add the ListingAgent table. What column is used to join the tables?

    • Click Add Data. Navigate to the directory where you stored the downloaded data. Double-click ListingAgent to add it to the query.
    • Click the Tables tab, then click the join symbol. The PROPID column is used to join the tables.

  3. Display the following columns (in order) in the query results: AGENT, STYLE, SQFEET, PRICE.

    • Click the Select and Sort tab.
    • Select AGENT and click Move Up until the column is at the top of the list.
    • Select the PricePerSqFoot and PROPID columns and click Delete.
  4. Sort the columns by ascending values of AGENT.

    • Click the sort box to the right of the Agent column and select Ascending from the drop-down list.
  5. Run the query and do not replace the previous results. Which agent has the greatest number of listings?

    • Click Run Query. When you are prompted to overwrite the results, click No.
    • Examine the results. Joe Schnell has the greatest number of listings.
  6. Rename the query Practice3.

    • In the Project window, rename this query Practice3.
