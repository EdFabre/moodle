Additional Features

At this point, you've finished your queries for your specialty foods shop. But as you continue to create queries, keep in mind the following features of SAS Enterprise Guide.


Computing and Displaying Values

In this lesson, you used values of existing columns to compute new columns and entered values to replace existing values. You can also take advantage of the large selection of SAS functions when you create columns or replace values.




To define the way data values are displayed in query results, you can use any of a number of SAS formats, including numeric, date, time, and currency formats. You can even define and apply your own custom formats.



Modifying SQL Code

As you saw earlier in this lesson, if you are familiar with SQL, you can modify the SQL code that the Query Builder generates.




When you modify code that was generated by a task, a separate code node is created in the project tree. This new code node contains a copy of the task code and is no longer associated with the original task. Editing the code does not affect the task, and modifying the original task does not affect the contents of the new code node. You can run the modified code in SAS Enterprise Guide, or you can save it as a SAS program.

For more information about functions, formats, and working with SQL code, select HelpEnterprise Guide Help in SAS Enterprise Guide.
