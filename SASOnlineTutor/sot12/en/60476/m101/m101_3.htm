Using the Query Builder

Before you create your first query for the specialty foods shop, let's look at SAS Enterprise Guide's Query Builder, which enables you to extract data from one or more tables according to criteria that you specify. You'll learn how to use the Query Builder later in this lesson. For now, let's explore the Query Builder's capabilities.


Example: Overview of the Query Builder

Select buttons below to view some of the tasks you can perform using the Query Builder. You can also see a text-only version of the information.



The Query Builder and SQL Code

You don't need to know anything about Structured Query Language (SQL) to use the Query Builder—as you work in the Query Builder, it generates SQL code automatically. For example, when you select rows of data on the Filter Data tab, the Query Builder creates a WHERE clause based on the information you specify.

You can preview the results of your query, and you can view and modify the SQL code that the Query Builder generates.
