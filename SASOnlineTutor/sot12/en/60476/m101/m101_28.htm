Joining Tables Manually and Replacing Values (continued) Begin a new query and manually join the Orders and Products tables.
  1. Select ToolsQueryCreate Empty.


  2. In the Query window, click Add Data.

  3. In the Add Data window, click the Existing tab if it is not already selected. Locate Suppliers$, select it, and click OK.

  4. Click Add Data again. Navigate up one level in the sample data directory. Change the Files of type drop-down box to SAS Data Files. While holding down the Ctrl key, select Products.sas7bdat and Orders.sd2. Click OK. Because there are no like-named columns in these two tables, a message window informs you that you must perform a manual join.

  5. In the list of columns for the Orders table, click PRODID and drag your cursor to the ProductID column in the Products table, then release. This joins the two tables.
