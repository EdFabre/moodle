Starting a Query

Now that you've selected HTML reports as your default result format, let's see how you create your first query for the specialty foods shop—a report on inventory levels for various products.

Your report will show information for each product, such as how much inventory is in stock and how much has been reordered. To see the report that you want to create, click the image below.



Click to view the full window.


Data for the Report

To create your report, you'll use the Products table. For each product that you sell in your specialty foods shop, this table stores

  • the product ID
  • the product name
  • the supplier ID
  • the category ID
  • the quantity per unit
  • the unit price
  • the number of units in stock
  • the number of units on order
  • the reorder level
  • whether or not the product has been discontinued.
