Joining Tables Manually and Replacing Values (continued) Replace values in the Country column.
  1. On the Select and Sort tab, select the Country column. Then, click Properties.


  2. In the Properties window, click the Replace Values tab.

  3. The Country column from the SupplyInfo(Suppliers$) data contains the names of countries where the suppliers are located. Click the drop-down box under Replace so you can view the values of the Country column. You need to replace USA with Domestic and all other country names with International. From the Replace drop-down box, select USA.

  4. In the Replace with box, type Domestic. Click Add to List.

    You must click Add to List to complete the definition; otherwise, you might generate syntax errors in the query.

  5. In the Replace all others with box, type International.

  6. Click OK.


Click to view the full window.
