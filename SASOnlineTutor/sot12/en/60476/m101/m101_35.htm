Running Tasks against Query Output (continued) Start the Summary Tables task using the Profit query.
  1. In the Project window, select the Profit query to make it active.

  2. Select AnalysisDescriptiveSummary Tables.

  3. Select the Profit column, then drag and drop it on the Analysis variables role. Analysis variables are the numeric variables on which you want to have statistics calculated.

  4. Select _Country and ProductName, then drag and drop them on the Classification variables role. Classification variables are character or discrete numeric variables that you use to divide the input data into categories. The statistics are calculated on all selected analysis variables for each unique combination of classification variables.

You must assign at least one variable to either the Analysis variables role or the Classification variables role.
