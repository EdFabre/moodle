Example: Overview of the Query Builder

some of the tasks you can perform using the Query Builder.

  1. In the Filter Data tab, you can select only rows of data that meet certain criteria, based on values of columns in the data.

  2. In the Select and Sort tab, you can select and order columns, specify aliases for columns, and sort the output from your query. You can also classify your data into groups here.

  3. The Tables tab displays the tables that you have selected for your query. You can also add tables here. If you combine data from different sources, you can specify the type of join to use.

  4. If you group data in the Select and Sort tab, you can use the Group Filters tab to subset the grouped data based on specified conditions.

  5. In the Advanced tab, you can select the server on which you want to execute the query. By default, the query executes on the server where the first table in the query resides.

  6. In the Parameters tab, you can define parameters for your query. When the query is run, it prompts the user to select or type in values for the parameter.
