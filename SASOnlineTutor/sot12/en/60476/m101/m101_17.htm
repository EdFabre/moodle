Subsetting Data in a Query (continued) Modify the Product Inventory query to add a filter.
  1. In the Project window, double-click Product Inventory to reopen the query. The Filter Data tab is selected by default.

  2. In the list of columns for the Products table, click UnitsInStock and drag it to the filter area and drop it there. The Edit Filter Condition window opens.

  3. First you want to filter records that have UnitsInStock less than or equal to the ReorderLevel. Click the Operator drop-down arrow and select <=, less than or equal to.

  4. In the Filter constants area, click Column Names. Then, select Products.ReorderLevel.


Click to view the full window.


  1. Click OK.

  2. Next, you want to filter records that have UnitsOnOrder equal to 0. Drag the UnitsOnOrder column to the end of the first filter condition. Drop the column when the cursor changes to AND. The Edit Filter Condition window opens.

  3. Click the Operator drop-down list and select =, equals. In the Value box, type 0.


Click to view the full window.


  1. Click OK. Now you've created the two conditions that each record must meet in order to appear in the query results.


Click to view the full window.
