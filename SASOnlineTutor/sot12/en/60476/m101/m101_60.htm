Using Outer Joins (continued) Delete columns from the query results. Then preview the results.
  1. Click the Select and Sort tab.

  2. Delete all columns from the results except
    • CategoryID and UnitsInStock from the Products table
    • CategoryName and CategoryID (CategoryID2 in the list) from the ProdCategories table.

  3. Click the function box to the left of CategoryName and select <group> from the drop-down list.

  4. Click the function box to the left of UnitsInStock and select SUM from the drop-down list.

  5. Clear the Allow duplicate rows check box.
