Specify underlining and a blank line following column headings in a report of Sasuser.Diabetes.

  1. Copy and paste the following program into the code editing window: 
    proc report data=sasuser.diabetes nowd;
   column id sex weight fastgluc postgluc;
   where age>40;
   define weight / format=comma6.2 
                   spacing=4;
   define sex / width=7 spacing=4 center
                'Sex of/Patient';
   define fastgluc / 'Fasting/Glucose';
   define postgluc / width=12
                     'Postprandial/Glucose';
run;
  2. Add options to specify an underline for all column headings and a blank line beneath all column headings.

  3. Submit the program and view the report.
