Assign a width to a column in a report of Sasuser.Diabetes.

  1. Copy and paste the following program into the code editing window: 
    proc report data=sasuser.diabetes nowd; 
   column id sex weight fastgluc 
               postgluc;
   where age>40;
   define weight / format=comma6.2;
run;
  2. Submit the program and view the heading for the column Sex.

  3. For the column Sex, specify a width that is large enough so that the column heading appears on one line.

  4. Re-submit the program and view the report.
