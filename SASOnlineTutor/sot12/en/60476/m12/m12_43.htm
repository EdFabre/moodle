Write a program to write the observations in the SAS data set Sasuser.Stress2 to a raw data file.

  1. Write a program to write the observations in the SAS data set Sasuser.Stress2 to a raw data file named Stresdat. Save Stresdat to a location in your operating environment, specifying a path and file extension as appropriate. Specify one blank space between each field.

    Assume that the sample data reflects the actual length of the numeric variable values.

  2. Submit the program.

  3. Check the log to verify that the program ran correctly. The Log window should indicate that 21 records were written from Sasuser.Stress2 to Stresdat.

  4. Using Windows Explorer, open Stresdat in a text editor such as Notepad.

    To browse this file using SAS Enterprise Guide, select FileOpen. Navigate to the location where you saved the file. In the Open As drop-down list, select Note.

  4. Use one of the following methods to browse the contents of Stresdat.

    • Assign a File Shortcut (or fileref) to the file. Then open it from the SAS Explorer.

    • Drag and drop the file into the code editing window.

    • Open the file using a text editor.
Submit the following program, replacing filename with a fully qualified filename for Stresdat: 
 data _null_;
    set sasuser.stress2;
   file 'filename';
    put id 1-4 name 6-25 resthr 27-28 maxhr 30-32
        rechr 34-36 timemin 38-39 timesec 41-42
        tolerance 44;
 run;
 The filename specified in the FILE statement will vary based on your operating environment and the storage location you specified.
