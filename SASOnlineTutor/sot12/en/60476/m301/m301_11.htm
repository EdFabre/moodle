Writing and Submitting SAS Code (continued)

Viewing Log Messages

After a task or program runs, an item for the log is added in the Project window. You'll recognize the SAS log—it displays the same system messages that it does in the SAS windowing environment. The difference in SAS Enterprise Guide is that a new log is generated each time you run the code, so a separate log is attached to each program or task that you run.

You open the Log window by double-clicking the log item in the Project window. To find log messages that relate to your code, scroll through the Log window until you see the beginning of your code.



Click to view the full window.


You probably noticed that SAS processed additional code along with the code that you submitted. SAS Enterprise Guide automatically submits programming statements and macros with all programs that you submit. This code sets options, creates HTML output, and adds QUIT and RUN statements to the end of your code.
