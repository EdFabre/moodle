Working with Data

In your SAS programs, you access data by using programming statements. Before you can view data or use it as input to a point-and-click task or query in SAS Enterprise Guide, you must add it to your project. You can access data on both your local machine and remote servers. Most data files on a remote server are added to a project using the Files or Libraries items in the Server List.

After you have added data to your project, you can view the data properties by right-clicking the data in the Project window and selecting Properties from the pop-up menu.

 


Adding SAS Data Sets

You can add an existing SAS data set by doing either of the following:

  • Use the Server List to open the server and then add SAS data sets from the Libraries or Files folders.

  • Select InsertData and click the Existing tab to locate and add SAS data sets.

When you add data to your project, a shortcut to the data is added to the Project window and the data opens in the data grid in read-only mode.





The symbol next to each variable name in the data grid indicates the data type of the variable:


Symbol
Data Type
character data
numeric data
date data
time data
currency data


Date, time, and currency data are all numeric data. Using these data types enables SAS Enterprise Guide to validate the values and display them in a particular format.

Keep these points in mind when you are working with data in SAS Enterprise Guide: 

  • You can edit a SAS data set in the data grid by selecting DataProtected to change to update mode. Any changes that you make to the data are applied directly to the data file.

  • If you add a SAS data set that is not in an active SAS library, SAS Enterprise Guide submits a LIBNAME statement for you.

  • When you run a task on data, it is most efficient if the data is located on the same server where the program is running. If the data is remote, SAS Enterprise Guide copies the data to the server where the code will run. Copying data over a network can be time and resource intensive.

    2. In the selection pane, click Data General.
    3. Clear the Automatically open data when added to project checkbox.
