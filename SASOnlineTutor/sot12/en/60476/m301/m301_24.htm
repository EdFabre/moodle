Select formats and styles for the ProdSales results.

This practice uses programs that were created in a previous practice. If you haven't created these programs, return to the practice and complete it before continuing here.

  1. Create HTML and PDF results from the ProdSales code.

    • In the Project window, right-click ProdSales and select Properties.
    • Click the Results tab.
    • Select the Override the preferences set in ToolsOptions check box.
    • Select PDF. Leave HTML selected.
    • Click OK.
    • In the Project window, right-click ProdSales and select Run on <server-name>.
    • In the message window, click Yes.
    • View the PDF and HTML results.
  2. Apply the statdoc style to the HTML results from the ProdSales code. How would you set statdoc as the default style for all HTML results?

    • If it is not open already, open the HTML results under ProdSales.
    • Click the styles drop-down list in the menu bar and select statdoc.
    • To make statdoc the default style, you would select it on the Results tab in the Options window.
