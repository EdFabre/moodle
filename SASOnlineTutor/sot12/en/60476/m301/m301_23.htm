Specifying Styles for HTML Output

You can change the style of your HTML output by applying a predefined style or by creating and applying your own style. Styles are cascading style sheets (CSS) that SAS Enterprise Guide applies to the HTML results.

As with result formats, you can apply styles to individual results or select a style to apply to all results. To apply a style to particular HTML results, open the HTML results and select a style from the drop-down styles list in the menu bar.



The HTML results below have the SAS Web style applied.





Applying a style in this manner changes the style for this report only. To make a style the default style, you must select the style on the Results tab of the Options window.
