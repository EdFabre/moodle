Generating and Editing Task Code (continued) Insert code into the bar chart task code.
  1. In the Project window, double-click Bar Chart to reopen the bar chart task.

  2. Select the Preview task code check box.

  3. In the Code Preview window, click Insert Code. The Add User Code window opens.

  4. In the Statement box, select Graph Options. Then click the In statement tab. Type CLM=95.


Click to view the full window.


All statements that can appear in the code that is generated by the task are listed in the Statement list, including statements that correspond to options that you haven't selected in the task window. You can add code to any of these statements. However, your code will be included in the code that is generated for the task only if you select the option for that statement in the task window.

  1. Click OK.

  2. View the code and verify that the option was added in the correct location. No validation is performed on the code, so you need to be sure the code that you added is accurate.




  1. Click Finish. View the results and notice that the confidence limits were added to the mean.
