Submit a SAS program and view the messages in the Log window. Then edit and resubmit the program.

  1. Open a Code Editor window and name it ProdSales. Copy, paste, and submit the following code:

       data sasuser.sofasales;
      set sashelp.prdsale;
      if product="SOFA"; 
   run;
   proc print data=sasuser.sofasales;
   run;
    • Select InsertCode. In the Name box, type ProdSales. Click OK.
    • Copy and paste the code into the Code Editor window.
    • Press F3 or select CodeRun on <server-name>.
  2. View the messages in the Log window. How many observations does the Sofasales data set have?

    • In the Project window, double-click the Log item for the ProdSales code.
    • The Sofasales data set has 216 observations.

  4. Run the code and do not overwrite the previous results. How many observations does the Desksales data set have?
