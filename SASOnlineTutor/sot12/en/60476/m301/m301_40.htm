Generating and Editing Task Code (continued) Add custom code to all programs that run in SAS Enterprise Guide.
  1. Select ToolsOptions.


  2. Click the SAS Programs tab.

  3. Select the Insert custom SAS code before submitted code check box, then click Edit.


Click to view the full window.


  1. In the Edit window, type the following:

    options obs=10;

  2. Click OK.


Click to view the full window.


  1. In the Options window, select the Insert custom SAS code after submitted code check box and click Edit.

  2. In the Edit window, type the following:

    options obs=max;

  3. Click OK.


Click to view the full window.


  1. Click OK to close the Options window.

Next, you run a program with these new options set. Then you clear the options for including code.
