Automatically Adding Code to All Tasks and Programs

In addition to adding code to a particular task, you can also add custom code to the beginning or end of all task or program code that SAS Enterprise Guide runs. This is particularly useful for setting global options, library names, and so on.





To add code before and after any task that SAS Enterprise Guide runs, you enter code by using the Tasks tab in the Options window.

To add code before and after any program that SAS Enterprise Guide runs, you enter code by using the SAS Programs tab in the Options window.
