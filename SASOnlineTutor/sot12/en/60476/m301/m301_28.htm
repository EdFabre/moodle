Working with Data (continued) Add a Microsoft Excel file from the sample data directory.
  1. Select InsertData.

  2. In the Insert Data window, click the Existing tab.

  3. Navigate to the SAS Enterprise Guide sample data directory.

  4. Click the Files of type drop-down list and select All Files (*.*).

  5. Double-click as_products.xls.


Click to view the full window.


  1. Notice that there are two files to choose from. The file name ending with the dollar sign ($) represents an entire sheet in Microsoft Excel. The other file is a subset of cells that was defined in Excel. Double-click as_products($).


Click to view the full window.

  1. Select FileOpenFrom My Computer.

  2. Navigate to the SAS Enterprise Guide sample data directory. Make sure the Files of type drop-down list is set to All Files (*.*).

  3. Double-click as_products.xls.


Click to view the full window.


  1. Notice that there are two files to choose from. The file name ending with the dollar sign ($) represents an entire sheet in Microsoft Excel. The other file is a subset of cells that was defined in Excel. Select as_products($) and click Open. Select the option to open the file as is.
