Editing a Copy of the Task Code

When you edit task code, SAS Enterprise Guide copies the code to a new item in the project and the code is completely separate from the task. Any changes that you make to the bar chart by using the point-and-click task are not applied to this code.





You can do either of the following to edit a copy of the task code:

  • Right-click the task in the Project window and select Add as Code Template.

  • Open the task code by double-clicking the Code item under the task in the Project window. Press the space bar or another key anywhere in the Code window. You will be prompted whether to save the code as an editable copy. Click Yes.
