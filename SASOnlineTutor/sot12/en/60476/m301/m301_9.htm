Writing and Submitting SAS Code

Now that you are familiar with the windows that you will use in the SAS Enterprise Guide workspace, let's start using them to write and submit SAS programs.

When you submit a SAS program, SAS Enterprise Guide sends it to a SAS session on the active server. As SAS runs the program, messages about the status appear in the Task Status window. After the code runs, the log and any data sets and results automatically open and are added as items in the Project window.



You can submit code by using any of these methods:

  • Select CodeRun on <server-name>.
  • Click the Run tool ().
  • Right-click the code item in the Project window and select Run on <server-name>.
  • Press F3.


Submitting a Code Selection

To submit only a selection of code, you highlight the code that you want to submit in the Code Editor, then right-click and select CodeRun Selection on <server-name>. The selected code runs, and an item for the selected code is added to the project.


Changing Servers

By default, code that you submit runs on the active server. If you want to run your code on a different server, you can specify where you want the code to run by selecting CodeSelect ServerChange Server. The Select Server window opens, listing the registered servers. When you change servers in this way, the setting applies only to the code that is active in the project when you made the selection.
