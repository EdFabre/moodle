Writing and Submitting SAS Code (continued)

Resubmitting Code

When you resubmit code, SAS Enterprise Guide prompts you to choose whether to overwrite the previous results.



 

  • If you click Yes, you overwrite the results, and the original code and log are overwritten by the new code and log.

  • If you click No, you do not overwrite the results, a new code item (with the default name Coden) is added to the project, and the original code is unchanged.

The results of the Sales program that you submitted listed only rows in which actual sales were greater than $500.00.
