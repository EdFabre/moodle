Storing and Accessing Code (continued) Save the Sales code to an external file and create a shortcut in the Project window.
  1. In the Project window, right-click Sales and select Properties.


  2. In the Properties window, notice that the file location is (embedded in project). Click Save As.


Click to view the full window.


  1. In the Insert Code window, save the code to the default location (or select another location). In the File name box, type Sales.

  2. Click OK.

  3. In the Properties window, notice that the file location now has the path to the file that you just saved. Click OK.


Click to view the full window.


  1. In the Project window, notice that the icon for the Sales code item has changed to indicate that it is a shortcut to the saved file. Any changes you make to this code will be made directly to the file saved in the location that you specified.

  1. In the Project window, right-click Sales and select Properties.


  2. In the Properties window, notice that the file location is (Embedded In Project). Click Save As.


Click to view the full window.


  1. In the Save To window, select Local Computer. Save the code to the default location (or select another location). In the File name box, type Sales.

  2. Click Save.

  3. In the Properties window, notice that the File path box displays the path to the file that you just saved. Click OK.


Click to view the full window.


  1. In the Project window, notice that the icon for the Sales code has changed to indicate that it is a shortcut to the saved file. Any changes you make to this code will be made directly to the file saved in the location that you specified.

    You can also save the code and create a shortcut by right-clicking the code item and selecting Save <filename> As from the pop-up menu.
