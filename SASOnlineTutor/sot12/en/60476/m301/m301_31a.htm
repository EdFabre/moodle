Running Tasks in SAS Enterprise Guide

  1. The tabs at the top of each task window organize the options you can select for a task. All tasks have a Columns tab where you assign variables to roles.

  2. The Variables to assign box lists the columns, or variables, in the data set. An icon next to each variable represents the variable's type. STYLE is a character variable.The other variables are numeric variables.

  3. The task roles box lists the ways that variables can be used in the task. To run the List Data task, one variable is required in the List variables role. An icon next to each task role indicates the type of variables that you can assign to the role.

  4. You assign variables to roles by dragging them from the variables list and dropping them on the appropriate role in the task roles list. Variables can be assigned to more than one role.

  5. Every task has options that you can set for that task. Some tasks have many options, which are grouped into categories on tabs. The List Data task has an Options tab and a Titles tab. Many commonly used options are set by default.

  6. As you assign variables to roles and select options in the task window, SAS Enterprise Guide generates SAS code. You can see this code by selecting the Preview code check box.

  7. When you have finished selecting options for the task, you click Finish to submit the code and run the task. Although the task window closes when you run the task, your selections are saved and you can reopen the task from the Project window.
