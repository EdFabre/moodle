Opening a Stored SAS Program (continued)

Using Windows Explorer

You can drag and drop a SAS program from Windows Explorer to SAS Enterprise Guide. The program opens in the Code Editor window and an item is added to the Project window.

To open a file that is stored in a folder in your operating environment:

  1. Position the Windows Explorer window so that you can see both it and the Project window in SAS Enterprise Guide.

  2. Drag the file from the Explorer window to the Project window. As you drag the file, the cursor changes to indicate where you can drop the file.

    When you drag a file from Explorer, SAS Enterprise Guide makes a copy of the file and the original file is not affected by any changes you make.

You can also right-click the file in the Explorer window and select Open with Enterprise Guide from the shortcut menu.
