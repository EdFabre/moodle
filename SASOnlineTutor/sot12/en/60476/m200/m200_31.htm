Opening a Stored SAS Program
A SAS program that is stored in an external file can be opened in the Code Editor window using the Insert Code window, the Files folder in the Server List, or the Windows Explorer.

Editing SAS Programs
SAS programs consist of SAS statements. Although you can write SAS statements in almost any format, a consistent layout enhances readability and enables you to understand the program's purpose.

The Code Editor window enables you to enter and view text and

  • select one or more lines of text
  • use color-coding to identify code elements
  • automatically indent the next line when you press the Enter key
  • collapse and expand sections of SAS procedures, DATA steps, and macros
  • bookmark lines of code for easy access to different sections of your program.

Using the Code Editor window, you can also find and replace text, use abbreviations, and set options.

To save your SAS program to an external file, first activate the Code Editor window and select File Save As filename/Export, then specify the name of your program.

Interpreting Error Messages
When a SAS program that contains errors is submitted, error messages appear in the log. SAS can detect several types of errors, including syntax and data errors. This lesson focuses on identifying and resolving common syntax errors.

Correcting Errors
To modify programs that contain errors, you can correct the errors in the code editing window and then resubmit the revised program.

Detecting and Resolving Common Problems
You might need to resolve several types of common problems: missing RUN statements, missing semicolons, invalid options, and unbalanced quotation marks.

The Code Editor window has features that allow you to detect problems, such as unbalanced quotation marks, in your SAS programs before you submit them to SAS for processing.

Points to Remember
  • It's a good idea to begin DATA steps, PROC steps, and RUN statements on the left and to indent statements within a step.

  • End each step with a RUN statement.

  • Review the messages in the Log window each time you submit a SAS program.
