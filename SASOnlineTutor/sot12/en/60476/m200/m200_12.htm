Editing SAS Programs (continued)

Setting Code Editor Options

You can customize the Code Editor window by setting options. To open the Options window, select ToolsOptions and then click the Editor tab.

You can set various options here. For example, Show line numbers specifies whether to display line numbers in the margin. When line numbers are displayed, the current line number is red. Insert spaces for tabs specifies whether to insert the space character or the tab character when you press the Tab key. If this option is selected, the space character is used. If this option is not selected, the tab character is used.

When you are finished, click OK.







For more information about setting options in the Code Editor window, press F1 from the Code Editor window.

Setting Code Editor Options

You can customize the Code Editor window by setting options. To open the Options window, select ToolsOptions and then click SAS Programs in the selection pane. Click the Editor Options button.

You can set various options here. For example, Show line numbers specifies whether to display line numbers in the margin. When line numbers are displayed, the current line number is red. Insert spaces for tabs specifies whether to insert the space character or the tab character when you press the Tab key. If this option is selected, the space character is used. If this option is not selected, the tab character is used.

When you are finished, click OK to close the Enhanced Editor Options window. Click OK to close the Options window.





For more information about setting options in the Code Editor window, press F1 from the Code Editor window.
