Saving SAS Programs

To save your SAS program to an external file, first activate the code editing window and select File Save filename As/Export, then select a location and specify the name of your program in the Save As window.

Unless you name your code, SAS Enterprise Guide names it by appending a number to the word "code." It's good practice to assign the extension .SAS to your SAS programs to distinguish them from other types of SAS files. When you use the Save As window, the .SAS is added for you.
