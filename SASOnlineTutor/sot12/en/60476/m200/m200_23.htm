Submit a SAS program and resolve any errors.

  1. Copy and paste the program below into a code editing window: 
    data work.balnew2;
   set sasuser.insure;
run;
proc print data=work.balnew2
   var id name total balancedue;

  2. Submit the program and check for errors. In the log, notice that although you have not submitted a final RUN statement in this program, there is a RUN statement provided by SAS Enterprise Guide.

  3. Resolve any errors.

  4. Resubmit the program.

  5. Scroll through the output and verify that the revised program ran correctly.

    proc print data=work.balnew2;
  4. Select Code Run on Local and overwrite the previous results.
