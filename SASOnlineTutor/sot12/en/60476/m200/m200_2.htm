Opening a Stored SAS Program

So far, you've copied and pasted SAS programs into the Code Editor window, and submitted them for execution. You can also open a stored program in the Code Editor window, where you can edit the program and submit it again.

You can open a stored SAS program using any of the following methods:

  • InsertCode and locate the program
  • open the Files folder in the Server List
  • drag and drop from Windows Explorer.
