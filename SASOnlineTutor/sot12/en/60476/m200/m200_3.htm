Opening a Stored SAS Program (continued)

Using Menus

To use menus:

  1. Select InsertCode or click the Insert Code tool () on the toolbar.

  2. In the Insert Code window, click the Existing tab.

  3. Using the Look in drop-down list, select the directory where the SAS program is stored. Then double-click the file that you want to open. The program opens in a code editing window and an item for the code is added to the project.

    You can also use the Places bar on the left side of the window to select the directory or server location.




The file opens in a new code editing window, with the file name shown as the window title.
