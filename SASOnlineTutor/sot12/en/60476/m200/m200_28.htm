Are you ready for a challenge? Practice resolving several common syntax errors.

  1. Copy and paste the program below into a code editing window. Do not submit the program. 
    data work.reliable;
   set sasuer.insure;
where company='FRED SMYTHE;
run;
proc print data=work.reliable
   var id name company total balancedue;
run;
  2. In the code editing window, look at the program and notice that you have an unbalanced quotation mark. Find the error and correct it.

  3. Submit the program and check for errors.

  4. Resolve any errors and submit the revised program as needed.

  5. Scroll through the output and verify that the revised program ran correctly.
