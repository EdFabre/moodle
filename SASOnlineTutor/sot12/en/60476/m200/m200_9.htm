Editing SAS Programs (continued)

Entering and Viewing Text

The Code Editor window uses visual aides such as color coding and code sections to help you write and debug your SAS programs.

You can use the margin on the left side of the Code Editor window to

  • select one or more lines of text
  • expand and collapse code sections
  • display bookmarks.

Select buttons below to learn about some of the features of the Code Editor window. You can also see a text-only version of the information.
