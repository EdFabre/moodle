Opening a Stored SAS Program (continued)

Using the Files Folder in the Server List Window

You can locate and open stored SAS programs by using the Files folder in the Server List window. The Files folder enables you to browse through the file structure in your operating environment.

To include a file that is stored in a folder in your operating environment:

  1. In the Server List, double-click the Files folder.

  2. Browse to the location where you stored SAS programs by clicking the expansion icon next to the folders you want to open. Double-click the file, or select Open from the shortcut menu for the file.

    The first time you click Local to open the local server, it might take a few minutes if SAS Enterprise Guide has to establish a connection with SAS. When the connection is established, the icon for the server changes color.



The file opens in a new code editing window, with the file name shown as the window title.
