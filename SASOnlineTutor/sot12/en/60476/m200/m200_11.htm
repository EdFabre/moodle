Edit and submit two SAS programs.

You must complete this practice in order to do the next practice in this lesson.

  1. Copy the following code and paste it into a new code editing window that you name Tryit-11: 
         data sasuser.admit2;
     set sasuser.admit;run;
    Indent the statements as necessary to enhance the program's readability. Then submit the program.

  2. Copy the following code and paste it into the same code editing window: 
         proc print data=sasuser.admit2;
     run;
  3. Collapse the DATA step.

  4. Add the following statement between the PROC PRINT statement and the RUN statement:
         var id name actlevel fee;
  5. Bookmark the VAR statement.

  6. Submit the program again and overwrite the previous results.

 run;

  2. Copy the PROC PRINT step and paste it into the same code editing window.

  4. Add the new statement between the PROC PRINT statement and the RUN statement: 
     proc print data=sasuser.admit2;
    
    var id name actlevel fee;
    
 run;
  5. Position the cursor in the VAR statement and press Ctrl+F2.
