Edit and resubmit a SAS program. Then, view the results.

  1. Copy and paste the following PROC FREQ step into a new code editing window: 
    proc frq data=sasuser.heart order=freq;
       table sex*survive*shock / 
         nopercent list;
    run;
  2. Submit the program.

  3. Check for errors.

  4. Correct any errors.

  5. Resubmit the program and overwrite the previous results.

  6. View the output created by the program.

  7. Verify that the program ran correctly.
  1. Open a new code editing window and then copy and paste the program.

  2. Select Code Run on Local.

  3. As you can see, the code has errors. Open the log. Notice that there is an error message stating that the FRQ procedure cannot be found.

  4. In the first line, change FRQ to FREQ.

  5. Select Code Run on Local and overwrite the previous results.

  6. View the output.

  7. View the log. The program ran without errors.
