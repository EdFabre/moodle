Create a running total of a variable's values using the raw data file to which the fileref Saledata has been assigned.

  1. Copy the program below and paste it into the code editing window. 
    data sasuser.sales;
   infile saledata;
   input LastName $ 1-7 Month $ 9-11
         Residential 13-21 
         Commercial 23-31;
run;
proc print data=sasuser.sales;
run;
  2. Create the variable Total as the sum of Residential and Commercial. Then create the variable CumTotal so that it creates a running total of the values for Total. Initialize CumTotal to 1254657.

  3. Submit the DATA step and view the output. The last value of CumTotal should be 4827169.56.
