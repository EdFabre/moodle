Subsetting Data (continued)
Suppose you are reading data for the variables Age, Height, Weight, ActLevel, and Sex. You are also creating the variable BodyMass. However, you don't need all these variables in your data set. Specify a data set option so that only the variables Age, Sex, and BodyMass will be used. (Remember to end the statement with a semicolon.)
  

     data test.results 
        infile cltrials;
        input Age 1-2 Height 4-6 Weight 8-10
              ActLevel $ 12 Sex $ 14;
        BodyMass=weight/height**2;    
     run;
