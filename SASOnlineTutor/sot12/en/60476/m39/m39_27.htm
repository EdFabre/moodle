Format and label variables using the raw data file to which the fileref Saledata has been assigned.

  1. Copy the program below and paste it into the code editing window. 
    data sasuser.sales;
   infile saledata;
   input LastName $ 1-7 Month $ 9-11
         Residential 13-21 
         Commercial 23-31;
run;
proc print data=sasuser.sales;
run;
  2. Format the variables Residential and Commercial using the DOLLAR12.2 format. Label Month as Month of 1999. Edit the PROC PRINT step to display labels.

  3. Submit the DATA step and view the output. Is the PROC PRINT report correctly formatted and labeled?
