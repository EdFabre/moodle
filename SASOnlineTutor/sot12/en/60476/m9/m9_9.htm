Setting ODS HTML Options

As the next step, before creating linked graphs and HTML pages, you must specify HTML as your output destination (or format).

If you completed the lesson , you learned that the ODS HTML statement writes program output to HTML files. To conserve system resources, you should close ODS listing output, and then open ODS HTML output. 
     ods listing close;
     ods html;


The ODS HTML Statement for Drill-Down Graphs

For drill-down graphs, add two options to the ODS HTML statement:

  1. First, use the PATH= option to specify a location for storing GIF files and HTML files. The directory that you specify must already exist on your system; ODS will not create it for you. 
     ods html path='c:\data\sales\reports';
You can add the GPATH= option to specify a separate location for graphics output. For details, see the SAS/GRAPH documentation.
  1. Next, use the BODY= option to name the HTML file that will hold the drill-down chart. This file will be created in the location that you specified in the PATH= option. 
     ods html path='c:\data\sales\reports'
         body='totsales.html';

Example

In this example, both SAS/GRAPH GIF files and HTML output will be stored in C:\Data\Sales\Report in the Windows operating environment. The file totsales.html will display your program's SAS/GRAPH output when that output is viewed using a Web browser. 

     data work.saletrnd;
        set finance.prdsal2;
        format actual dollar10.0;
        length PriDrill $ 40;
        if year=1995 then pridrill='HREF="sales95.html"';
        if year=1996 then pridrill='HREF="sales96.html"';
        if year=1997 then pridrill='HREF="sales97.html"';
        if year=1998 then pridrill='HREF="sales98.html"';
     run;
     ods listing close;
     ods html path='c:\data\sales\reports'
         body='totsales.html';
     run;
