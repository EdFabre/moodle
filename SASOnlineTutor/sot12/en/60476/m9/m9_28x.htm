1.  

 To associate bars on a chart with target output, you start by

a. creating a variable to hold the HREF address for the primary chart.
b. creating a variable to hold the HREF addresses of the drill-down targets.
c. defining link values.
d. assigning links within the primary chart.


To associate chart elements with other output, you begin by creating a variable to hold the addresses of the drill-down targets. 

2.  

 HREF addresses are long text strings, so which statement do you use for creating a variable to hold them?

a. CHAR statement
b. HREF statement
c. LENGTH statement
d. VAR statement


To ensure that HREF values will fit within a variable, use the LENGTH statement to create the variable, and specify a length that will hold your largest address string, as in this example:
length varname $ 40;
 


3.  

 Which of the following statements correctly assigns an HREF address to the variable PriDrill?

a. if year=1998 then pridrill=sales98.html;
b. if year=1998 then pridrill="sales98.html";
c. if year=1998 then pridrill='HREF=sales98.html';
d. if year=1998 then pridrill='HREF="sales98.html"';


Text strings must be enclosed in quotation marks. Begin the string with HREF= to indicate a link address, then enclose the address itself in quotation marks. Be sure to nest quotation marks correctly, using single quotation marks for the outside set and double quotation marks for the inside. 


4.  

 HREF addresses can lead to anchors within HTML pages. Which of the addresses shown below correctly specifies an anchor?

a. sales98.html#office
b. sales98.html%office
c. sales98.html&office
d. sales98.html?office


To specify an anchor, add a pound sign (#) and the anchor name to the HREF address string. 


5.  

 Which pair of statements closes listing output and opens ODS HTML output?

a. 
listing close;
ods html output specifications... ;
b. 
listing close;
ods html open;
c. 
ods listing close;
ods html output specifications... ;
d. 
ods listing close;
ods html open;


You close listing output with the statement:
ods listing close;

You open HTML output with an ODS HTML statement:
ods html output specifications... ;
 

6.  

 How do you specify a location for storing GIF files and HTML files?

a. ODS HTML DIR='output location';
b. ODS HTML GIF='output location';
c. ODS HTML OUT='output location';
d. ODS HTML PATH='output location';


You use the PATH= option in an ODS HTML statement to specify a location for GIF files and HTML output. 


7.  

 In order to create a drill-down graph, the GOPTIONS statement must specify GIF output. The same statement can set the size of the SAS/GRAPH chart. Which of the following statements will perform both these tasks?

a. goptions device=gif x=480 y=360;
b. goptions device=gif xpixels=480 ypixels=360;
c. goptions output=gif xdim=480 ydim=360;
d. goptions output=gif xsize=480 ysize=360;


GIF output and chart size are specified with the DEVICE=, XPIXELS=, and YPIXELS= options in the GOPTIONS statement. 


8.  

 What function does the HTML= option in the following PROC GCHART statement perform? 
proc gchart data=work.saletrnd;
   vbar year / subgroup=prodtype discrete
        sumvar=actual html=pridrill;

a. It identifies to the GCHART procedure the variable that holds the HREF address strings.
b. It assigns a named anchor to the current location in the HTML output.
c. It directs chart output to the specified HTML file.
d. It identifies the primary HTML page.


The HTML= option in the GCHART procedure identifies the variable that contains HREF addresses. 


9.  

 What function does the ANCHOR= option in the following ODS HTML statement perform?

ods html anchor='salesq1';

a. It assigns elements of the chart to a variable that contains HREF address strings.
b. It assigns a named anchor to the current location in the HTML output.
c. It directs chart output to the specified HTML file.

The ANCHOR= option in the ODS HTML statement assigns a named anchor to the current location in HTML output.
