Using the Sasuser.Admit data set, create a drill-down chart and linked output.

  1. Copy the program below and paste it into the code editing window.
    data work.activity;
   set sasuser.admit;
   length DrillDown $ 40;
   if sex='F' then 
      drilldown='HREF="actfem.html"';
   if sex='M' then 
      drilldown='HREF="actmale.html"';
run;

ods listing close;
ods html path='?' body='?';
goptions device=gif xpixels=440 
         ypixels=330;
title 'Participant Activity Levels';
proc gchart data=work.activity;
   hbar sex / subgroup=actlevel 
        raxis=axis1 nostats html=?;
   axis1 minor=none 
         label=('Participants');
run;

ods html ?;
title 'Age and Activity Levels for Females';
proc  ?;
   where sex='F';
run;

ods html ?;
title 'Age and Activity Levels for Males';
proc ?;
   where sex='M';
run;

quit;
ods html close;
ods listing;
    If you are using SAS in the z/OS operating environment, you can view a sample program for platform-specific reminders.

  2. Replace the question marks (?) with code to

    • provide a valid path for GIF files and HTML output. For this option, add a suboption to specify that no path information is used in the URLs for linked files.

    • name the primary HTML page active.html.

    • link chart elements to HREF values in the variable Drilldown.

    • specify two HTML pages for output, one where Sex=F (actfem.html) and one where Sex=M (actmale.html).

    • specify the URLs actfem.html and actmale.html for these files.

    • complete two PROC steps to generate output that is subset or categorized by values of Sex (one where Sex=F and one where Sex=M). Use a SAS procedure that you are familiar with, such MEANS, FREQ, TABULATE, UNIVARIATE, REPORT, or PRINT.

  3. Submit the program and view active.html in your browser.

  4. Click the bars of the chart to link to output where Sex=F or Sex=M.
