Set system options.

  1. If you have opened the SAS System Options window, close it. Then write and submit an OPTIONS statement to
    • suppress the printing of the date and page numbers on output
    • specify 1950 as the beginning of the 100-year span that is used to interpret two-digit year values.

  2. Open the SAS System Options window and view the option settings that you revised.

  3. Important! In the SAS System Options window, change the value of the YEARCUTOFF= option back to the default so that the rest of your SAS session will not be affected.
