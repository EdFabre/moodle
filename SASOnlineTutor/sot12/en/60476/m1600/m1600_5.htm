Assign a libref and check librefs in effect.

  1. Open the Server List and view the libraries that are defined for the current SAS session.

  2. Write and submit a LIBNAME statement to assign the libref Mylib to a location in your operating environment.

  3. View the messages in the log. Was the libref Mylib assigned successfully?

  4. In the Server List, view the librefs that are now in effect, including Mylib.
