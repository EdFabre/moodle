Set and view system options.

  1. Open a code editing window. Write and submit an OPTIONS statement to specify 1950 as the beginning of the 100-year span that is used to interpret two-digit year values.

  2. Write and submit a procedure step that lists the value of the option you set.

  3. View the value of the option.

  4. Important! Write and submit an OPTIONS statement to the change the value of the option back to the default so that the rest of your SAS session will not be affected.
