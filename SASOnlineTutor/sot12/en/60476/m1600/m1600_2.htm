Defining SAS Libraries
SAS files are stored in libraries. When you define a library, you indicate the location of your SAS files to SAS. To define a library, you assign a library name and specify a path, such as a directory path. (In some operating environments you must create the directory or other storage location before defining the library.)

You also specify an engine, which is a set of internal instructions that SAS uses for writing to and reading from files in a library.
