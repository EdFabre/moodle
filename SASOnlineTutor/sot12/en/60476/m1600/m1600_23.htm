     run;

  1. Select Tools Options. Click the Results tab and select the Text output check box.

  2. Copy and paste the code into a code editing window.

  3. Submit the code. In the Project window, double-click Output to open the Text output for the procedure. Notice that the date and page number are printed at the top of the output.

  4. Write and submit the following OPTIONS statement: 
    options nodate nonumber;
  5. Resubmit the code. You can choose whether you want to overwrite the previous results. View the new output and notice that the date and page number are no longer printed at the top.
