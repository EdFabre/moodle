Setting System Options

Once you have created libraries, you might want to set certain SAS system options for your SAS session. There are numerous system options that you can set. Each option has a default setting that is used unless you specify otherwise. In this lesson you learn to set system options that control the appearance of text output and that control how SAS interprets date values.

There are times when you want to have precise control over the format of your output. This is particularly important if you are creating output for predefined forms. When you create your procedure output as Text output, you can control the appearance by setting system options such as

  • line size (the maximum width of the log and output)
  • page size (the number of lines per printed page of output)
  • the display of page numbers
  • the display of date and time.

These options do not affect the appearance of HTML output.
