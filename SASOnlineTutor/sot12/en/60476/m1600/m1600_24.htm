Viewing System Options

The OPTIONS Procedure

You can use the OPTIONS procedure to display the current setting of one or all SAS system options. The results are displayed in the log.


General form, OPTIONS procedure:
PROC OPTIONS <option(s)>;
RUN;

where option(s) specify how SAS system options are displayed.


Example

To list all SAS System options, their settings, and a description, submit the following code: 

     proc options;
     run;

The log lists the options and their settings.

Partial Log 

9          proc options;
10         run;
SAS (r) Proprietary Software Release 8.2 TS2M0
Portable Options:
ASYNCHIO         Enable asynchronous input/output
NOAUTOSIGNON     SAS/CONNECT remote submit will not automatically attempt to SIGNON
NOBATCH          Do not use the batch set of default values for SAS system options
BINDING=DEFAULT  Controls the binding edge for duplexed output
BOTTOMMARGIN=    Bottom margin for printed output
BUFNO=1          Number of buffers for each SAS data set


The log specifies the SAS software release number that corresponds to the version of SAS you are using.

To list the value of one particular option, use the OPTION= option on the PROC OPTIONS statement as shown below:

     proc options option=yearcutoff;
     run;
If a SAS system option uses an equals sign, such as YEARCUTOFF=, you do not include the equals sign when specifying the option to OPTION=.

The log shows that the setting of the YEARCUTOFF= option is 1920.

 


9          proc options option=yearcutoff;
10         run;
 SAS (r) Proprietary Software Release 8.2 TS2M0
 YEARCUTOFF=1920 Cutoff year for DATE and DATETIME informats and functions
