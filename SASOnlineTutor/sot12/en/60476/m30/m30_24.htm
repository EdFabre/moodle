Convert numeric data to character data using the data set Sasuser.Talent.

Be sure to complete this practice. You'll use the output data set in a later practice.

  1. Copy the program below and paste it into the code editing window. 
    data sasuser.njtalent;
   set sasuser.talent;
   NewPhone='(201)'||phone;
proc print data=sasuser.njtalent;
   var id phone newphone;
run;
  2. Submit the program and view the new variable NewPhone.
     
  3. Note the conversion message in the SAS log.
     
  4. Revise the DATA step to explicitly convert Phone to a character variable with a width of 7. Resubmit the program.
