Replace the contents of a character variable.

For this practice you use the data set Sasuser.Njtalent that was created in a previous practice. If you haven't created the data set, return to the practice and create it before proceeding.

  1. Create a data set named Sasuser.Detalent from Sasuser.Njtalent. In the new data set, replace a portion of the variable NewPhone. Change the area code from 201 to 302 in each value of the variable.

    Here is a sample value of NewPhone: 
         (201)5554200
  2. Print the new data set.
