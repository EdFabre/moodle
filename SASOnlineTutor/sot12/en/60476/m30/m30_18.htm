Converting Data with Functions (continued)
Which statement correctly describes the result of submitting the DATA step that is shown below? The variables TotalPay and Commission are numeric. The variable Pay is character. 
     data reg.newsales;
        set reg.sales;
        TotalPay=pay+commission;
     run;
The numeric variables TotalPay and Commission are converted to character variables because they are used in a character context.
The character variable Pay is converted to a numeric variable because it is used in a numeric context.
