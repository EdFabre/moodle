Convert character data to numeric data.

  1. Copy the program below and paste it into the code editing window. 
    data sasuser.talent2;
   set sasuser.talent;
   FtHeight=height/12;
proc print data=sasuser.talent2;
run;
  2. Submit the program and view the new variable FtHeight.

  3. Note the conversion message in the SAS log.

  4. Revise the DATA step to explicitly convert Height to a numeric variable. Use an informat with a width of 2.

  5. Resubmit the program and examine the output and log.
