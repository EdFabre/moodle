Create one observation for each repeating block of data in a single record, using the raw data file to which the fileref Excdata1 has been assigned.

  1. Create Sasuser.Actlevel from the raw data file that is referenced by the fileref Excdata1. Each block of values for ID and ActLevel forms one observation.

  2. Use the PRINT procedure to display the data set.

  3. Submit the step and view the output.
