Processing a DATA Step That Contains an Iterative DO Loop

Now that the program is complete, let's see how SAS processes a DATA step that contains an iterative DO loop. 

     data perm.sales97; 
        infile data97;
        input ID $ @;
        do Quarter=1 to 4;
           input Sales : comma. @;
           output;
        end;
     run;
