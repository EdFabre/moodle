Processing a DATA Step That Has a Varying Number of Repeating Fields

Here is the new version of the DATA step. 

     data perm.sales97;
        infile data97 missover;
        input ID $ Sales : comma. @;
        Quarter=0; 
        do while (sales ne .);
           quarter+1;
           output;
           input sales : comma. @;
        end;
     run;
