Submit a SAS program and view output in the Output window.

  1. If necessary, clear the code editing window. Then copy the program below and paste it into the window: 
    proc print data=sasuser.laguardia;
run;

    This program prints a listing of the Laguardia table, which is stored in the Sasuser library.

  2. Submit the program and scroll through your output in the Output window.

  3. Redisplay the code editing and Log windows.
