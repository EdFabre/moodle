Open the Report Editor from the Tools menu. The Report Editor is used for designing detail or summary reports that are based on the selected data.

  1. Select Tools Report Editor. The PROC REPORT: Data Selection window appears.

  2. In the Libraries list, select SASHELP. In the Datasets list, select PRDSALE.

  3. Click OK. The interactive REPORT editor window displays the default report for the Sashelp.Prdsale data set. (If a MESSAGES window appears, click OK.)

  4. Select View Next Page to scroll forward in the report.

  5. You can return to the REPORT editor window later to try out a report or two. For now, select File Close to close the REPORT editor window. In the Close window, click OK.
