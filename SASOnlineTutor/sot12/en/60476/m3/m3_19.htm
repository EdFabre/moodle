Create a file shortcut and use it to open a SAS program.

  1. In the top level of the Explorer window, open File Shortcuts. (If the Explorer window still displays two panes, select View Show Tree to return to a single-pane window.)

  2. Select File New. In the File Shortcut Assignment window,

    • Type Printnew as the file shortcut name.

    • Leave DISK as the method.

    • Select Enable at startup so that the file shortcut is created each time you start SAS.

    • For File, click Browse and select or type the mysales.sas program that you created in a previous practice. (Remember that in the CMS operating environment, this file is named mysales sas.) Then click Open or OK if shown.

    • Click OK to assign the file shortcut.

    File Shortcuts now contains your Printnew file shortcut.

  3. Open the Printnew file shortcut. The program opens in a new code editing window.

  4. Close the code editing window that contains mysales.sas.
