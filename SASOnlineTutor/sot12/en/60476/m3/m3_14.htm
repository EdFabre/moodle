View, save, and delete output in the Results window.

  1. If necessary, clear the code editing window. Then copy the program below and paste it into the window: 
    proc print data=sasuser.europe;
run;
  2. Submit the program to print a listing of the Sasuser.Europe table. Then scroll through your output in the Output window.

  3. In the Results window, click the expand icon beside the Results node to open the node if necessary. Then click the expand icon to open the folder for the second PRINT procedure output. (You created the first PRINT procedure output in the previous practice.)

  4. In the Results window, display the pop-up menu for Data Set SASUSER.EUROPE and select Open. The Output window scrolls to the top of the second PRINT procedure output.

  5. In the Results window, display the pop-up menu for Data Set SASUSER.EUROPE and select Save As. In the Save As window, select or enter a destination for your output. Then click OK or Save to save your output.

  6. In the Results window, display the pop-up menu for the top folder, which is labeled Print: The SAS System. Select Delete to delete the first PRINT procedure output from the Output and Results windows. When the Delete Confirmation box appears, click Yes to delete the output.

  7. Redisplay the code editing and Log windows.
