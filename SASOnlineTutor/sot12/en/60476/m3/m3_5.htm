In the Explorer window, view, sort, and display details about files. Then view file properties and open a file.

If a menu bar is not displayed for your SAS windows, please turn on menus before proceeding with SAS practices. Type pmenu on at the Command ===> prompt or in the ToolBox command window. Then press Enter. Menus are turned on in all SAS windows.

  1. In the Explorer window, open the Libraries folder. The active libraries are listed.

  2. Open the Sashelp library. All members in the Sashelp library are listed.

  3. If file details are not displayed, click the Toggle Details tool on the toolbar (if available).

  4. Click the Type column to sort the files by type. (You might need to widen the Explorer window in order to see the Type column.)

  5. Scroll to the Prdsale table. Display the pop-up menu and select Properties. The Properties window opens with General Properties displayed. After viewing the file's properties, click OK to close the Properties window.

  6. From the pop-up menu for the Prdsale table, select Open and view the data in the table.

  7. When you are finished looking at the data, close the window in which the table is displayed.

  8. Activate the Explorer window and move back to the top level by clicking the Up One Level tool on the toolbar twice or by selecting View Up One Level twice.
