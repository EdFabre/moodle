Reading Records That Have a Varying Number of Fields

Now, let's think about another type of variable-length record format, in which the record lengths vary because the records don't all contain the same number of fields.

The file that is referenced by the fileref Bpdata contains variable-length records in which an employee ID number is followed by repeating blocks of data that represent dates and blood pressure readings. Notice that each repeating block of data in the file has the same length. However, the records contain varying numbers of repeating blocks.
