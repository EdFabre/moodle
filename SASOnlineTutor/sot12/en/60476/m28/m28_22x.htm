1.  

 Variable-length records

a. have an end-of-record marker at the end of the last field.
b. do not have a predetermined number of columns.
c. use space more efficiently than fixed-length records.
d. all of the above


Variable-length records have an end-of-record marker at the end of the last field. Unlike fixed-length records, variable-length records do not have a predetermined number of columns. When records contain fields that have varying lengths or a varying number of fields, a variable-length record format uses space more efficiently than a fixed-length record format. 

2.  

 Which of the following is not true for the LENGTH= option?

a. It is used in the INFILE statement.
b. It determines the total length of each record.
c. It defines a variable that is stored as part of each observation.
d. It defines a numeric variable.


The LENGTH= option is used in the INFILE statement to create a numeric variable whose value is the length of the current record. A variable that is defined by the LENGTH= option is not stored as part of the observation.

7.  

 Variable-length records can contain

a. fields that have varying lengths.
b. varying numbers of fields.
c. both a and b
d. none of the above


Variable-length records can contain either fields that have varying lengths or a varying number of repeating blocks of data. In order to read both types of variable-length records, you need to determine the total length of each record by using the LENGTH= option.
