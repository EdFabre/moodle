Reading Records That Have a Varying Number of Fields (continued)

When the execution phase is complete, you can use the PRINT procedure to display the data set. The FORMAT statement associates the DATE9. format with the variable Date.

 

     proc print data=perm.health;
        format date date9.;
     run;

     proc print data=perm.health (drop=index);
        format date date9.;
