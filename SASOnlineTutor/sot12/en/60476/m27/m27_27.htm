Read a hierarchical file and create one observation per header record, using the raw data file to which the fileref Jan98dat has been assigned.

  1. Create Sasuser.Billing2 from the file that is referenced by the fileref Jan98dat. Header records are indicated by a P in the Type field. Detail records are indicated by a C in the Type field.

    Sasuser.Billing2 should contain only three variables: Name, ID, and TotalCost. (Hint: Use TotalCost as the sum of all values for Amount.)

  2. Use the PRINT procedure to display the data set.
