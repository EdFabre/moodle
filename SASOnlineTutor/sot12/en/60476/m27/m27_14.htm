Read a hierarchical file and create one observation per detail record, using the raw data file to which the fileref Jan98dat has been assigned.

  1. Create Sasuser.Billing1 from the file that is referenced by the fileref Jan98dat. Header records are indicated by a P in the Type field. Detail records are indicated by a C in the Type field.

    Sasuser.Billing1 should contain only four variables: ID, Name, Date, and Amount. Format the variables as needed.

  2. Use the PRINT procedure to display the data set.
