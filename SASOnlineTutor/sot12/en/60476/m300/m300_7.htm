Viewing SAS Libraries and Files

The Server List

You can use the Server List to view

  • SAS libraries
  • SAS data sets in your libraries
  • files in your operating environment.

The Server List does not open automatically the first time you start SAS Enterprise Guide. You can open it by clicking the Server List tool () on the toolbar or by selecting View Server List. By default, the Server List is docked on the right side of the workspace. Once you have opened this window, it will open automatically the next time you start SAS Enterprise Guide.





SAS Enterprise Guide can access SAS running on different servers, and the servers that you have access to are determined by what is set up in the Administrator. In some installations there could be several servers listed in this window. With the SAS Learning Edition, SAS is installed on the same machine as SAS Enterprise Guide, and SAS is a local server.

The first time you click Local to open the local server, it might take a few minutes if SAS Enterprise Guide has to establish a connection with SAS.

Viewing SAS Libraries

The libraries that are currently defined for your SAS session are listed under Libraries in the Server List. By default, SAS defines several libraries when the SAS session begins: Sashelp, Sasuser, and Work. SAS Enterprise Guide can also define libraries when it starts. In the SAS Learning Edition, SAS Enterprise Guide defines a library named Local.

To view the contents of a library, double-click the library name (or right-click the library name, and select Open from the pop-up menu). A list of the SAS data sets contained in the library is displayed.
