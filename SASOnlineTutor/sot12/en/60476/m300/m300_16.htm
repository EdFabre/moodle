Specifying Result Formats

To choose the type of output you want to produce, select ToolsOptions and then click the Results tab. Select the check boxes next to the appropriate output formats.

When you submit code that produces output, each type of output appears as a separate item in the project window. HTML output is labeled HTML and Text output is labeled Output.


PDF and RTF are experimental result formats for SAS Enterprise Guide 1.3 and the SAS Learning Edition.







The Results tab of the Options window has other preferences and options that you can set. You will learn more about some of these options later in this lesson.

Specifying Result Formats

To choose the type of output you want to produce, select ToolsOptions and then click Results in the selection pane. Select the check boxes next to the appropriate output formats.

When you submit code that produces output, each type of output appears as a separate item in the project window. HTML output is labeled HTML, PDF output is labeled PDF, RTF output is labeled RTF, and Text output is labeled Listing.




The Results page in the Options window has other preferences and options that you can set. You will learn more about some of these options later in this lesson.
