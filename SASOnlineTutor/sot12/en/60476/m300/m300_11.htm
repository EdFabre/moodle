Writing and Submitting Code (continued)

The Code Editing Window

In SAS Enterprise Guide, you use the Code Editor window to enter, edit, and submit SAS programs. You can open a Code Editor window by

  • selecting InsertCode
  • clicking the Insert Code tool on the toolbar
  • double-clicking Create Code in the Task List.

The default name for the Code Editor window is Coden. If you specify a name in the Insert Code window or rename the code item in the Project window, the new name appears in the Code Editor window title.

Because the title of the window changes each time you insert new code, the general term code editing window is used in other lessons to refer to the Code Editor window.

The Code Editor window has many features specifically designed to make editing and debugging SAS programs easier. You can learn more about the editing features of this window in the lesson .




Text that you enter into the Code Editor window is automatically saved when you close the window. After you submit the program you can reopen the code you have submitted by double-clicking the item for the code in the Project window.

When you edit and resubmit code, SAS Enterprise Guide prompts you to choose whether you want to overwrite the previous results. If you

  • overwrite the results, the original code and the log are overwritten by the edited code and new log
  • do not overwrite the results, a new code item (with the default name Coden) is added to the project and the original code is unchanged.
For more information about the Code Editor window, select Help Enterprise Guide Help and open the topic Working with Code from the Contents tab. You can also place your cursor in a blank area of the Code Editor window and press F1.
