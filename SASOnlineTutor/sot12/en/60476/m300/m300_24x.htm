1.  

 Which window displays the currently assigned SAS libraries?

a. Task List window
b. Server List window
c. Project window
d. Options window


The Libraries folder in the Server List window displays the currently assigned SAS libraries. 

2.  

 Which Enterprise Guide windows do you use to perform these tasks (in the order given)?

  • enter, edit, and submit SAS programs
  • open the Log
  • view messages about your SAS session and about any SAS programs that you submit
  • select output formats to generate.

a. Code Editor, Options, Log, Results windows
b. Code Editor, Project, Log, Options windows
c. Code Editor, Project, Log, HTML Results windows
d. Log, Results, Project, Code Editor windows


You use the Code Editor window to write and submit SAS programs; the Project window to open items such as the Log; the Log window for viewing messages; and the Options window for setting options such as the format of output to generate.  


3.  

 Which statement about SAS Enterprise Guide and SAS is true?

a. SAS Enterprise Guide submits code to SAS, SAS processes the code and sends the results back to SAS Enterprise Guide.
b. Essential components of SAS software are included in SAS Enterprise Guide software and these components process SAS programs.
c. SAS must be installed on the same PC with SAS Enterprise Guide.
d. When you first start SAS Enterprise Guide, it is a good idea to start a SAS session by selecting RunStart SAS Session.


SAS Enterprise Guide is a client application that submits code to SAS for processing. SAS can be on the same machine or it can be on a remote server. You don't have to start a SAS session; SAS Enterprise Guide does that for you. 


4.  

 What does the Task Status window display?

a. messages about the tasks that are available to you in Enterprise Guide.
b. messages about any errors that occured when code was submitted to SAS
c. messages about the status of code that is currently running or pending in SAS
d. messages in this window are only displayed when you use the point-and-click tasks in SAS Enterprise Guide.


The Task Status window lists the tasks that are currently running or pending in Enterprise Guide. 


5.  

 What is true about Projects in SAS Enterprise Guide?

a. You must save each item in the project individually.
b. You can rename and delete items in a Project.
c. Log messages for all the code you submit are grouped together under a Log item in the project.
d. Projects are saved with the file extension .proj.


You can perform file management tasks, such as renaming and deleting, on all the items in a project. When you save a project, all items in the project are automatically saved. Projects have the file extension .seg.  

6.  

 After you have submitted code and looked at the results, you realize it would be interesting to make a change to the code and compare the output of the two programs. You edit the code and submit it again. What happens?

a. You are prompted to select whether or not you want to overwrite the previous results. You select Yes so you can compare the output of the two programs.
b. By default, if you edit code you have previously submitted, a new code item is added to the Project. You compare the output of the two programs.
c. You are prompted to select whether or not you want to overwrite the previous results. You select No so that you can compare the output of the two programs.
d. Nothing. Once you have submitted code, you cannot use the same Code Editor window to submit code again.


By default, SAS Enterprise Guide prompts you to choose whether you want to overwrite the previous results when you resubmit code. If you want to compare the output of two programs, you must select No. If you select Yes, the previous code, results and log are replaced. 


7.  

 Which of the following is true about results generated in SAS Enterprise Guide?

a. You can create both Text output and HTML output.
b. You select the type of output to create in the Options window.
c. Text output and HTML output appear as separate items in the Project window.
d. all of the above


You can create text output, HTML output, or both types of output. The Project window displays separate icons for each type of output. 


8.  

 Which statement about styles is true?

a. Styles can be applied to HTML results that have already been generated.
b. To make a style the default, you select it in the Options window.
c. You can create a custom style.
d. all of the above


Styles are predefined or custom CSS that are applied to HTML results. You can apply the style after the results are generated. If you want to select a new default style, you specify the style in the Options window.  


9.  

 How can you view the properties for a SAS file?

a. Select ViewProperties from the main Enterprise Guide menu.
b. Print the file and view its properties in the Log window.
c. Right-click the file in the Project window and select Properties from the shortcut menu.
d. Right-click the file in the Server List and select Properties from the shortcut menu.


To view properties for a file, you must add it to the Project and then select Properties from the shortcut menu for the file in the Project window.  


10.  

 Which statement about the Log window is true?

a. The log contains messages about the processing of SAS programs.
b. Some of the messages in the log pertain to options that SAS Enterprise Guide adds to the code that you submit.
c. The log is generated each time you submit code.
d. all of the above


A log is generated each time you submit code to SAS. It contains messages about your SAS session and the code you submit. Because SAS Enterprise Guide submits some code with your code, some of the log messages refer to the code that SAS Enterprise Guide submits.
