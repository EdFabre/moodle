Submit a SAS program and work with the results.

  1. Specify that your results appear in both HTML and Text formats.

  2. Copy the program below and paste it into a code editing window named Tryit-16: 
    proc print data=sasuser.laguardia;
run;

    This program prints a listing of the Laguardia table, which is stored in the Sasuser library.

  3. Submit the program and scroll through the HTML results.

  4. Apply the SansPrinter style to the results.

  5. View the Text output.

  6. Save the HTML results as an HTML file in your operating environment.

  2. Open a new code editing window, name it, and copy and paste the program into the code editing window.

  3. Select CodeRun on Local. Your report displays 46 rows and 9 columns.

  4. From the toolbar, click the styles drop-down list and select the SansPrinter style. Notice that the new style is immediately applied to the HTML results.

  5. In the Project window, double-click the Output item under the code you submitted. Browse the output.
