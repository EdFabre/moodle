Managing Your Workspace

As you use SAS Enterprise Guide, you will use and customize the workspace in ways that make it easier and more efficient for you to complete your tasks. You can customize the window layout, set options, and customize toolbars. You can also manage items in the Project window to keep your work organized.

Managing Projects

  • If you submit code, edit the code, and then resubmit it, you have to decide whether you want to replace the previous results. If you need to save the previous code or compare the output of the two programs, you should not overwrite the previous results so that the original code remains unchanged. If there is no reason to save the original code, you can overwrite the previous results.

  • You can perform file management tasks such as renaming, deleting, copying, and pasting on any files in the Project window. To perform these tasks, right-click the file and select the appropriate item from the shortcut menu.

  • It is a good idea to save your project each time you exit SAS Enterprise Guide.

Customizing SAS Enterprise Guide

SAS Enterprise Guide saves the layout of your workspace so that it opens with the same window arrangement that you used when you exited the software. In addition, you can change elements of the workspace in the Customize window.

The Customize window enables you to control toolbars, the Tools menu, shortcut keys, and editor keys. To open the Customize window, select ToolsCustomize.

Setting Options

As you have learned, you set Results options in the Options window. In this window you can also set options for the workspace, data, editor, and code. You can learn more about setting Editor options in the lesson .
