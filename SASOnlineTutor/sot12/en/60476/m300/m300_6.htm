Create a new project, add a note, and save the project.

You must complete this practice in order to do the other practices in this lesson.

  1. With the Project window active, select FileNew.

  2. Create a new project and name it UseProgWkspace.

  3. Using the Task List, add a note to the project. Type a note into the Note window and then close it.

  4. Save the project as UseProgWksp.seg.

  1. Select FileNew.

  2. In the New window, click the New tab and the Project icon. In the Name box, type UseProgWkspace. Click OK.

  3. In the Task list, double-click Create Note. Type any note into the Note1 window and then close the window. The contents of the note are saved automatically.

  4. Select FileSave UseProgWkspace As. In the Save As window, select a location to save the project and type UseProgWksp in the File Name box. Click Save.
