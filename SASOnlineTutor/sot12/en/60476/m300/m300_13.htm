Writing and Submitting Code (continued)

Viewing Log Messages

The Log window displays messages about your SAS session and about any SAS programs that you submit. You can open the Log window by double-clicking the log item in the Project window.

Some of the messages in the log pertain to the code that SAS Enterprise Guide submits to set options and create HTML output. To find log messages that relate to the code you submit, scroll the Log window until you see the beginning of your code. These log messages are the ones that are important to you.

Example

In the previous practice, you submitted the following code: 

     data sasuser.mysalesdata;
        set sashelp.prdsale;
        if product="SOFA";
     run;

The messages in the log for this code are shown below. Notice that messages relating to this code begin on line number 9. (Your log may differ from this example).



The log is generated each time you run code.
