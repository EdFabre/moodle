Using the Main SAS Enterprise Guide Windows (continued)

The Task List

The Task List is primarily used to select analytical tasks to run on your data. The Tasks by Category tab lists tasks grouped into categories of similar types of tasks. The Tasks by Name tab lists tasks alphabetically and includes the SAS procedures associated with each task.

SAS Enterprise Guide includes dozens of point-and-click tasks that you can run without knowing how to program in SAS. However, because you are learning SAS programming, the only relevant task is Create Code. This task opens a new Code Editor window and adds a code item to the project. You can also select InsertCode to open a new Code Editor window.



You double-click a task in the Task List to begin that task. If you close the Task List, you can restore it by clicking the Task List tool () on the toolbar, or by selecting ViewTask List.

If you want to learn more about using the point-and-click tasks in SAS Enterprise Guide, select HelpGetting Started with Enterprise Guide.
If you want to learn more about using the point-and-click tasks in SAS Enterprise Guide, select HelpGetting Started Tutorial.
