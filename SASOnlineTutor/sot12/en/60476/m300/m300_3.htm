Using the Main SAS Enterprise Guide Windows

By default, SAS Enterprise Guide has the following windows open and docked in the workspace: Project, Task List, and Task Status. You use these windows to manage your work, select tasks to perform, and view the processing status of code that you submit to SAS. There are additional windows that you open as you work and other windows that open automatically when you perform tasks.

The Agent also opens by default when you start SAS Enterprise Guide. You can hide the Agent by right-clicking the agent character and selecting Hide from the shortcut menu.







All windows in SAS Enterprise Guide have familiar features that help you get your work done. For example, you can

  • maximize, minimize, restore, and reposition windows
  • use pull-down menus, pop-up menus, and toolbars
  • get more help.

You're probably familiar with most of these features, but you might want to check these tips before you continue.

Minimizing and Restoring Windows

You can click the Minimize button to send some windows, such as Results or Code windows to the bottom of the SAS Enterprise Guide workspace. To restore the window to its former position, click the corresponding Restore button .

Docking and Undocking Windows

In SAS Enterpise Guide, the windows are docked in the workspace by default. The bar at the top of the window allows you to move the window and dock it in another position within the workspace or move it out of the workspace. You can also use the arrow to make the window expand up or down.

Using Shortcut Menus

Shortcut menus are context sensitive; they list actions that pertain only to a particular screen region or selection. Generally, you display shortcut menus by clicking the right mouse button.

Getting Help

Help is available for all windows in SAS Enterprise Guide. You can also open the entire help contents by selecting HelpEnterprise Guide Help.

Customizing Your Workspace

You can customize many features of the SAS Enterprise Guide workspace such as toolbars, pop-up menus, icons, and so on. Select the Tools menu to explore some of the customization options that are available.
