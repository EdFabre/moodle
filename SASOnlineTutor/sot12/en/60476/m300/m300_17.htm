Specifying Styles for HTML Output

You can change the style of your HTML output by applying a predefined style or by creating and applying your own style. Styles are Cascading Style Sheets (CSS) that SAS Enterprise Guide applies to the HTML results.

You can apply a style after results are generated by opening the HTML results and selecting a style from the drop-down styles list in the menu bar.


The HTML results below have the SAS Web style applied.




To make a style the default style, you must select the style on the Results tab of the Options window.
