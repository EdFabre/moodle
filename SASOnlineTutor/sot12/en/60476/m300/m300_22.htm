Browse the help for SAS Enterprise Guide and SAS language elements.

  1. Open the code named Tryit-12.

  2. From the Code Editor window, get help for using the Code Editor window.

  3. On the Contents tab, browse the help topics to see the help that is available for Enterprise Guide. Close the Enterprise Guide Help window.

  4. From the Code Editor window, get help for the SET statement.
  1. In the Project window, double-click Tryit-12.

  2. In the code editing window, click in a blank area of the window and press F1. The Enterprise Guide Help window opens and displays the topic Working with Code.

  3. Browse the contents by expanding books and clicking on topic titles on the left side of the window, and viewing topic content on the right side of the window. When you are finished browsing topics, close the Enterprise Guide Help window.

  4. In the code editing window, click to place the cursor in the word SET, then press F1. Notice that this time the Online Help window for SAS opens to the topic Data Step Statements: Syntax. If you scroll the topic, you will find the SET statement syntax.

    If you know the SAS language element that you need help for, you can also click the Index tab in the Enterprise Guide Help window and type the name of the element into the box.
