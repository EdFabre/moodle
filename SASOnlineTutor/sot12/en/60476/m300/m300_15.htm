Viewing Results
Not all SAS programs create output. Some only produce messages in the log. If your program includes procedure code that produces output, the results open automatically in SAS Enterprise Guide. By default, all programs submitted using SAS Enterprise Guide have programming statements added so that HTML output is created.

You can select other types of output to produce by setting results options. You can select formats such as

  • an HTML document, which is the default type of output
  • Text, which is traditional SAS listing output
  • both of the above.
