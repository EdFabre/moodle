Viewing SAS Libraries and Files (continued)

Viewing a File's Contents

To display a SAS data set from the Server List, double-click the data set name (or right-click the data set and select Open from the pop-up menu). The data set is displayed in the data grid and an item is added to the open project.



To display a file's properties, you right-click the filename in the Project window, and then select Properties from the shortcut menu.
