Getting Help

You've learned to use SAS Enterprise Guide to perform common SAS programming tasks. As you work, be sure to take advantage of the different types of online help that are available to you.





  • Enterprise Guide Help is task-oriented help for all of SAS Enterprise Guide. You can also open this help by pressing F1.

  • Getting Started Tutorial is a step-by-step tutorial that introduces you to the main features of SAS Enterprise Guide. This tutorial focuses on using queries and tasks rather than SAS programming.

  • If you have internet access, SAS on the Web has links to SAS Web sites such as SAS Enterprise Guide Home and Frequently Asked Questions about SAS Enterprise Guide.

  • From the Code Editor window, you can access help for SAS language elements by placing your cursor on a keyword and pressing F1.
