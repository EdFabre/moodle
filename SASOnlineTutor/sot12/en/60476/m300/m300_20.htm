  1. Select ToolsOptions to open the Options window.

    • Click the General tab. In the Assistance area, clear the Show Agent check box. This keeps the Agent from opening each time you start SAS Enterprise Guide.

    • Click the Results tab. In the Preferences area, clear the Open output data automatically check box. With this new setting, output data is created and an item is added to the Project, but the data does not open automatically.

    • On the Results tab, select a style from the Styles drop-down list. Once you select a style here, it becomes the default style.

  2. Close the Task List and the Server List. If these windows are closed when you exit Enterprise Guide, they will be closed the next time you start the software.

  3. In the Project window, right-click Prdsale and select Delete from the shortcut menu. Then select FileSave UseProgWkspace.
