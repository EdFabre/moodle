View SAS libraries and the properties of a SAS data set.

You must complete this practice in order to do the other practices in this lesson.

  1. Open the Server List.

  2. Open the Sasuser library. All the SAS data sets in the library are listed.

  3. From the Sashelp library, add the SAS data set Prdsale to the project.

  4. View properties for Prdsale. Close the Prdsale table.
  1. Select ViewServer List. Click the expansion icon next to Local.

  2. Click the expansion icon next to Libraries. Then click the expansion icon next to Sasuser. Notice that the SAS data sets for this programming course are displayed in the Sasuser library. Click the expansion icon again to collapse the Sasuser library.

  3. Open the Sashelp library and scroll the list of data sets to the Prdsale table. Double-click Prdsale.

  4. In the Project window, right-click the Prdsale table and select Properties from the shortcut menu. When you are finished viewing properties, close the Properties window.
