Writing and Submitting Code

When you submit code in SAS Enterprise Guide, several things happen:

  • a message about the processing status appears in the Task Status window
  • after the code is processed, the log is generated and added as an item under the code in the Project window
  • results and data sets, if any, open in separate windows.

All of the items relating to that code are stored in the project under the code item.
