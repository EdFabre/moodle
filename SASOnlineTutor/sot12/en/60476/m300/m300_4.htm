Using the Main SAS Enterprise Guide Windows (continued)

The Project Window

SAS Enterprise Guide uses projects to manage a collection of related data, tasks, code, and results. Each time you start SAS Enterprise Guide, you must either create a new project or open an existing project. Only one project can be open at a time in SAS Enterprise Guide.

You insert code, data, or a note in a project by selecting the appropriate item from the Insert menu. After you submit the code or run a task on your data, the log and any associated results appear as items under the associated code in the project.





You can collapse and expand items in a project to hide or display the contents. You can also use shortcut menus to perform actions on items in the project (such as viewing properties, renaming, or deleting). You will learn more about managing projects later in this lesson.

If you close the Project window, you can reopen it clicking the Project tool () on the toolbar, or by selecting View Project. Note that closing the Project window does not close the project in SAS Enterprise Guide.

The Process Flow Window

In SAS Enterprise Guide, the work that you do is saved in projects. A project is a collection of related data, tasks, code, and results. Each time you start SAS Enterprise Guide, you must either create a new project or open an existing project. SAS Enterprise Guide uses the Process Flow window as the default window to display the items in a project and the relationship between the items. This display is also called the process flow diagram (PFD) for the project.

An empty Process Flow window opens when you create a new project. As you add data, run tasks, and generate output, your work is automatically displayed in the Process Flow window. Although only one project at a time can be open in SAS Enterprise Guide, you can have multiple process flow diagrams for one project.

The Project Window

The Project window also displays the items in a project in SAS Enterprise Guide. The Project window is a hierarchical view of the active project and its associated data, code, notes, and results. You can choose whether you want to use the Project view or Process Flow view of your project. The tabs at the bottom of the window enable you to select the view that you want to see. In this lesson you will use the Project window.





You can collapse and expand items in a project to hide or display the contents. You can also use shortcut menus to perform actions on items in the project (such as viewing properties, renaming, or deleting). You will learn more about managing projects later in this lesson.

If you close the Project window, you can reopen it clicking the Project tool () on the toolbar, or by selecting ViewProject. Note that closing the Project window does not close the project in SAS Enterprise Guide.
