Introduction

Now that you understand how SAS programs are structured and processed, it's time to learn how to use the programming workspace to write and submit SAS programs. In this lesson you learn how to use SAS Enterprise Guide to accomplish your SAS programming tasks.








This lesson contains pages and takes approximately 1 hour to complete.


In this lesson, you learn to

  • understand the relationship between SAS Enterprise Guide software and SAS software
  • use the main SAS Enterprise Guide windows and their features
  • view SAS libraries and files
  • enter and submit SAS programs
  • view log messages
  • view results
  • manage your workspace.
