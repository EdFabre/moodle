Use a PROC SQL inner join and summary functions to combine the tables Sasuser.Staffmaster and Sasuser.Payrollmaster.

To view the expected PROC SQL output, click the related link in the text for each step.

  1. Write a PROC SQL query that displays LastName, FirstName, JobCode, and DateofHire for employees who have more than 20 years of service as of January 1, 2001. Your query should meet the following requirements:

    • Use an inner join to combine the two tables.

    • Use the following table aliases: s for Sasuser.Staffmaster and p for Sasuser.Payrollmaster.

    • Order the output by LastName.

    • In the WHERE statement, use a function to calculate each employee’s years of service, rounded to an integer value. (Remember to divide by 365.25 when calculating the number of years.)

    • Display the title Employees with More than 20 Years of Service.

    Add a QUIT statement to the query. Submit the query and view the output.

  2. Modify the query to display the column Years (years of service) after DateOfHire. (Hint: Remember to use the CALCULATED keyword in the WHERE clause.) Submit this modified query and view the output.

  3. Modify the query as follows:

    • Remove the columns LastName, FirstName, DateOfHire, and Years from the output.

      Your query should still display data for employees with more than 20 years of service, so you will need to copy the Years calculation to the WHERE clause. (Because Years is no longer listed in the SELECT clause, you can not reference it by a column alias in the WHERE clause.)

    • After JobCode, use a summary function to create the new column Employees, which displays the number of employees.

    • Group and sort the data by JobCode.

    Submit this modified query and view the output.

  4. Submit a null TITLE statement to cancel titles.
