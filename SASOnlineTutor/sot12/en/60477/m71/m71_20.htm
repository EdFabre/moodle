Using Outer Joins (continued)

The PROC SQL query below uses a join to combine two tables. The two tables contain data about the computer equipment owned by a very small company:

  • Repairs lists all equipment repaired in the current year.
  • Inventory lists all computer equipment owned by the company.
proc sql;
   select *
      from repairs 
           right join
           inventory
      on repairs.equipid = 
         inventory.equipid;
