Processing a PROC SQL Join

  1. This animation shows how PROC SQL processes the following inner join: 
         proc sql;
        select *
           from one,two
           where one.x = two.x;
     The tables One and Two are shown on page 7, above the animation.

  2. To start, PROC SQL creates the Cartesian product of the two tables.

  3. PROC SQL then evaluates all rows in the Cartesian product and eliminates any rows that do not meet the join conditions that are specified in the WHERE clause.

  4. Finally, PROC SQL returns the one remaining row as output.
