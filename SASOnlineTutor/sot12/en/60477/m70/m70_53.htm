Subsetting Data by Using Correlated Subqueries (continued)

You are writing a PROC SQL query to create a list of flight assignments for level-3 pilots. The data you need is stored in the following tables:

  • Sasuser.Flightschedule contains the EmpID for all employees who have been assigned to flights, along with data about the flights. However, this table does not contain a JobCode column.

  • Sasuser.Payrollmaster contains the JobCode for all employees.

Your query should contain a correlated subquery. Results will be sorted by EmpID, Date, and FlightNumber.

Which of the following queries fits the stated requirements and will produce the output that you want?

 
proc sql;
title 'Flight Assignments for Level 3 Pilots';
   select empid, date, flightnumber, 
          destination
      from sasuser.flightschedule
      where empid =
         (select empid
            from sasuser.payrollmaster
            where jobcode='PT3')
      order by empid, date, flightnumber;
 
proc sql;
title 'Flight Assignments for Level 3 Pilots';
   select empid, date, flightnumber, 
          destination
      from sasuser.flightschedule
      where empid in
         (select empid
            from sasuser.payrollmaster
            where payrollmaster.empid=
                  flightschedule.empid)
          and jobcode='PT3'
      order by empid, date, flightnumber;
