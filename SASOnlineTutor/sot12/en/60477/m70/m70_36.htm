Summarizing and Grouping Data (continued)
You are creating a PROC SQL query using the table Sasuser.Staffmaster, which lists all employees of an airline. Add a SELECT clause that specifies three columns. The first two columns are the existing columns State and City. The third column is a new column called Employees, which displays the number of people employed in each city. Use the COUNT summary function to define the new column. (There are no missing values in the table.) 

proc sql;
title 'Number of Employees in Each City';
   
      from sasuser.staffmaster
      group by state, city;
