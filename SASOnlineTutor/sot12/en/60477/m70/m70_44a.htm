Processing a Multiple-Value Noncorrelated Subquery

  1. This animation shows how PROC SQL processes a noncorrelated subquery using the following program: 
         proc sql;
        select empid, lastname, firstname, city, state
           from sasuser.staffmaster
           where empid in
              (select empid
                 from sasuser.payrollmaster
                 where month(dateof birth)=2);
  2. PROC SQL evaluates the inner query and builds a virtual table that satisfies the WHERE criteria.

  3. The virtual table contains the values 1403, 1404, 1834, 1103, 1420, and 1390.

  4. PROC SQL passes the values 1403, 1404, 1834, 1103, 1420, and 1390 to the outer query.

  5. The PROC SQL output shows only rows where the value of EmpID in Sasuser.Staffmaster matches a value in the virtual table.
