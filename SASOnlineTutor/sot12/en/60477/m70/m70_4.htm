Displaying All Columns (continued)

The table Sasuser.Flightschedule contains four columns in the following order: FlightNumber, Date, Destination, and EmpID.

Which of the following PROC SQL queries will display all columns and rows as they appear in the table?

 
proc sql;
   select all
      from sasuser.flightschedule;
 
proc sql;
   select *
      from sasuser.flightschedule;
 
proc sql;
   select flightnumber, destination, date, empid
      from sasuser.flightschedule;
 
proc sql;
   select *4
      from sasuser.flightschedule;
