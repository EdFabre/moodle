Subsetting Data by Using Noncorrelated Subqueries (continued)
You are writing a PROC SQL query that will list all international flights that have a delay that is longer than the delay of every domestic flight. Which of the following queries will accurately complete the task? 
proc sql;
   select flightnumber, date, delay
      from sasuser.flightdelays
      where destinationtype='International' and delay >
         (select delay
            from sasuser.flightdelays
            where destinationtype='Domestic');
 
proc sql;
   select flightnumber, date, delay
      from sasuser.flightdelays
      where destinationtype='International' and delay > all
         (select delay
            from sasuser.flightdelays
            where destinationtype='Domestic');
 
proc sql;
   select flightnumber, date, delay
      from sasuser.flightdelays
      where destinationtype='International' and delay > any
         (select delay
            from sasuser.flightdelays
            where destinationtype='Domestic');
