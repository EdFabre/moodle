Subsetting Data by Using Noncorrelated Subqueries (continued)

The PROC SQL query shown below will list the contact information for all level-3 mechanics, using two tables:

  • Sasuser.Staffmaster contains contact information for all employees but does not contain each employee's job code.

  • Sasuser.Mechanicslevel3 contains the names of all level-3 mechanics but does not contain contact information.

Both tables identify employees by EmpID.

As the first line of the subquery, write a SELECT clause that returns the EmpID values of all level-3 mechanics to the outer query.

 

proc sql;
title 'Contact Information for Level 3 Mechanics';
   select empid, lastname, firstname, phonenumber
      from sasuser.staffmaster
      where empid in
         
            from sasuser.mechanicslevel3)
      order by lastname;
