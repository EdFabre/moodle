Subsetting Rows by Using Conditional Operators (continued)

The table Sasuser. Payrollmaster lists all employees of an airline. Which of the following PROC SQL queries will display rows of values only for male flight attendants?

Each employee's job title is indicated by a 2- or 3-character code at the beginning of the JobCode column. The last digit of each JobCode value is an integer that identifies the job level. FA is the code for flight attendant. 
proc sql;
   select empid, gender, jobcode, salary, 
          salary/3 as Tax
      from sasuser.payrollmaster
      where gender='M' and 
            jobcode between 1 and 3 = 'FA';
 
proc sql;
   select empid, gender, jobcode, salary, 
          salary/3 as Tax
      from sasuser.payrollmaster
      where gender='M' and 
            jobcode in ('FA');
 
proc sql;
   select empid, gender, jobcode, salary, 
          salary/3 as Tax
      from sasuser.payrollmaster
      where gender='M' and 
            jobcode contains 'FA';
 
proc sql;
   select empid, gender, jobcode, salary, 
          salary/3 as Tax
      from sasuser.payrollmaster
      where gender='M' and 
            jobcode contains ? 'FA';
