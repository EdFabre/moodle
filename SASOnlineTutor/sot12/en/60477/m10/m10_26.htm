Querying Multiple Tables (continued)
Here is the PROC SQL step that joins the tables Clinic.Admit and Clinic.Insure. Remember that these two tables both contain a column named ID. As the last clause in the SELECT statement, write a clause to select only rows that have matching values for the column ID. (End the clause with a semicolon.) 

proc sql;
   select admit.name,fee,total,balancedue,
          total-balancedue as AmountPaid
      from clinic.admit,clinic.insure
