Create multiple macro variables with the SQL procedure, using the Sasuser.Schedule data set.

  1. Use the SQL procedure to create a set of macro variables named begin1 through begin18, one for each course in the Sasuser.Schedule data set. The value of each begin macro variable should be the starting date of the corresponding class (that is, the value of the data set variable Begin_date) in MMDDYY10. format. Include an option that will prevent the results from being displayed as output.

  2. Submit a %PUT statement that will list the new macro variables and their values in the SAS log.

  3. Copy the following program and paste it into the code editing window. Modify the TITLE2 statement so that the series of Xs is replaced with the appropriate indirect macro variable reference, based on the current value of num, which represents the course number (1 through 18). Submit the modified program. 
    
%let num=4;
