Create macro variables with the SYMPUT routine, using the Sasuser.Courses data set.

  1. Submit an OPTIONS statement to reset the system option DATE | NODATE to NODATE, and to activate the SYMBOLGEN option.

  2. Write a DATA step that creates a macro variable named date. This macro variable's value should be today's date in the MMDDYY10. format.

    The TODAY function returns today's date as a SAS date value.

  3. Write a PROC PRINT step that prints data in the Sasuser.Courses data set. Precede this step with a TITLE statement that prints the following title on the report, substituting the value of the macro variable date for the Xs:

    "Courses Offered as of XXXXXX"

    Submit the program and examine the output.

  4. Modify the DATA step so that value of the macro variable date is written using the WORDDATE20. format (month dd, year). Submit the program and examine the output.

  1. Submit the following OPTIONS statements: 
    options nodate symbolgen;
  2. Write the following DATA step: 
    data _null_;
call symput('date',put(today(),mmddyy10.));
run;
  3. Add the following statements to your program: 
    title "Courses Offered as of &date";
proc print data=sasuser.courses;
run;
     Submit the program and examine the output. Remember that you must enclose your title in double quotation marks in order for the macro variable reference to resolve.

    call symput('date',put(today(),worddate20.));
