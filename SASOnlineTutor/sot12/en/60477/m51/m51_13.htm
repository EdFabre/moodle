Creating a Macro Variable During DATA Step Execution (continued)
Write a statement that creates a macro variable named daily_fee whose value is the cost of a class per day of instruction according the the data set shown below. Use the DOLLAR8. format.


Partial SAS Data Set Courses
Course_Code Course_Title Days Fee
C001 Basic Telecommunication 3 $795
C002 Structured Query Language 4 $1150
C003 Local Area Networks 3 $650
C004 Database Design 2 $375 


%let crsnum=C003;
data price;
    set sasuser.courses;
    where course_code=&crsnum;
    
run;
