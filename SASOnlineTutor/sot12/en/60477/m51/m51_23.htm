Create multiple macro variables, using the Sasuser.Schedule data set.

Be sure to complete this practice. You'll use the macro variables that you create here in later practices.

  1. Write a DATA step that will create a series of macro variables named start1 through startn, one for each course in the Sasuser.Schedule data set. The value of each start macro variable should be the starting date of the corresponding class (that is, the value of the data set variable Begin_date in Sasuser.Schedule) in the MMDDYY10. format.

    Add a %PUT statement to display the user-defined macro variables in the SAS log. Submit the program and examine the log.

    Remember that you can specify the _USER_ argument in the %PUT statement in order to list all user-defined macro variables and their values in the SAS log.

  2. Copy the program below and paste it into the code editing window: 
    
%let crs=4;
proc print data=sasuser.all noobs n;
   where course_number=&crs;
   var student_name student_company;
   title 
     "Roster for Course &crs Beginning on XXXXX";
run;
    Modify the TITLE statement so that the series of Xs is replaced with an indirect macro variable reference to one of the start variables, based on the current value of crs. Submit the program and examine the output.

  3. Change the value of crs to 9. Submit the program again and examine the output.
