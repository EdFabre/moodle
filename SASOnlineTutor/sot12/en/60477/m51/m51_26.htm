Resolve macro variables with the SYMGET function, using the Sasuser.Register data set.

  1. In the previous practice you created the macro variables Start1 - Start18. The value of each variable is the starting date for the corresponding course number. You can view these variables in the SAS log if you need a reminder.

  2. Create a temporary data set named Unpaid that contains the observations for the students in the Sasuser.Register data set who have not yet paid their registration fee.

    Create a new variable named Begin that indicates the starting date for the corresponding course number. You can obtain the value for the starting date from one of the macro variables that were referred to in step 1.

    Remember that you will need to left-align the value of Course_number in order to concatenate it to the string start when you indicate the name of the macro variable in your statement.

  3. Submit a null TITLE statement to clear any previous titles. Print the Unpaid data set and examine the output.
