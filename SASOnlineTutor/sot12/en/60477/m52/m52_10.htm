Define and execute a macro.

Be sure to complete this practice. You'll use the macro that you create here in later practices.

  1. Copy the program below and paste it into the code editing window: 

    proc print data=sasuser.all label noobs n;
   where course_number=&num;
   var student_name student_company;
   title "Enrollment for Course &num";
run;
    Convert this program into a macro named Printnum.

  2. Add a statement to set the MCOMPILENOTE option so that a NOTE will be written to the SAS log when the macro compiles. Submit the macro definition to compile the macro, and check the log to be sure that it has compiled successfully.

    Remember that the MCOMPILENOTE option is available beginning in SAS 9. If you are working in the an earlier version of SAS or the SAS Learning Edition, skip this step.

  3. Submit a %LET statement to assign the value 6 to the macro variable num. Then call the macro that you defined in the previous step.
