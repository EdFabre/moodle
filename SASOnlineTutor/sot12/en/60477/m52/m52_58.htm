Use the %EVAL function and the %SYSEVALF function.

  1. Copy the following statements and paste them into the code editing window. The %LET statements create a macro variable named saldiff that has a value of the difference of two salaries. The %PUT statement prints a message with this value to the SAS log. 
    %let sal1=25000;
%let sal2=27000;
%let saldiff=%eval(&sal2-&sal1);
%put The salary difference is &saldiff;
    Submit the program and examine the messages that are written to the SAS log.

  2. Revise the program so that the value of sal2 is 27050.45 and submit the program again. Examine the messages that are written to the SAS log.

  3. Revise the program so that the assignment statement for saldiff can accept non-integer values. Submit the program again and examine the messages in the SAS log.
