Define and execute a macro that includes keyword parameters.

  1. Copy the program below and paste it into the code editing window: 

    %let start=%upcase(&start);
%let stop=%upcase(&stop);
proc freq data=sasuser.all;
   where begin_date between "&start"d and "&stop"d;
   table location / &opts;
   title1 "Enrollment from &start to &stop";
run;
    Convert this program to a macro named Attend2. Use keyword parameters in your macro definition to define values for the macro variables opts, start, and stop. Use a null value as the default value for opts, 01jan2001 as the default value for start, and 31dec2001 as the default value for stop. Submit the macro definition to compile the macro.

  2. If you have not already done so, specify the appropriate system options to display the source code that is received by the SAS compiler and to track the macro's execution.
