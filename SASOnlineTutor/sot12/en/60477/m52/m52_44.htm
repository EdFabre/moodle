Define and execute a macro in order to conditionally process SAS language steps.

  1. Copy the program below and paste it into the code editing window. This program takes all observations from the Sasuser.All data set that have a Begin_date in 2001 and writes them to a temporary data set with a Begin_date in the current year. 
    data current(drop=diff);
   set sasuser.all;
   if year(begin_date)=2001;
      diff=year(today())-year(begin_date);
      begin_date=begin_date+(365*diff);
run;
    Submit the program.

  2. Copy the program below and paste it into the code editing window: 
    proc sort data=work.current out=thisweek;
   where put(begin_date,monyy7.)=
         "%substr(&sysdate9,3,7)"
         and begin_date ge "&sysdate9"d;
   by begin_date location course_title;
run;
proc print data=thisweek noobs n;
   by begin_date location course_title;
   var student_name student_company paid;
   title "Course Registration as of &sysdate";
run;
    Convert this program to a macro named Reports.

  3. Add the following steps to the macro so that they will be processed only if the macro is executed on a Friday. Hint: Use the automatic macro variable SYSDAY.

    proc sort data=work.current out=thisweek;
   where put(begin_date,monyy7.)=
         "%substr(&sysdate9,3,7)"
         and begin_date le "&sysdate9"d;
   by begin_date location course_title;
run;
proc means data=thisweek maxdec=0 sum;
   by begin_date location course_title;
   var fee;
   class paid;
   title "Revenue for Courses as of &sysdate9";
run;
    Submit the macro to compile it.
