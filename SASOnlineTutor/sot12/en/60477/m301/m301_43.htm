Generating and Editing Query Code

Just as you can generate a SAS program by using a task window, you can also generate Structured Query Language (SQL) code by using the Query Builder in SAS Enterprise Guide.


Example: Overview of the Query Builder

Select buttons below to view some of the tasks you can perform using the Query Builder. You can also see a text-only version of the information.


Editing Query Code

If you want to edit the SQL code that the Query Builder generates, you must edit a copy of the code. You cannot insert lines of code into the query as you can in other SAS Enterprise Guide tasks. You can do either of the following to edit a copy of the query code:

  • right-click the query in the Project window and select Add as Code Template

  • open the query code by double-clicking the Code item under the query in the Project window. Press the space bar or another key anywhere in the Code window. You will be prompted whether you want to save the code as an editable copy. Click Yes.

Next, you set options to create HTML results from the query and then you begin using the Query Builder.
