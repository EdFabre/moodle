Working with Results
If your program includes procedure code that produces output, HTML results open automatically in SAS Enterprise Guide. You can select other types of output to produce by selecting result formats in the Options window. You can select from formats such as the following:
  • HTML, which is the default type of output
  • Text, which is traditional SAS listing output
  • RTF
  • PDF
  • any combination of these formats.

Although you can create more than one type of output, each one takes time to create. The more types of output that you create, the longer it will take to run your task.


Specifying Result Formats

You can specify default result formats for all tasks and programs, and you can override the default settings for individual tasks and programs. HTML is the default result format for SAS Enterprise Guide.

To specify default result formats for all programs, you open the Options window and specify result formats on the Results tab. To specify override the defaults for an individual program, you open the Properties window for the program or task and specify result formats on the Results tab.
