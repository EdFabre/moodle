Writing and Submitting SAS Code (continued) Edit and resubmit a SAS program.
  1. In the Project window, double-click the Sales code item.


  2. Replace the WHERE clause with the following:

       where (actual-predict)>900;




  1. Select CodeRun on <server-name>.

  2. In the message window, click No.

  3. View the new results. The results show only those rows in which actual sales exceeded predicted sales by more than $900.00.


Click to view the full window.


  1. Now look at the Project window. There is a new code item named Code1 that contains revised code. The new log and results are saved there too. If you had selected Yes in the message window, the Sales code would be replaced with the revised code, and the results, log, and data would also be replaced.




  1. In the Project window, click Code1 and change the name to HighSales.
