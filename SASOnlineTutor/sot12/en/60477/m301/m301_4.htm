Touring the Programming Windows (continued)

The Project Window

SAS Enterprise Guide uses projects to manage a collection of related data, tasks, code, and results. Each time you start SAS Enterprise Guide, you must either create a new project or open an existing project. Only one project at a time can be open in SAS Enterprise Guide.

You insert code, data, or a note in a project by selecting the appropriate item from the Insert menu. After you submit code or run a task on data, the log and any results appear as items under the associated code in the project.




You can collapse and expand items in a project to hide or display the contents. You can also use pop-up menus to perform actions on items in the project (such as viewing properties, renaming, or deleting).

When you save a project, you save a single file with the extension .SEG.

If you close the Project window, you can reopen it by clicking the Project tool () on the toolbar or by selecting View Project. Note that closing the Project window does not close the project in SAS Enterprise Guide.

In SAS Enterprise Guide, the work that you do is saved in projects. A project is a collection of related data, tasks, code, and results. Each time you start SAS Enterprise Guide, you must either create a new project or open an existing project.

There are two different windows that you can use to view the items in your project — the Process Flow window and the Project window. The tabs at the bottom of the window enable you to select the view that you want to see. In this lesson you will use the Project window.


The Process Flow Window

The Process Flow window opens by default and displays the items in a project and the relationship between the items. This display is also called the process flow diagram (PFD) for the project. As you add data and code to your project and run tasks, items are added to the process flow diagram.


You can also build your own process flow diagram that orders or groups tasks and code that you want to execute. Although only one project at a time can be open in SAS Enterprise Guide, you can have multiple process flow diagrams for one project.


The Project Window

The Project window also displays the items in a project in SAS Enterprise Guide. The Project window is a hierarchical view of the active project and its associated data, code, notes, and results.




You can collapse and expand items in a project to hide or display the contents. You can also use pop-up menus to perform actions on items in the project (such as viewing properties, renaming, or deleting).

When you save a project, you save a single file with the extension .EGP.

If you close the Project window, you can reopen it by clicking the Project tool () on the toolbar or by selecting View Project. Note that closing the Project window does not close the project in SAS Enterprise Guide.
