  1. In the Project window, right-click the Bar Chart item and select Add as Code Template. A new item named Code for Bar Chart is added to the project and the code opens in a code window. You can edit this code.

  2. You want to create a bar chart that shows net sales by region. The Shoes data does not contain a column for net sales, but you can write a short DATA step to create one. Make the following changes to the code:

    • Find the PROC SQL query that creates the View. Add the variable Returns to the SELECT statement so you can use the column in the DATA step. The name of the view that is created by this procedure will be different in your code, so take note of the name so you can reference it later in the DATA step.





    • Find the TITLE statements. Change TITLE1 to Net Sales by Region. Copy the following DATA step and paste it after the FOOTNOTE statements. Edit the SET statement to reference the view that was created by the query.
         data work.net;
      set work.sort7680;
      netsales = sales - returns;
      format netsales dollar12.;
      label netsales="Net Sales";
   run;


    • In the GCHART procedure, use work.net for the input data and change the SUMVAR variable to Netsales.

  1. Press F8 to run the code.

  2. The results open and you can see that the bar chart now represents net sales for each region.
