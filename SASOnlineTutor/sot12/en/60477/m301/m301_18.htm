Storing and Accessing Code (continued)

Accessing and Running Code

If you have a SAS program that you have stored, you can open it in SAS Enterprise Guide by using any of the following methods:

  • select Insert Code and click the Existing tab to browse for the file
  • use the Server List to open the active server and then locate the program in the Files folder
  • drag and drop from Windows Explorer.

When you open a saved SAS program, you create a shortcut to the file in the Project window.





Any changes that you make to the code in SAS Enterprise Guide are made to the code file. Alternatively, you can embed the file in the project so that your changes do not affect the original file.

Next, you add a program to your project and run the code.
