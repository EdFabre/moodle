Add a SAS data set and a Microsoft Excel file to your project.

  1. Using the Server List, add the PRDSALE data from the SASHELP library to the project.

    • Select ViewServer List.
    • Double-click Local (or the active server in the Server List).
    • Double-click Libraries, then double-click SASHELP.
    • Double-click PRDSALE. The data set is added to the project. Close the Server List.
  2. Using the Insert Data window, add the Microsoft Excel file SupplyInfo(Suppliers$) to the project. The file is located in the sample data folder.

    • Select InsertData.
    • Navigate to the SAS Enterprise Guide sample data folder.
    • Click the Files of type drop-down list and select Microsoft Excel Files(*.xls).
    • Double-click SupplyInfo.xls.
    • Double-click Suppliers$.
