Additional Features

Turning Off the Code That SAS Enterprise Guide Submits

As you learned in this lesson, SAS Enterprise Guide automatically submits code to set options and produce HTML output. If you want to turn this feature off, go to the SAS Programs tab in the Options window and clear the Automatically direct results back to Enterprise Guide check box.





When you clear this option, the only additional code that will be added to your code are QUIT and RUN statements. Be aware that HTML results will no longer be displayed in SAS Enterprise Guide as long as this option remains off.

Displaying All the Code That SAS Enterprise Guide Submits

If you want to see all the code that SAS Enterprise Guide submits to SAS, go to the SAS Programs tab in the Options window and select Display all generated SAS code in code output. When you run your own code, a new code item is added under your code in the Project window. Open the code to see all the code that SAS Enterprise Guide submitted.
