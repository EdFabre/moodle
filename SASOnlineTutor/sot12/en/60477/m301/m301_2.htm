Understanding SAS Enterprise Guide and SAS Software

SAS Enterprise Guide is a Windows client application that uses the processing power of SAS software. You can use SAS Enterprise Guide to write SAS programs and submit them to SAS for processing. When SAS finishes processing the code, it sends the results and log messages back to SAS Enterprise Guide to display.

SAS can either be on the same computer with SAS Enterprise Guide or it can be on a remote server. The connection between SAS Enterprise Guide and SAS is set up by an administrator.

Starting a SAS Session

With SAS Enterprise Guide, a SAS session begins when you perform a task that connects with SAS. In this lesson, a SAS session starts when you submit code to SAS, when you add data to a project, or when you open the server in the server list.

The first time you submit a SAS program, it will take slightly longer to run because the connection with SAS is being established. Programs that you submit subsequently will run faster.
