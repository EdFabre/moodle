Create a bar chart by using the Bar Chart task and then edit the code to include two charts in the same results.

This practice uses data that was added to the project in the last practice. If you haven't added the data, return to the practice and complete it before continuing here.

  1. Select the PRDSALE data set and start the Bar Chart task. Select a Vertical Colored Bars chart.

    • In the Project window, select PRDSALE.
    • Select GraphBar.
    • On the Chart Gallery tab, select Vertical Colored Bars.
  2. Assign PRODUCT to the Column to chart role and ACTUAL to the Sum by role.

    • Click the Columns tab.
    • In the Columns to assign list, select PRODUCT and drag and drop it on the Column to chart role.
    • In the Columns to assign list, select ACTUAL and drag and drop it on the Sum by role.
  3. Add the title Total Sales by Product and delete the footnote.

    • Click the Appearance tab, then click the Titles button.
    • In the Title box, type Total Sales by Product.
    • Delete the text in the Footnote box.
  4. Add the Sum value above each bar. Run the Bar Chart task. Which product had the highest sales?

    • Click the Advanced tab.
    • Select the Additional statistical value to show next to bar check box.
    • Click the drop-down list and select Sum.
    • Click Finish.
    • Sofas had the highest sales at $117,751.00.
  5. Edit the task code to add a second bar chart that displays total predicted sales for each product. Make the following changes to the code:

    • In the Axes statements, change the label for Axis1 to label=('Sales').


    • Copy the GCHART procedure and paste the copy below the existing GCHART procedure.


    • Add the following statement between the two GCHART procedures:
      TITLE "Predicted Sales by Product";


    • In the second GCHART procedure, change SUMVAR=ACTUAL to SUMVAR=PREDICT.


    • Change the description to DESCRIPTION="Predicted Sales by Product".

    • In the Project window, right-click Bar Chart under the Prdsale data and select Add as Code Template.
    • Revise the program as specified in the step.
  6. Run the code. Look at the two charts. For which product were actual sales less than predicted sales?

    • Press F3.
    • Total actual sales for tables was less than the total predicted sales.

  1. Select the PRDSALE data set and start the Bar Chart task. Select a Vertical Colored Bars chart.

    • In the Project window, select PRDSALE.
    • Select GraphBar Chart.
    • Click Bar Chart in the selection pane.
    • Select Vertical Colored Bars.
  2. Assign PRODUCT to the Column to chart role and ACTUAL to the Sum of role.

    • Click Task Roles in the selection pane.
    • In the Columns to assign list, select PRODUCT and drag and drop it on the Column to chart role.
    • In the Columns to assign list, select ACTUAL and drag and drop it on the Sum of role.
  3. Add the title Total Sales by Product and delete the footnote.

    • Click Titles in the selection pane.
    • Clear the Use default text check box and delete the default title.
    • In the Title box, type Total Sales by Product.
    • Select Footnote in the Section box and clear the Use default text check box.
    • Delete the text in the Footnote box.
  4. Add the Sum value above each bar. Run the Bar Chart task. Which product had the highest sales?

    • Click Advanced in the selection pane.
    • Select the Additional statistical value to show next to bar check box.
    • Click the drop-down list and select Sum.
    • Click Run.
    • Desks had the highest sales at $149,232.00.
  5. Open an editable version of the task code and make the following changes to the code:

    • In the SQL procedure that creates the view, add PREDICT to the SELECT statement as follows:
      AS SELECT PRODUCT,ACTUAL,PREDICT FROM SASHELP.PRDSALE;


    • In the Axis1 statement, add 'Sales' to the LABEL= option.


    • Copy the GCHART procedure and paste the copy below the existing GCHART procedure.


    • Add the following statement between the two GCHART procedures:
      TITLE "Predicted Sales by Product";


    • In the second GCHART procedure, change sumvar=actual to sumvar=predict.

    • In the Project window, right-click the Bar Chart task under the Prdsale data and select Add as Code Template.
    • Revise the program as specified in the step.
  6. Run the code. Look at the two charts. For which product were actual sales less than predicted sales?

    • Press F8.
    • Total actual sales for tables was less than the total predicted sales.
