Create a new project, view libraries, and save the project.

  1. Create a new project and name it ProgWithEG.

    • With the Project window active, select FileNew.
    • In the New window, select the Project icon and type ProgWithEG in the Name box.
  2. Determine the server that SAS Enterprise Guide is accessing.

    • Select ToolsAdministrator Options.
    • Find the server that is highlighted in the Default SAS server list. This is the server that SAS Enterprise Guide is using to process the programs that you submit.
    • Click Cancel.
  3. Using the Server List, open the active server. What libraries are assigned?

    • Select ViewServer List.
    • Double-click the default server.
    • Double-click Libraries.
    • The Sashelp, Sasuser, and Work libraries are assigned. There might be other libraries assigned as well.
  4. Save the project as ProgWithEG.seg.

    • Select FileSave ProgWithEG.
    • Select the default location or another location of your choice. The project file is automatically saved with a .SEG extension.

  1. Create a new project.

    • With the Project window active, select FileNewProject.
  2. Determine the server that SAS Enterprise Guide is accessing.

    • Select ToolsOptions.
    • Click Administration in the selection pane.
    • Find the server that is highlighted in the Default SAS server list. This is the server that SAS Enterprise Guide is using to process the programs that you submit.
    • Click Cancel.
  3. Using the Server List, open the active server. What libraries are assigned?

    • Select ViewServer List.
    • Double-click the default server.
    • Double-click Libraries.
    • The Sashelp, Sasuser, and Work libraries are assigned. There might be other libraries assigned as well.
  4. Save the project as ProgWithEG.egp.

    • Select FileSave Project As.
    • In the Save Project To window, select Local Computer.
    • In the Save As window, select the default location or another location of your choice.
    • In the File name box, type ProgWithEG.
    • Click Save. The project file is automatically saved with an .EGP extension.
