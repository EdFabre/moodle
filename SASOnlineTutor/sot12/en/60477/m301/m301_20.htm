Save a SAS program and add a saved SAS program to your project.

This practice uses programs that were created in the last practice. If you haven't created these programs, return to the practice and complete it before continuing here.

  1. Export a copy of the ProdSales code to a location on your computer.

    • In the Project window, right-click ProdSales and select Save ProdSales As/Export.
    • Select a location on your computer and click Save.
  2. Change the properties of the Code1 program so that it points to a saved location. Name the program DeskSales.

    • In the Project window, right-click Code1 and select Properties.
    • Click the Save As button next to the File location box.
    • In the Name box, type DeskSales.
    • Select a location on your computer and click Save.
    • Click OK to close the Properties window.
  3. Add the ProdSales code that you exported. Rename the program SofaSales.

    • Select Insert Code.
    • Click the Existing tab.
    • Navigate to the location where you saved the ProdSales code.
    • Double-click ProdSales.
    • In the Project window, right-click the ProdSales code that you just added and select Rename from the pop-up menu.
    • Type SofaSales.

  1. Export a copy of the ProdSales code to a location on your computer.

    • In the Project window, click ProdSales and select FileExport ProdSales.
    • In the Export ProdSales To window, select Local Computer. Select a location and click Save.
  2. Save the second ProdSales program so that you create a shortcut to the program in the Project window. Name the program DeskSales.

    • In the Project window, right-click ProdSales and select Save ProdSales As.
    • In the Save To window, select Local computer. Select a location.
    • In the File name box, type DeskSales. Click Save.
    • Notice that the code in the Project window is renamed DeskSales and is a shortcut to the external file.
