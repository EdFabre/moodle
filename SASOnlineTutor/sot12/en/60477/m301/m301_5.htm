Touring the Programming Windows (continued)

The Task List

You use the Task List to select analytical tasks to run on your data and to add items to the project. The Tasks by Category tab lists tasks that are grouped into categories of similar types of tasks.

All tasks have a corresponding menu path that you can use if you prefer to close the Task List.





The Tasks by Name tab has an alphabetical list of tasks that includes the name of the SAS procedure that is associated with each task. You may want to explore this list to see which SAS procedures have point-and-click tasks available in SAS Enterprise Guide.


If you close the Task List, you can restore it by clicking the Task List tool () on the toolbar or by selecting ViewTask List.


The Task Status Window

The Task Status window displays messages about the status of your programs and tasks as they are running. If you submit more than one task or program, the position of each program or task in the queue is displayed in the window. Simple programs run so quickly that you may not notice messages in the Task Status window.





The Task Status window does not replace the SAS log. You will learn more about viewing the log later in this lesson.

Now that you have seen the windows that are open by default in SAS Enterprise Guide, let's look at some additional windows that you will use for your programming tasks.
