Touring the Programming Windows (continued)

The Code Editor

In SAS Enterprise Guide, you use the Code Editor window to enter, edit, and submit SAS programs. The Code Editor is similar to the Enhanced Editor in the SAS windowing environment and includes the same features that make editing and debugging SAS programs easier. You can open a Code Editor window by using any of these methods:

  • selecting InsertCode
  • clicking the Insert Code tool () on the toolbar
  • double-clicking Create Code in the Task List.

When a new Code Editor window opens, an item for the code is added to the Project window. The default name for the Code Editor window is Coden. You specify a name in the Insert Code window or rename the code item in the Project window.

Text that you enter into the Code Editor window is automatically saved when you close the window.





You can customize the Code Editor (such as displaying line numbers, setting tab size, and customizing the appearance) by selecting ToolsOptions and clicking the Editor tab.

For more information about the features of the Code Editor window, select Help Enterprise Guide Help and open the topic Working with Code from the Contents tab. You can also place your cursor in a blank area of the Code Editor window and press F1.
