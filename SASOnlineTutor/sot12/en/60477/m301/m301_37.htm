Inserting Code into a Task

Another option for customizing code is to insert lines of code into the task code. This code remains with the task so that any updates that you perform using the task window include your inserted code. When you use this method you can only add lines of code. You cannot edit the code that SAS Enterprise Guide generates.

To insert custom code into the task, you click the Insert Code button in the Code Preview window. Then you can select where you want to add lines of code in the Add User Code window. The code that you add is not validated by SAS Enterprise Guide.
