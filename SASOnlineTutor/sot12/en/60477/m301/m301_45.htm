Generating and Editing Query Code (continued) Add a computed column to the query results.
  1. In the Query Builder window, click the Select and Sort tab.

  2. Click New.


  3. In the Alias box, type StockLevel. This is the name for the new column.


Click to view the full window.


  1. Click the Expression tab. You use this tab to build an expression to calculate values for the StockLevel column. You can select columns and operations to build the expression, or you can copy and paste the following expression into the expression box at the top of the window:

    (Products.UnitsInStock + Products.UnitsOnOrder) - Products.ReorderLevel


Click to view the full window.


  1. Click OK to add the new column to the query.


Click to view the full window.
