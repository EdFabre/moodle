Adding Other Data Types

One of the advantages of SAS Enterprise Guide is that it enables you to access and view many types of data. In addition to SAS data sets, you can access data in other formats such as Microsoft Excel, Microsoft Access, Lotus, text, and HTML.

You can add data files by doing either of the following:

  • Use the Server List to open the server and then add data from the Files folder.

  • Select InsertData and click the Existing tab. Click the Files of type drop-down list and select All Files (*.*) to locate and add data.

When you add data files to your project, they open in the data grid but you cannot modify them.





Keep these points in mind when you are working with other data types in SAS Enterprise Guide: 

  • The Import Data task can import Microsoft Excel files and delimited and fixed-width text files as SAS data sets that you can manipulate. You can export a SAS data set as a Microsoft Excel file when you are finished.

  • There are options that you can set that affect how SAS Enterprise Guide accesses a Microsoft Excel file. You can view and set these options by using the Options window. Click the Data and Query tab, then click Data Schema Options. The Use longest value as column width option overrides the default length (255 bytes) of character values.

  • You can access files from databases such as Oracle and DB2 by using licensed SAS database engines, and you can access MDDBs with additional SAS software.

  • For more information about accessing various types of data, select HelpEnterprise Guide Help and read the topics in the Working with Data folder.
