Writing and Submitting SAS Code (continued) Write and submit a simple SAS program.
  1. Select InsertCode.


  2. In the Name box, type Sales.


Click to view the full window.


  1. Copy and paste the following program into the Code Editor window:
    data sasuser.myproductsales;
   set sashelp.prdsale;
run;
proc print data=sasuser.myproductsales;
   where actual>500;
run;




  1. Select CodeRun on <server-name>.

  2. Notice that the PROC PRINT output opens automatically in the workspace. The report lists all rows in which actual sales were greater than $500.00.

  3. Look at the Project window. Items for the results, output data, and log have been added under the Sales code item.
