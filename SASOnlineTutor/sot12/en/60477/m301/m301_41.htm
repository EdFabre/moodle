Generating and Editing Task Code (continued) Submit a SAS program to test the options.
  1. Select InsertCode. In the Insert Code window, click OK.


  2. Type or copy and paste the following code into the Code Editor window: 
    proc print data=sashelp.shoes; 
run;
  3. Press F3 to submit the code.

  4. Look at the HTML results. As expected, only 10 observations are included in the results.
