Generating and Editing Task Code

You have learned how to write and submit code by using the Code Editor in SAS Enterprise Guide. Now let's see how you can use the point-and-click tasks in SAS Enterprise Guide to do some of the programming for you.

SAS Enterprise Guide has tasks for many of the SAS procedures that you use in your programs. These tasks enable you to quickly and easily select variables and options to run a procedure. However, not all options available in a SAS procedure are available through the point-and-click task window. You can save time by letting the task generate the basic code and then, if necessary, editing the code to include additional options.





You can export the code that is generated by using task windows and run the code in any SAS environment. However, you might need to adjust librefs if SAS Enterprise Guide has created libraries for the data that you have used.

There are three ways that you can customize task code:

  • Separate the generated code from the task and edit it.
  • Insert lines of code into the task code.
  • Specify code that you always want to insert before or after task code.

You will try each of these methods in this lesson, but first let's learn a little about SAS Enterprise Guide tasks.
