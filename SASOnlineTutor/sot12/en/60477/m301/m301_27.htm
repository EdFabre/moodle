Working with Data (continued) Add a SAS data set from the SASHELP library to the project.
  1. If the Server List is closed, select ViewServer List .

  2. Double-click the default server to open it (if SAS is installed on your machine, the server will be Local).

  3. Double-click Libraries, then double-click SASHELP.

  4. Double-click SHOES.

    You can also use the Insert Data window and click Servers in the Places bar on the left side of the window.




  1. In the Project window, notice that a shortcut to the SHOES data has been added. The data opens in the data grid in read-only mode.




Next, you add Microsoft Excel data to your project.
  1. If the Server List is closed, select ViewServer List .

  3. Double-click Libraries, then double-click SASHELP.

  4. Double-click SHOES.

    You can also select FileOpenFrom SAS Server/Binder. Click Servers in the Look in section on the left side of the window.




  1. In the Project window, notice that a shortcut to the SHOES data has been added. The data opens in the data grid in read-only mode.
