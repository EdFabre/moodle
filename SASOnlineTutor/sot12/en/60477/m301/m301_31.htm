Generating and Editing Task Code

SAS Enterprise Guide Tasks

A task is an analysis that you perform or a report that you create from your data. Tasks are based on SAS procedures. Remember that you can use the Tasks by Name tab of the Task List to see a list of tasks and the SAS procedure that corresponds to the task. In SAS Enterprise Guide, task windows have a common format, so once you are familiar with running one task, running other tasks is easy.

You can learn more about using SAS Enterprise Guide tasks in the Getting Started with SAS Enterprise Guide tutorial. The tutorial is available from the Help menu.


Using the SHOES data, suppose that you want to see the average dollar amount of returns for each sales region. You can use the Bar Chart task to make a chart that shows this information.
