Create an index on the Sasuser.Flighttimes data set in a DATA step.

Be sure to complete this practice. You'll use the indexes that you create here in later practices.

  1. Use the DATA step and the INDEX= option to create the Sasuser.Flighttimes data set and two indexes:

    • a simple index based on the Date variable
    • a unique composite index named Fltdte, based on the Flight and Date variables.

    Use the following INFILE, INPUT, and FORMAT statements in your DATA step: 
    infile flighttm dlm=",";
input flight $ depart date;
format Date date9.;
  2. Add a statement to the program to set the MSGLEVEL= system option so that messages about the index will be written to the SAS log.

  3. Submit the program and examine any messages that are written to the SAS log.
