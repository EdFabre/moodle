Perform maintenance tasks and view a list of the indexes on the Sasuser.Flighttimes data set.

This practice uses indexes that were created in a previous practice.

  1. Write a PROC CONTENTS step that lists the contents of the Sasuser.Flighttimes data set. Submit the program and examine the output.

  2. Write a PROC DATASETS step that copies the Sasuser.Flighttimes data set into the Work library.

  3. Write another PROC DATASETS step that renames Work.Flighttimes to Work.Departures. Submit the program.

  4. Write a PROC DATASETS step that lists the contents of the Work.Departures data set. Submit the program and examine the output. Compare the list of indexes for Work.Departures to the list of indexes for Sasuser.Flighttimes that was generated in Step 1 above.
