Finding Your Way Around

The main tools for navigating among and within SAS online lessons are the Learning Path page and the drop-down Lesson Contents menu.

To open the Learning Path page, click Learning Path on the home page or click Path on the navigation bar at the top of lesson pages. To open the Lesson Contents menu, click Lesson Contents on the navigation bar at the top of lesson pages.

You can also access these features through a set of keyboard shortcuts.





Navigation Link Function Keyboard Shortcut
previous page Alt-b (+ Enter)
next page Alt-n (+ Enter)
Lesson Contents lesson menu Alt-m (+ Enter)
Path or Learning Path Learning Path page Alt-p (+ Enter)
Help     help menu Alt-h (+ Enter)


Once you begin a lesson, use the(back) and (next) arrows on the navigation bar to move page by page through the topics. Or you can jump to any topic in a lesson by selecting it from the Lesson Contents menu. Each time you reach a topic, e-learning for SAS programming saves your place so that you can resume where you left off.

In the next few pages of the tour, you'll see additional details about each of these features.
