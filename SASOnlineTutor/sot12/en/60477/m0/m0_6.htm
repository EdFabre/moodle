Looking at Lesson Content

Each lesson in e-learning for SAS programming uses some or all of the following features and a consistent structure so that you can concentrate on learning the material.



Overview
Lessons start with an overview that explains the concepts you will learn and the prerequisites for the lesson.
Each topic in the lesson includes instructional material such as text explanations, code and output, graphics, and animations to facilitate learning.
Interactive questions help you gauge your understanding of the material as you go along.
Guided practices let you practice your new programming skills.
Lesson Summary
Lessons end with a summary that includes a review with links back to specific topics.
A quiz at the end of each lesson gives you a chance to test your mastery of the lesson material.
