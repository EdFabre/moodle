Using the EXCEPT Set Operator (continued)
Which PROC SQL step combines the tables Suitcase1 and Suitcase2 to produce the output displayed below?

Suitcase1
Item Desc
1 skirt
2 shirt
2 shirt
3 black
Suitcase2
Item Color
1 blue
3 black
4 yellow
Item Desc
1 skirt
2 shirt 

proc sql;
   select *
      from suitcase1
   except
   select *
      from suitcase2;
 
proc sql;
   select *
      from suitcase1
   except all
   select *
      from suitcase2;
 
proc sql;
   select *
      from suitcase1
   except corr
   select *
      from suitcase2;
None of the above. It is not possible to obtain this output by using the keyword EXCEPT, with or without a keyword as a modifier.
