Using the INTERSECT Set Operator (continued)

You need to generate a report that lists the first and last name (and no other columns) for all authors who have had a book published by XYZ Books, Inc., and who have won a national literary prize. You will use a set operation to combine data from two tables:

  • Authors lists information for all authors who have had books published by XYZ Books, Inc., some of whom have won a national literary prize.

  • Prizewinners lists information for all authors who have won a national literary prize, some of whom have had books published by XYX Books, Inc.

The first few rows of each table are shown below, along with the first few rows of the output that you want.

Authors
LName FName PubNum
Aardema Doris xyz_432
Abbot Ted xyz_300
Adelman Leah xyz_105
Prizewinners
LName FName Year
Adelman Leah 1991
Brown Anna 1985
Bulina Carson 2000

LName FName
Adelman Leah
Collins William
Detry Claude


Which query will generate the results that you want?

 
proc sql;
   select *
      from authors
   intersect corr
   select *
      from prizewinners;
 
proc sql;
   select *
      from prizewinners
   intersect corr
   select *
      from authors;
 
proc sql;
   select lname, fname
      from authors
   intersect
   select lname, fname
      from prizewinners;
all of the above
