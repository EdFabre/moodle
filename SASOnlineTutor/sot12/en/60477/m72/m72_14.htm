Use the set operator EXCEPT to combine the tables Sasuser.Empdata and Sasuser.Allemps.

The relationship between the two tables is described below:

  • Sasuser.Allemps contains rows for a subset of the employees in an airline's Airport Operations division.

  • Sasuser.Empdata contains rows for a subset of the employees in each of the airline's six divisions, including Airport Operations.

  • A certain number of employees are listed in both tables. However, each table lists at least one or more employees in the Airport Operations division that are not listed in the other table.
To view the expected PROC SQL output, click the related link in the text for each step.

  1. Write a PROC SQL step that uses a set operation to display all of the employees that are listed in Sasuser.Empdata except for those employees that are also listed in Sasuser.Allemps. In each of the two queries, select the columns EmpID, LastName, Division, and Location. Add a QUIT statement to the step.

  2. Submit the step and view the output. How many employees are listed?
