Use Sasuser.Flights2 and Sasuser.Wchill to create a new SAS data set.

  1. Sasuser.Flights2 contains temperature and wind speed information. Sasuser.Wchill contains data values in a coordinate chart structure that helps to calculate wind chill values. Wind chill is based upon temperature and wind speed. The columns represent temperatures and the rows represent wind speeds.

    Create a new SAS data set, titled Work.Wndchll2. It should contain the same variables as Sasuser.Flights2 plus the wind chill for each flight.

    If necessary, use the example from earlier in this lesson as a guide to calculate the row and column values.
