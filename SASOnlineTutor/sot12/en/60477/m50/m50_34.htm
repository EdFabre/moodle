Using Macro Functions to Mask Special Characters (continued)
What will happen when the following code is submitted? 
%let example=proc print data=class.beg; where rate=free; run;
The macro variable example will be created with the value
proc print data=class.beg; where rate=free; run; 		 
The macro variable example will be created with the value
proc print data=class.beg, and SAS will issue an error message.
