Save a macro as a catalog SOURCE entry.

  1. Copy the following program and paste it into the code editing window. Do not submit the program. 
    %macro sortlast(sortby);
   %if &syslast ne _NULL_ %then %do;
      proc sort data=&syslast out=sorted;
         by &sortby;
      run;
   %end;
   %else
      %put No data set has been created yet.;
%mend;
     Save the program in a SOURCE entry in the SAS catalog Sasuser.Mymacs. You might need to create the Mymacs catalog before you save the SOURCE entry in it.

  2. Make sure that the code editing window is clear. Write a statement that will compile the Sortlast macro that you created and saved in Step 1. Also write a step that will create a new data set named Course1 that contains all of the observations from the data set Sasuser.Register that have a value of 1 for Course_number. Submit this code.

  3. Write a call to the Sortlast macro. Use paid as the parameter value in the macro call. Also write a step to print the new data set (Work.Sorted) that the Sortlast macro creates. Submit this code and examine the output.

  4. Write a PROC CATALOG statement to display the contents of the Sasuser.Mymacs catalog. Submit the statement and examine the output.
