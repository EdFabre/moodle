Create and access a stored compiled macro.

  1. Copy the following program and paste it into the code editing window: 
    %macro printit(dataset);
   proc print data=&dataset(obs=5);
      title "Listing of &dataset data set";
   run;
%mend;
    Modify this macro definition in order to save it permanently in the Sasuser.Sasmacr catalog. Submit the program.

  2. Write a PROC CATALOG step to list the contents of the Sasuser.Sasmacr catalog. Submit the program and examine the output to verify that the Printit macro has been saved to the catalog.

  3. Suppose you were to exit your SAS session (do not exit SAS at this time) and begin a new session. In the new SAS session, how would you call the Printit macro to print the Sasuser.Courses data set?

   run;

    Modify this macro definition in order to save it permanently in the Sasuser.Sasmacr catalog. Submit the program.

    You will not be able to submit this code using the SAS Learning Edition, but you can practice writing this PROC CATALOG step and then compare your code to the solution code.


  3. Suppose you were to exit your SAS session (do not exit SAS at this time) and begin a new session. In the new SAS session, how would you call the Printit macro to print the Sasuser.Courses data set?
  1. Submit the following program: 
    options mstored sasmstore=sasuser;
 %macro printit(dataset)/store;
    proc print data=&dataset(obs=5); 
       title "Listing of the &dataset data set";
    run;
 %mend;
  2. Submit the following step: 
    proc catalog catalog=sasuser.sasmacr;
    contents;
 quit;
     The output should indicate that the Printit macro is now stored in the Sasuser.Sasmacr catalog.

  3. To execute the Printit macro in a new SAS session, you would submit the following code: 
    options mstored sasmstore=sasuser;
 %printit(sasuser.courses)
