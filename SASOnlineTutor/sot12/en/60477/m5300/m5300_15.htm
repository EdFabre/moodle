Execute autocall macros from a SAS catalog.

  1. Copy the following program and paste it into the code editing window:
    options mautosource mlogic;
proc print data=sasuser.courses;
   title "this title is in %lowcase(LOWERCASE)";
run;
    Submit the program. Examine the title in the output.

  2. Examine the messages that were written to the SAS log when you submitted the program in Step 1. Find the MLOGIC message that gives the location where the Lowcase macro was found. The directory that is listed is an autocall library that has been provided by SAS.
