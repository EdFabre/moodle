Updating Values in Existing Table Rows (continued)

In the table Survey, you want to change the values of the column Q1 as follows:

  • If the value of Q1 is a1, then change the value to YES.
  • If the value of Q1 is a2, then change the value to NO.
  • If Q1 has any other value, then change the value to NO ANSWER.

To update the rows as specified, complete the CASE expression in the following PROC SQL step:

 

proc sql;
   update survey 
      set q1=
         case q1
            when 'a1' then 'YES'
            when 'a2' then 'NO'
            
         end;
