Creating an Empty Table That Is Like Another Table (continued)

You want to use PROC SQL to create a new, empty table named Work.Morph_Jr. The new table should contain two of the four columns and column attributes in the existing table Morph: NeurID and SomaArea. Morph is shown below.


NeurID SomaArea Neurites BrPts
001 15.61 3 24
002 22.49 5 43


Which PROC SQL step will create Work.Morph_Jr, exactly as specified?

 
proc sql;
   create table work.morph_jr
          (NeurID num,
          SomaArea num Format=5.2)
      like morph;
 
proc sql;
   create table work.morph_jr
          (drop = neurites brpts)
      from morph;
 
proc sql;
   create table work.morph_jr 
          (drop = neurites, brpts)
      like morph;
 
proc sql; 
   create table work.morph_jr
          (drop = neurites brpts)
      like morph;
