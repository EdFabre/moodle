Deleting Rows in a Table (continued)

You are writing the following PROC SQL step:

     proc sql;
        delete from one;
           where y contains '1';

What will happen when you submit this step?
Only the rows in which the column Y contains 1 will be deleted in the table One.
All rows in the table One will be deleted, and the WHERE clause will generate an error message.
No rows in the table One will be deleted. An error message will be displayed, because the required keyword TABLE has been omitted after the keyword FROM.
