Create a temporary table that is like the table Sasuser.Newadmit, alter and drop columns in the new table, and drop the new table.

  1. Write a PROC SQL step that creates the temporary table Work.Newadmit3 as an empty table that has the same columns and attributes as Sasuser.Newadmit. In the same step, include a statement to display the structure of Work.Newadmit3 in the SAS log.

    Add a QUIT statement to the step. Submit the step and view the SAS log to verify that the table was created.

  2. Write a PROC SQL step to alter columns in Work.Newadmit3 as follows:

    • drop the columns Height, Weight, and ActLevel
    • add this label to the column Fee: Admit Fee
    • add the numeric column Pulse and assign a format of 3.

    In the same step, include a statement to display the altered structure of Work.Newadmit3 in the SAS log. Add a QUIT statement to the step. Submit the step and view the SAS log to verify that the alterations were made correctly.

  3. Write a PROC SQL step to drop the table Work.Newadmit3. Add a QUIT statement to the step. Submit the step and view the SAS log to verify that the table was dropped.
