Create a temporary table based on column specifications, insert rows of data, and join the temporary table with the existing table Sasuser.Frequentflyers.

  1. Write a PROC SQL step that creates a temporary table based on column specifications. The table is named Work.Awards and is intended to store data about awards that are earned by members of a frequent-flyer program. The table should include zero rows and three columns with the following attributes:

    • PtsReqd: numeric, length=8 (default), label=Points Required
    • Rank: numeric, length=8 (default), format=3.
    • Award: character, length=25.

    Add a QUIT statement to the step. Submit the step and view the SAS log to verify that the table was created with the correct number of rows and columns.

  2. Write a PROC SQL step to load the following rows of data into the table Work.Awards, using whichever method you prefer. Values are listed in the order in which the columns appear in the table:

    • 2000, 1, free night in hotel
    • 10000, 2, 50% discount on flight
    • 20000, 3, free domestic flight
    • 40000, 4, free international flight

    Add a QUIT statement to the step. Submit the step and view the SAS log to verify that the rows were inserted into the table.

  3. Write a PROC SQL query that displays all columns and rows of the table Work.Awards. Add a QUIT statement to the query. Submit the query and view the output.

  4. Write a PROC SQL query to generate a report that lists frequent-flyer program members and all the awards (one or more) that are available to each member, based on the member's remaining points. The query joins the tables Work.Awards, which contains data about the available awards, and Sasuser.Frequentflyers, which contains data about the number of points that each member has earned and used. In the report, display the following columns for only the frequent-flyer program members who live in Arizona (AZ):

    • FFID
    • Name
    • AvailablePoints (calculated by subtracting PointsUsed from PointsEarned)
    • Award

    Display this title: Awards for AZ Frequent Flyers. Order the report by FFID.

    Add a QUIT statement to the query. Submit the query and view the output.

  5. Submit a null TITLE statement to cancel titles.
