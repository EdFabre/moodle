Control Charts for Measurement Data (continued) Specify options for the control table.

In this task, you specify tests and select output data to save. Then you run the analysis.

  1. Click the Control Limits tab. Notice that you can customize how the control limits are applied. The default is 3 sigma limits (this refers to 3 sigma above and 3 sigma below, or 6 sigma). However, you can change the limits to another value (for example, 5.15 sigma), use previously saved control limits, or enter all the control limits on your own. Leave the default limits for this analysis.

  2. Click the Tests tab. Select all eight tests, and select the Display zone lines option. Change the Text labeling test signals color to black.
