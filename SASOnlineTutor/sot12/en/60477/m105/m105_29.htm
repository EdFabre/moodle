In this task, you specify options that allow you to save predictions from the original data table to an output data set.

  1. Click the Predictions tab. This tab allows you to save predictions from the original sample to a separate data table, or to save predictions for another data set. This tab also includes options for saving various diagnostic statistics to a separate data table.

  2. Under Data to predict, select Original sample. Under Save output data, select Predictions. Under Additional statistics, select Residuals.



Click to view the full window.


  1. Click Finish.
