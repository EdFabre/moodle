Summary Statistics (continued) Start the task and assign variables to roles.

In this task, you start the Summary Statistics task and assign variables to roles.

  1. If the Cargo data is already in your project, select it to make it the active data set. If it is not in your project, select InsertData. In the Insert Data window, navigate to the location where you saved the practice data for this lesson. Double-click Cargo to add the data to the project.

  2. Scroll the Tasks by Category list to the Descriptive category. Double-click Summary Statistics.

  3. On the Columns tab, assign Lostbags to the Analysis variables role. Assign Operator and Location to the Classification variables role.
