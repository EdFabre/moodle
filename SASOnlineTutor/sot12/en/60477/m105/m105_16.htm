t-Tests (continued) Specify titles and run the analysis.

In this task, you view the options available for the analysis and then specify a title for the report. Then you run the analysis.

  1. Click the Analysis tab. This tab allows you to specify a hypothesized mean difference. The value is set to 0 by default. Leave the default value.

  2. Click the Titles tab. In the Section box, select Analysis Titles, and change the title for this analysis to
    Paired t Test on Operator Measures.


Click to view the full window.


  1. Click Finish.

In this task, you view the options available for the analysis and then specify a title for the report. Then you run the analysis.

  1. In the selection pane, click Analysis.. Here, you can specify a hypothesized mean difference. The value is set to 0 by default. Leave the default value.

  2. In the selection pane, click Titles. In the Section box, select Analysis Titles. Clear the Use default text check box and delete the default title. Type Paired t Test on Operator Measures.
