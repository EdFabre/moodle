Distribution Analysis

Overview

It is often desirable to estimate population parameters from sample statistics. However, many commonly used inferential statistics operate under the assumption that the population is normally distributed. These analyses are generally useful only to the extent that this assumption of normality is met. Therefore, often the first hypothesis test you run is a test of the null hypothesis that the population your data was drawn from is normally distributed.

In this example, you test the Cargo data set for a normal distribution with Time as the primary outcome variable of interest. You also test a lognormal distribution to see if that is a better fit for the data. The Cargo data set has the following variables:


Variable Description
Obs unique identifier for the measurement occasion
Date date of the measurement
Location gate of the arriving flight
Operator primary employee on duty in charge of baggage handling
Time amount of time for the first bag to reach customers at baggage claim
Lostbags total number of baggage items that did not arrive with the flight
