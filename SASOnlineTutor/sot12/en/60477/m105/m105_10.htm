Distribution Analysis (continued) Customize results and run the analysis.

In this task, you add a table to the results and add a title for the report. Then you run the analysis.

  1. Click the Results tab. Notice that there are several tables selected by default. Click Add New Table. The Add/Remove Tables window opens.

  2. In the Tables list, select Moments and click the right arrow to move it to the Tables to include list. Click OK.


Click to view the full window.


  1. Click the Titles tab. In the Section box, select Analysis Titles. Delete the default title in the Text box and type Distribution and Normality Tests for Time.


Click to view the full window.


  1. Click Finish.
