Simple Linear Regression

Overview

A simple linear regression analysis enables you to estimate the relationship between a continuous response variable and a continuous predictor variable using a line.

In a simple linear regression, you assume that

  • the response and predictor variable are related linearly
  • the data points are independent and normally distributed for each value of the predictor variable
  • the variance about the fitted line is constant.

Suppose the research department in a concrete company is interested in whether humidity has any effect on the strength of concrete. The data is stored in the Concrete data set which has the following variables:


Variable Description
Strength1 measured strength of concrete by operator 1 (the primary operator on duty)
Strength2 measured strength of concrete by operator 2 (the trainee on duty)
Brand brand of concrete (Graystone, Consolidated, EZ Mix)
Additive presence of reinforcement material (reinforced, standard)
Humidity observed humidity


In this example, you use SAS Enterprise Guide to examine the assumptions of linear regression, and you use simple linear regression to examine the relationship between the two continuous variables.
