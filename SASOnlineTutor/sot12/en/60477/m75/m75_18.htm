Update and drop a PROC SQL view using the table Sasuser.Mechanics.

  1. To create the view Sasuser.Mechview, copy and paste the following PROC SQL step into the code editing window. Submit the step and view the messages in the SAS log. 
    proc sql;
   create view sasuser.mechview as 
      select id, lastname, firstname, 
             int((today()-hired)/365.25)
             as YearsEmployed, city
         from mechanics;
quit;
  2. Create a PROC SQL step that contains a DELETE statement to update the view Sasuser.Mechview, so that the updated view includes only the rows where City is equal to NEW YORK. Add a QUIT statement to the step. Submit the step and view the messages in the SAS log.

  3. Create a PROC SQL query that displays all columns from Sasuser.Mechview. Add a QUIT statement to the query. Submit the query and view both the output and the messages in the SAS log.

  4. Write and submit a PROC SQL step to drop Sasuser.Mechview. Be sure to add a QUIT statement to the step.
