Use an option to control how SAS handles missing values.

You must complete the previous practice before doing this practice.

  1. Submit the following code to create a transaction data set with a missing value: 
    data missing;
   input EmpID $ Salary;
   datalines;
   E00002 .
   E00005 25000
   ;
run;
  2. Write and submit a DATA step that uses the MODIFY statement with a BY statment to update Work.Empdata using the transaction data set Missing that you created in Step 1. Match observations by the EmpID variable. Set an option so that the missing values in the transaction data set replace values in the master data set.

  3. Write and submit a PROC PRINT step to view the first 5 observations of the updated data set.
