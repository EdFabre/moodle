Use the transaction data set Sasuser.Empdatu2 to modify the Work.Empdata data set.

Because this data set is used in other practices and lessons, you must copy it to the Work library so that the original data in the Sasuser library is not modified.

  1. Submit the following program to copy the Empdata SAS data set from the Sasuser library to the Work library and print the first five observations of the Empdata data set:
    proc copy in=sasuser out=work;
   select empdata;
run;
proc print data=empdata (obs=5);
run;
    Notice that the employee with employee ID E00002 is Eileen Bower and her salary is $27,000.

  2. Write and submit a program that uses the transaction data set Sasuser.Empdatu2 to modify the Empdata SAS data set. Use the index EmpID to locate observations. Modify the variables Salary, Location, and LastName. Empdatu2 has changes for the employee with employee ID E00002.
