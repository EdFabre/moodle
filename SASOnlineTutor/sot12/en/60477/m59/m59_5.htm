Update all the observations of the variable Salary in the SAS data set Work.Empdata.

Because this data set is used in other practices and lessons, you must copy it to the Work library so that the original data in the Sasuser library is not modified. If you accidentally modify the data in Sasuser, you must recreate the data.

  1. Submit the following PROC COPY program to copy the Empdata data set from the Sasuser library to the Work library: 
    proc copy in=sasuser out=work;
   select empdata;
run;
  2. Write a PROC PRINT step to print the first 5 observations of Work.Empdata.

  3. Write a DATA step that uses the MODIFY statement to give all the employees in the Work.Empdata SAS data set a 5% salary increase. You must modify the variable Salary.

  4. Write a second PROC PRINT step to print the first 5 observations of Work.Empdata.

  5. Submit the program and compare the output to verify changes in the Salary column.
