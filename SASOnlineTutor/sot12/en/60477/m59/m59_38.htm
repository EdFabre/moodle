Initiate an audit trail on the SAS data set Pilotemp.

Because this data set is used in other practices and lessons, you must copy it to the Work library so that the original data in the Sasuser library is not modified.

  1. Submit the following PROC COPY program to copy the Pilotemp data set from the Sasuser library to the Work library: 
    proc copy in=sasuser out=work;
   select pilotemp;
run;
  2. Write and submit a PROC DATASETS step to initiate an audit trail for the Pilotemp data set.

  3. Write and submit a DATA step that uses the MODIFY statement to give all the employees a 20% salary increase. Modify the variable salary.

  4. Write and submit a PROC PRINT step to view the audit trail.
