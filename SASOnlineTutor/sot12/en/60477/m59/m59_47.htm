Suspend, resume, and terminate the audit trail for Work.Pilotemp.

You must complete the previous practice before you do this one.

  1. Write and submit a PROC DATASETS step to temporarily stop auditing the Work.Pilotemp data set.

  2. Submit the following DATA step to modify Work.Pilotemp: 
    data pilotemp;
   modify pilotemp;
   location='Honolulu';
run;
  3. Write and submit a PROC DATASETS step to resume auditing the Work.Pilotemp data set.

  4. Submit the following DATA step to modify Work.Pilotemp. This is the same code you submitted in the last practice, except the value of Reason is different. 
    data pilotemp; 
   modify pilotemp promotion;
   by empid; 
   salary=salary+increase;
   user="My Name";
   reason="Year 2 Raise";
run;
  5. Write and submit a PROC PRINT step to view the audit trail.

  6. Write and submit a PROC DATASETS step to end the audit trail.
