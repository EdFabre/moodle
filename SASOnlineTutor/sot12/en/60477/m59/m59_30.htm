Add integrity constraints to the SAS data set Empdata.

You must complete this practice to do the next practice in this lesson.

Because this data set is used in other practices and lessons, you must copy it to the Work library so that the original data in the Sasuser library is not modified.

  1. Submit the following PROC COPY program to copy the Empdata data set from the Sasuser library to the Work library: 
    proc copy in=sasuser out=work;
   select empdata;
run;
  2. Write and submit a PROC DATASETS step to place integrity constraints on the SAS data set Work.Empdata.

    • Place a primary key constraint named PKEmpID on the variable EmpID and add a custom message.

    • Place a constraint named Lname so that no missing values are allowed for the LastName variable and add a custom message.

  3. View the log to verify that the integrity constraints were created.
