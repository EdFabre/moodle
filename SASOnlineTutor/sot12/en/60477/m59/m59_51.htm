Create generation data sets for the Jobhstry data set.

You must complete this practice to do the next practice in this lesson.

Because this data set is used in other practices and lessons, you must copy it to the Work library so that the original data in the Sasuser library is not modified.

  1. Submit the following PROC COPY program to copy the Jobhstry data set from the Sasuser library to the Work library: 
    proc copy in=sasuser out=work;
   select jobhstry;
run;
  2. Write and submit a PROC DATASETS step to modify the SAS data set Work.Jobhstry to save a maximum of three generations.

  3. Submit the following two DATA steps to concatenate the data sets Sasuser.Y200061 and Sasuser.Y200062 to Jobhstry. This creates two new generations of the data.

    data jobhstry;   
   set jobhstry sasuser.y200061;
run;
data jobhstry;   
   set jobhstry sasuser.y200062; 
run;
  4. Use PROC DATASETS to list the contents of the Work library.
