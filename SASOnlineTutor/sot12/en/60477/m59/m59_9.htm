Use the transaction data set Empdatu to modify the data set Empdata by employee ID.

You must complete this practice to do the next practice in this lesson.

Because this data set is used in other practices and lessons, you must copy it to the Work library so that the original data in the Sasuser library is not modified.

  1. Submit the following program to copy the Empdata data set from the Sasuser library to the Work library and print the first 15 observations of the Empdata data set:
    proc copy in=sasuser out=work;
   select empdata;
run;
proc print data=empdata (obs=15);
run;
    Notice that the employee with employee ID E00014 is a MECH02 and has a salary of $19,000.

  2. Write a DATA step that uses the MODIFY statement with a BY statement to modify Work.Empdata using the transaction data set Sasuser.Empdatu. Locate observations with the EmpID variable. The Empdatu data set has changes for employee ID E00014.

  3. Write a PROC PRINT step to print the first 15 observations of Work.Empdata.

  4. Submit the program and notice the changes for employee ID E00014.
