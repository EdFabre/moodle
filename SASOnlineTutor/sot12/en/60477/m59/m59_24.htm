Use _IORC_ and %SYSRC to control the update process.

Because this data set is used in other practices and lessons, you must copy it to the Work library so that the original data in the Sasuser library is not modified.

  1. Submit the following program to copy the Empdata SAS data set from the Sasuser library to the Work library and print the first five observations of the Empdata data set:
    proc copy in=sasuser out=work;
   select empdata;
run;
  2. Submit the following program to create a transaction data set for this practice: 
    data sasuser.newsalary;
   input EmpId $ Increase;
   datalines;
   E00001 3250
   E00002 3500
   E00300 6250
   ;
run;
  3. Write and submit a program that uses the transaction data set Sasuser.Newsalary to modify the master data set Work.Empdata. Use the index EmpID to locate observations. For matching values in the master data set, add Salary and Increase and replace the value in the master data set. For any other condition, use the PUT statement to write ***Incorrect Employee ID*** to the log.

  4. Print the first 5 observations of Work.Empdata and view the results.

  5. View the log.
