Create a customized audit trail for SAS data set Pilotemp.

You must complete this practice to do the next practice in this lesson.

  1. Submit the following PROC COPY program to copy the Pilotemp data set from the Sasuser library to the Work library: 
    proc copy in=sasuser out=work;
   select pilotemp;
run;
  2. Submit the following DATA step to create a transaction data set to use when you modify Work.Pilotemp: 
    data promotion;
   length EmpID $ 6;
   input EmpID $ JobCode $ Increase;
   datalines;
   E00031 Pilot4 2400
   E00055 Pilot4 5000
   ;
run;
  3. Write and submit a PROC DATASETS step to initiate an audit trail for the Pilotemp data set. This time, specify

    • a user variable named User to track who edited the data set. Specify a label for the variable.


    • a user variable named Reason to track the reason the file was edited. Specify a label for the variable.


    • a LOG statement such that the audit trail file stores only the record after the update.

  4. Write and submit a DATA step that uses the MODIFY statement to update the the data set Work.Pilotemp by EmpID. For each match, add increase to salary and assign values to the user variables User and Reason.

  5. Write and submit a PROC PRINT step to view the audit trail.
