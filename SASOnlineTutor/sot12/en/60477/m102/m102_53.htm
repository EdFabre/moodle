Using Styles to Customize HTML Reports (continued) Modify images in a style.
  1. To add your logo, you need to specify a new banner image. In the Edit Style window, click the Images tab. Notice that for this style, Use banner image (can be a URL) is selected. To specify the existing image, a path similar to this is shown:

    C:\Program Files\SAS\Enterprise Guide\Styles\nls\Eng\startbanner.gif

  2. You want to replace the banner image with the Specialty Foods Board Meeting Reports image that follows. However, first you need to save the image on your computer. Right-click the image and select Save Picture As. For the image path, specify the same path as the existing image, but with the filename board.gif.


  1. Then, in the Edit Style window, change the path for Use banner image (can be a URL) to the path with the new image name. For example, for the path that is shown above, you would specify the following:

    C:\Program Files\SAS\Enterprise Guide\Styles\nls\Eng\board.gif

Click to view the full window.

Next, you change the background colors for column headings, row column headings, and data cells.



  1. To add your logo, you need to specify a new banner image. In the Style Editor window, click the Images tab. Notice that for this style, Use a banner image (can be a URL) is selected. Currently, startbanner.gif is the existing image.

  2. You want to replace the banner image with the Specialty Foods Board Meeting Reports image that follows. However, first you need to save the image on your computer. Right-click the image and select Save Picture As. Select a location to save the image. You can save the image in the same location as the startbanner.gif image (C:\Program Files\SAS\Enterprise Guide 3.0\Styles). Name the image board.gif.


  1. Then, in the Style Editor window, click the Select button and navigate to the location where you saved board.gif. Or, if you used the path shown above, you would specify the following:

    C:\Program Files\SAS\Enterprise Guide 3.0\Styles\board.gif

Click to view the full window.

Next, you change the background colors for column headings, row column headings, and data cells.
