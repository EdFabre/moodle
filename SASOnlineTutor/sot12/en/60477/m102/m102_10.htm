Setting Options for Results (continued) Specify options for footnotes and titles.
  1. Select ToolsOptions.


  2. In the Options window, click the Tasks tab.

  3. In the Default footnote text for task output box, type (or copy and paste) the following:

    Report created on %SYSFUNC(DATE(), EURDFDE9.)

Click to view the full window.

  1. In the SAS procedure settings area, clear the Include SAS procedure titles in results check box.

Click to view the full window.

  1. Click OK to set the options that you have selected and to close the Options window.
