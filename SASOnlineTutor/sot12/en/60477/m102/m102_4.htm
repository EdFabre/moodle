Your Reports and Data

Now let's get started with your reports. For your first report, the board wants to know which product suppliers and categories generate the highest sales. This is the list report that you'll create:





Later you'll also create and customize the following reports:

  • a frequency report
  • a summary table
  • a bar chart.

To create your reports, you'll use the ProductSales data set. This table was created from a query that joins product, supplier, and order data. The ProductSales data has the following variables:


Variable
 Type Description
CategoryName char Category
CompanyNameLong char Supplier Company Name
Country char Supplier location
ItemProfit num Profit per item
OrderProfit num Total profit per order
ProductName char Product name
RetailOrders num

Quantity sold
UnitCost num Unit wholesale cost
UnitPrice num Unit retail price
UnitsInStock num Units in stock
UnitsOnOrder num Units on order


Next, let's add the ProductSales data to your SAS Enterprise Guide project and view the data.
