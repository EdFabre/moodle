Creating and Applying Custom Formats

Another way to enhance your reports is by using formats. SAS Enterprise Guide uses SAS formats to write variable values in predefined ways. For example, the numeric format DOLLAR10.2 writes the value 35701 as $35,701.00. There are also predefined formats for character variables.

At some point, you might even want to create your own formats, especially if you have data that includes codes for variable values. For instance, you can do the following:

  • convert a number to a character string (for example, write the data value 1 as YES)
  • convert one character string to another character string (for example, write the data value F as Female).

An alternative to creating a format is to replace values in a column by using a query. The advantage of creating a custom format is that you can apply the format to other data sets.

User-defined formats always associate variable values with character values. If you attempt to perform arithmetic operations on user-formatted numeric values, the unformatted values are used.


Example: Formatting Countries as Domestic or International

For your specialty foods shop, you have created a bar chart that shows the amount of profit for each product category. Suppose that you want to see the amount of profit that comes from international and domestic suppliers in each product category. In your data, you have the Country variable that lists the supplier country. You can generate this bar chart by creating a format and applying it to the Country variable in the Bar Chart task.
