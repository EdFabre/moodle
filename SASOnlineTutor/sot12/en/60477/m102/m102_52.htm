Using Styles to Customize HTML Reports (continued) Modify borders in a style.
  1. To give the table a more three-dimensional appearance, you want to change its border edge color. In the Edit Style window, click the Borders tab.

  2. In the Active element drop-down list, select Table. Notice that you can either select items from this list or click them in the preview area.

  3. Under Show border edge attributes for, leave the default All selected.

  4. Click the drop-down arrow for Color and click the light gray box (directly under the black box) in the palette.
