Create and apply a style.

Edit the sasweb style and create a new style named Airlines, with the following features:

  • Report title should use this font: Arial Bold 14.

  • Background color of the column and row headings should be green
    (Red: 64; Green:128; Blue: 128).

  • The border around the outside of the table should be thin.

Then, apply the style to the Summary Table for the Marchflights data.



Click to view the report


  1. Select ToolsStyle Manager.

  2. In the Available styles list, select sasweb. Click Edit. In the Edit Style window, click Save As. Type Airlines for the style name. Click OK.

  3. In the preview area, click SAS System Title. Change the font to Arial Bold 14.

  4. In the preview area, click Column 1 in the table. Click the Background arrow to open the color palette. Click Custom Colors. Enter these values: Red: 64; Green: 128; Blue: 128. Click OK.

  5. In the preview area, click Row 1. Click the Background arrow and select the custom color that you just added from the palette.

  6. Click the Borders tab. For Line style, select None. For Line width, select Thin.

  7. Click OK in the Edit Styles window. When you are prompted to save changes, click Yes. Click OK in the Style Manager window.

  8. In the Project window, double-click HTML under Summary Tables for the Marchflights data. The Summary Tables HTML report opens.

  9. Click the styles drop-down list in the SAS Enterprise Guide main menu bar and select Airlines. The style is applied to the report.

Click to view the report


  1. Select ToolsStyle Manager.

  2. In the Styles List, select sasweb. Click Edit.

  3. In the preview area, click SAS System Title. Change the font to Arial Bold 14.

  4. In the preview area, click Column 1 in the table. Click the Background color drop-down list to open the color palette. Click Custom Colors. Enter these values: Red: 64; Green: 128; Blue: 128. Click OK.

  5. In the preview area, click Row 1. Click the Background color drop-down list and select the custom color that you just added from the palette.

  6. Click the Borders tab. For Line style, select None. For Line width, select Thin.

  7. Click OK in the Style Editor window. Select sasweb located in Built-In Style and click Set as Default. Click OK in the Style Manager window.

  8. In the Project window, double-click HTML - Summary Tables under Summary Tables for the Marchflights data. The Summary Tables HTML report opens.

  9. Click the styles drop-down list in the SAS Enterprise Guide main menu bar and select sasweb under My Styles. The style is applied to the report.
