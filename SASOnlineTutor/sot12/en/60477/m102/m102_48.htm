Create a combined HTML report.

Use the Document Builder to combine the bar chart and summary table that were created from the Marchflights data into one HTML report named MarchReport.


Click to view the report

  1. Select ToolsDocument Builder.

  2. In the Document title box, type MarchReport.

  3. Click Add Results. Press CTRL and select Empty Seats by Destination under Bar Chart for marchflights and select Table 1 under Summary Tables for marchflights. Click OK.

  4. In the Document Builder window, select the Bar Chart and click Move Up.

  5. Click Preview to view the report.

  6. Click OK to close the Document Builder.

Use the Document Builder to combine the bar chart and summary table that were created from the Marchflights data into one HTML report named MarchReport.


Click to view the report

  1. Select ToolsDocument Builder.

  2. In the Document name box, type MarchReport.

  3. Click the Add button. Use the Preview button to find the Gchart Procedure titled Empty Seats by Destination under the Bar Chart task. Click OK.

  4. Click the Add button again. Use the Preview button to find the Tabulate Procedure titled Summary Data for March under the Summary Tables task. Click OK.

  5. Click Preview to view the report.

  6. Click OK to close the Document Builder.
