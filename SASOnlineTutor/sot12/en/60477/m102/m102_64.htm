Using Traffic Lighting in a Summary Table

You've learned how to create a format and apply it to the values in a variable. One way that you can use a format in a summary table is to apply a style to a table cell based on the value of the data in that cell. This is called traffic lighting. Traffic lighting is a visual cue that often uses the colors of a traffic light: red, yellow, and green. When you use traffic lighting, you can see at a glance which values are within specifically defined ranges.

Suppose you want a summary table that shows the total profit that is generated by each product in your specialty foods shop. Over the years you have determined the following:

  • Products that generate less than $200 profit in a quarter might need to be discontinued.
  • Products that generate between $200 and $600 per quarter might need extra promotion to boost sales.
  • Products that generate over $600 per quarter are selling well.

You can use traffic lighting in your report to change the background color of table cells to indicate the sales performance for each product.





To add traffic lighting to table cells in a summary table, you do the following:

  • create a custom format
  • create the summary table that you want
  • modify the summary table code to add the format.

You have already learned how to create a custom format and a summary table. Before we begin this process, let's learn a little more about modifying code in SAS Enterprise Guide.
