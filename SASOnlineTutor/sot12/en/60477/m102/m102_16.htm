Creating List Reports (continued) Modify the report to show profit by supplier company.
  1. In the Project window, right-click the List Data task and select
    2. Copy. Then right-click again and select Paste.

  2. Double-click the copy of the List Data task to open it.

  3. Click the Columns tab. In the List data roles box, remove CategoryName from the Group table by role and from the Identifying label role. To remove a variable from a role you can either drag and drop the variable back to the Variables to assign list, or you can right-click the variable in the role and select Remove from role.

  4. Assign CompanyNameLong to the Group table by role and also to the Identifying label role.

Click to view the full window.

  1. Right-click CompanyNameLong and select Properties. Change the label to Company. Then click OK.

Click to view the full window.

  1. Click the Titles tab.

  2. In the Text box, change the title to Product Sales and Profit by Company.

Click to view the full window.

Next, you run the task and view the results.
