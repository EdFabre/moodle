Combining Reports (continued) Create an HTML report by using the Document Builder.
  1. Select ToolsDocument Builder.

  2. In the Document title box, type BoardReportWeb.
    3. This is the name of the document definition you are creating.

  3. Click Document Options. Notice that you can specify a style for this report. You can select the style of your choice or leave the sasweb style selected. Click OK.

  4. Click Add Results. Sometimes it can be difficult to determine what results to select from the listed items, particularly if you have more than one of each type of task. Click Table 1 under one of the two Summary Tables tasks. Click Preview to view the table that you have selected. Find the table that summarizes the data by category. When you have the correct summary table, click OK.

Click to view the full window.

  1. Click Add Results again. Select Total Profit by Category under the Bar Chart task. Click OK.

  2. In the Document Builder window, select the Bar Chart item and click Move Up.

Click to view the full window.

  1. Click Preview to see the combined report.

Click to view the full window.

  1. In the Document Builder, click OK. An item for the document is added to the project.

  2. In the Project window, right-click BoardReportWeb and select Save BoardReportWeb As/Export.

  3. Navigate to the location where you want to save the HTML file and click Save. You can edit and customize the HTML file that you exported.
