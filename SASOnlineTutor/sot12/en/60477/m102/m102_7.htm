Setting Options for Results (continued)

Selecting a Default Style for HTML Results

Let's assume that you want to present your reports to the board in a Web browser.

If you specify HTML as a result format, you can also use styles to modify the appearance of your reports. SAS Enterprise Guide uses cascading style sheets (CSS) to change the style of HTML results. You can apply many built-in styles to change the colors, fonts, and other aspects of your results. In addition, you can create your own styles or import styles that you have already created. You will learn more about creating styles later in this lesson.

You can select a default style for HTML results by using the Report style drop-down list in the Options window.




You can also set the default style by using the Style Manager window. This method enables you to preview the style before you select it.

The Results tab in the Options window also has options for creating graphs. You will learn more about these options later in the lesson.

Next, you view result formats in the Options window, and you select a default style in the Style Manager window.
