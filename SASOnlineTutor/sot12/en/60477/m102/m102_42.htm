Create a graph.

Use the Marchflights data to create a bar chart that shows the total number of empty seats for each destination. Arrange the bars in ascending order and display the sum beside each bar. Use a custom title.


Click to view the report

  1. In the Project window, select Marchflights to make it the active data. Select GraphBar.

  2. On the Chart Gallery tab, select Horizontal Colored Bars.

  3. Click the Columns tab. Assign Destination to the Column to chart role. Assign EmptySeats to the Sum by role.

  4. Click the Appearance tab. Click the Layout button. Click the Order drop-down list and select Ascending.

  5. Click the Titles button. In the Title box, type Empty Seats by Destination.

  6. Click the Advanced tab. Select the Additional statistical value to show next to bar check box, then click the drop-down box just below and select Sum.

  7. Click Finish to create the bar chart.

Use the Marchflights data to create a bar chart that shows the total number of empty seats for each destination. Arrange the bars in ascending order and display the sum beside each bar. Use a custom title.


Click to view the report


  1. In the Project window, select Marchflights to make it the active data. Select GraphBar Chart.

  2. Click Bar Chart in the selection pane and select Horizontal Color Bars.

  3. Click Task Roles in the selection pane. Assign Destination to the Column to chart role. Assign EmptySeats to the Sum of role.

  4. Click Layout in the selection pane. Click the Order drop-down list and select Ascending.

  5. Click Titles in the selection pane. Clear the Use default title check box and delete the default title. Type Empty Seats by Destination.

  6. Click Advanced in the selection pane. Select the Show statistics next to bar check box. Then, select the Additional statistical value to show next to bar check box, then click the drop-down box just below and select Sum.

  7. Click Run to create the bar chart.
