Combining Reports (continued) Create a Microsoft Word document to combine results.
  1. Using Windows Explorer or My Computer, browse to the location where you saved BoardReport.rtf. Double-click the file to open it in Microsoft Word.

  2. Go back to the SAS Enterprise Guide Project window. Under the Bar chart task for ProductSales, expand Results and double-click HTML to open the bar chart. Right-click in the bar chart and select Copy from the pop-up menu.

  3. Go back to your BoardReport.rtf document in Microsoft Word and select EditPaste. The bar chart is added to the document.

  4. Now that you have the two reports that you want in your document, you can edit the format, add text, and so on to get just the report that you want.

  5. Save the Microsoft Word document as BoardReport.doc.
