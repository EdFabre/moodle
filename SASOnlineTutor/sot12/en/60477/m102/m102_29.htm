Creating Summary Reports (continued) Format the data values.
  1. In the Preview area, right-click Sum for the RetailOrders column and select Data Value Properties.


  2. In the Properties window, click the Format tab.

  3. By default, the values are formatted to include two decimal places. Because this is a sum of the number of orders that were placed, there is no need to include decimal places. In the Categories box, select Numeric. In the Formats box, select BESTw.d.

  4. In the Attributes area, change the value in the Decimal places box to 0.
