Using Traffic Lighting in a Summary Table (continued) Modify the SAS program to apply the format.
  1. In the Project window, right-click the Summary Tables task that you just ran and select Add as Code Template.

  2. A code item named Code template from Summary Tables is added to the project, and the code opens in a code editor window.


  3. Examine the code. You are going to add traffic lighting to the OrderProfit sum column. The Sum column currently has the DOLLAR10. format applied to the data. Find the following line in the code:

    Sum*F=DOLLAR10.

  4. Copy and paste the following code after the format:

    *{style={background=flag.}}

    The code should now look like the following:
