Preparing Data for Reports (continued) Add data to your project and view the ProductSales data.
 
  1. Start SAS Enterprise Guide and create a new project if you have not already done so.

  2. Select InsertData. Click the Existing tab. Navigate to the location where you saved the ProductSales and Marchflights data that you downloaded for this lesson.

  3. Press Ctrl and select ProductSales and Marchflights. Then click OK to add the data to your project. The data sets open automatically in SAS Enterprise Guide. You can close the Marchflights data set for now.




  1. Scroll through the ProductSales data. Notice that the data has one or more rows (or observations) for each product. There is a row for each order that was placed during the first quarter of 2004. As you might guess, you will probably want to summarize this data in your reports.

  2. Close the ProductSales data.
