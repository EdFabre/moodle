Combining Reports

So far in this lesson, you've created several different HTML reports for your specialty foods shop. Suppose that you want to combine some of these individual reports to produce the following combined reports:

  • a printed document to give to each board member in the meeting
  • an HTML report to put on the company's intranet.


Combining Results in a Document

To create a document, you generate RTF output for the individual reports that you want to combine and then open them in a word processing program such as Microsoft Word. In addition, you can copy and paste ActiveX object graphs directly into Microsoft Word.

You can use any word processing program that will open a document that is saved as a Rich Text Format (RTF) file. This topic uses Microsoft Word as an example.

By default, the result format is set to HTML. The Properties window for each task enables you to override the default settings and select result formats, an image format, or a new style to apply to the results for the task individual task.
