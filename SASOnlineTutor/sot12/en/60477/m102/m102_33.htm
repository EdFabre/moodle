Create a summary report.

Use the Marchflights data to create a summary report that displays the following information:

  • summary statistics for each destination
  • a total (sum) for the variables Boarded, Transferred, Nonrevenue, and EmptySeats, formatted with the BESTw.d format
  • a mean for the variable PercentFilled
  • a total across rows
  • the company name, Sky Airlines, in the box area with a black background
  • the column heading # of Flights for the N variable
  • a custom title
  • no display class level headings.

Click to view the report

  1. In the Project window, select Marchflights to make it the active data. Select AnalysisDescriptiveSummary Tables.

  2. On the Columns tab, assign Boarded, Transferred, Nonrevenue, EmptySeats, and PercentFilled to the Analysis variables role. Assign Destination to the Classification variables role.

  3. Click the Table tab. Drag and drop Destination onto the row box in the Preview area. Drag and drop ALL to the bottom edge of the Destination box. This creates a row under Destination.

  4. In the Available variables list, press CTRL and select (in order) Boarded, Transferred, Nonrevenue, EmptySeats, and PercentFilled. Drag and drop these variables to the right of the N column.

  5. In the Available statistics list, select Mean. Drag it to the Sum box below PercentFilled and drop it when Sum is highlighted. This replaces the Sum statistic with Mean.

  6. In the Preview area, right-click Box Area and select Box Area Properties. In the Use the following text box, type Sky Airlines. Click the Font tab. Click the arrow for Background color and select the black color square. Click OK.

  7. Right-click Box Area and select Table Properties. Click the General tab. Select the Remove all class headings from the row dimension check box. Click OK.

  8. Right-click the N column heading and select Heading Properties. In the Label box, type # of Flights. Click OK.

  9. Right-click the Boarded column heading and select Data Value Properties. Click the Format tab. In the Categories list, select Numeric. In the Formats list, select BESTw.d. Click OK.

  10. Use the same procedure to apply the BESTw.d format to the data values in the Transferred, Nonrevenue, and EmptySeats columns.

  11. Click the Titles tab. In the Section box, click Table Titles. In the Text box, delete the default title and type Summary Data for March.

  12. Click Finish to create the report.
