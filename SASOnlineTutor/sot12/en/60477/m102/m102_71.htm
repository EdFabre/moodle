Now that you've completed all the reports you need for the board meeting, let's look at another customization that you can learn about on your own: custom style attributes.


Custom Style Attributes

When you create or edit a style, you can create and edit your own custom style attributes using the Custom tab in the Edit Styles window.

Because CSS standards are evolving, you can enter any value for a custom style attribute. This gives you the flexibility to add the newest style properties as they become available.



Click to view the full window.


For more information about creating custom style attributes, select HelpEnterprise Guide Help in SAS Enterprise Guide. For more information about the current status of CSS style properties and which browsers support them, see http://www.w3.org/.
