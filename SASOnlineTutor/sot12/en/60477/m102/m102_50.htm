Using Styles to Customize HTML Reports (continued) Create a new style from an existing style.
  1. Select ToolsStyle Manager.

  2. In the Available styles list, select EGDefault. Click Edit. The Edit Style window opens.

  3. It's a good idea to save the style under another name before you edit it, so click Save As.

  4. In the Save Style window, type MyEditedStyle and click OK. In the Edit Style window, the new name appears in the Styles drop-down list at the bottom left of the window.

Now you're ready to edit the MyEditedStyle style. The features that you want to modify are highlighted in the following illustration.

Let's see how you modify the new style.

You might find that some of the style settings that you select do not appear to work as you would expect. Although most of the style options in the W3C's CSS1 specification are included in the style editor, not all browsers support those features in the same way. In addition, some styles are hardcoded in the results and cannot be changed using style sheets.
