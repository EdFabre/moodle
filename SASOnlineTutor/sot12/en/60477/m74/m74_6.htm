The first column has the column name EmpID and the label Supervisor ID.

Which of the following statements is true?

You can create only one index on this, or any other, table.

To run the following PROC SQL query more efficiently, you should create a composite index on all three columns in the table:

     proc sql;
        select *
        from sasuser.supervisors;
If you need to join Sasuser.Supervisors with another table, and you are combining rows by using the equals (=) operator to match the values of EmpID, you can improve the performance of the join by creating an index on EmpID.
Whether you create a simple or composite index, you cannot specify the name of any column in the table as the index name.
