Create an index on the temporary table Work.Staffmaster (which is a copy of Sasuser.Staffmaster) and display index specifications.

Be sure to complete this practice. You'll use the temporary table Work.Staffmaster and the index LastName in later practices.

  1. Write a PROC SQL step to create the temporary table Work.Staffmaster by copying the table (all columns and rows) from the table Sasuser.Staffmaster. Add a QUIT statement to the step. Submit the step and view the SAS log to verify that the table was created.

  2. Write a PROC SQL step to create a simple, unique index on the column LastName in Work.Staffmaster. Add a QUIT statement to the step. Submit the step and view the SAS log to see whether the index was created. (Are there duplicate values for LastName in Work.Staffmaster?)

  3. Write a PROC SQL step to create a simple non-unique index on the column LastName in Work.Staffmaster. Add a QUIT statement to the step. Submit the step and view the SAS log to see whether the index was created.

  4. Write a PROC SQL step to display index specifications (and other information) for the table Work.Staffmaster in the SAS log. Add a QUIT statement to the step. Submit the step and view the SAS log to see the CREATE INDEX statement for the index LastName.
