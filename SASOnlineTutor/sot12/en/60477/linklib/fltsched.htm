SAS Data Set Flightschedule

The Flightschedule data set contains information about flights that have been scheduled.


Data Set Description
Variable Type Length Description
Date num 8 date of flight
Destination char 3 flight destination
FlightNumber char 3 flight number
EmpID char 4 employee ID
