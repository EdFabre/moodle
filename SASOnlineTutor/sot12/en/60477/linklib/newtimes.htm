SAS Data Set Newtimes

The Newtimes data set contains schedule delay information for flights of an airline company.

Variable Label Type Length Description
Flight Flight Number char 7 flight ID number
Date Scheduled Date of Flight num 8 date of flight
TimeDiff   num 8 delay in minutes
