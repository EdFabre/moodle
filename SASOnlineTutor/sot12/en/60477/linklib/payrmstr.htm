SAS Data Set Payrollmaster

The Payrollmaster data set contains payroll information for all employees of an airline.


Data Set Description
Variable Type Length Description
DateOfBirth num 8 date of birth
DateOfHire num 8 date of hire
EmpID char 4 employee ID number
Gender char 1 gender
JobCode char 3 job code
Salary num 8 salary
