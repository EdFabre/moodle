SAS Data Set Sale2000

The Sale2000 data set contains information on the capacity, the load, and the revenue for various flights of an airline company.

Data Set Description

Variable Type Length Description
FlightID char 7 Flight Number
RouteID char 7 Route Number
Origin char 3 Airport Code of Flight Origin
Dest char 3 Airport Code of Flight Destination
Cap1st num 8 1st Class Capacity
CapBusiness num 8 Business Class Capacity
CapEcon num 8 Economy Class Capacity
CapTotal num 8 Total Passenger Capacity
CapCargo num 8 Cargo Weight Capacity
Date num 8 Scheduled Date of Flight
Psgr1st num 8 Number of 1st Class Passengers
PsgrBusiness num 8 Number of Business Class Passengers
PsgrEcon num 8 Number of Economy Class Passengers
Rev1st num 8 Revenue from 1st Class
RevBusiness num 8 Revenue from Business Class
RevEcon num 8 Revenue from Economy Class
SaleMon char 7 Month of Sale
CargoWgt num 8 Weight of Cargo
RevCargo num 8 Revenue from Cargo
