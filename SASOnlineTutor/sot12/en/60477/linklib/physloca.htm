Determining the Location of Your Sample Raw Data Files

Copy and paste the following SAS program into the code editing window. Then submit the program. A message regarding the location of your sample raw data files will appear in the Log window.

     options nosource nonotes;
     data _null_;
        location=pathname("sasuser");
        put / 'The location of your raw data files is:'
            / location;
     run;
     options source notes;
