Setting Up Practice Data in Enterprise Guide 2.0

  1. If you have not already done so, start Enterprise Guide or the SAS Learning Edition.

  2. In the Welcome to Enterprise Guide window, select Create New Project.

  3. Select InsertCode from the Enterprise Guide menu bar.

  4. In the Insert Code window, click OK to accept the default name and open the Code Editor window.

  5. Copy and paste this SAS program into the Code Editor.

  6. Select CodeRun on Local. The code creates data sets that you will use in practices. When SAS finishes running the code, Enterprise Guide adds shortcuts to the data in the Project window. Also, one data set will open in the data grid.



  7. Because you won't need to use these shortcuts, you can delete the Code1 item (and all the shortcuts) from the Project. This will not delete the data you created. To delete the item from the project, right-click Code1 and select Delete from the shortcut menu.

  8. In the Confirmation window, click Yes. Congratulations. . . you have created the data you need for practices!
