SAS Data Set Jobhstry

The Jobhstry data set contains job history information for employees of an airline.


Data Set Description
Variable Type Length Description
Job1 char 6 employee's first job code
Job2 char 6 employee's second job code
Job3 char 6 employee's third job code
LastName char 25 employee's last name
