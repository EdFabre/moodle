SAS Data Set Payrollchanges

The Payrollchanges data set contains payroll information for employees of an airline who have had payroll changes.


Data Set Description
Variable Type Length Description
DateOfBirth num 8 date of birth
DateOfHire num 8 date of hire
EmpID char 4 employee identification number
Gender char 1 gender
JobCode char 3 job code
Salary num 8 salary
