SAS Data Set Insure

The Insure data set contains  information about patients' insurance at a hypothetical wellness clinic.


Data Set Description

Variable Type Length Description
ID num 8 client ID number
Name char 20 client name
Policy num 5 client policy number
Company char 11 name of company
PctInsured num 3 percent insured
Total num 8 total balance
BalanceDue num 8 balance due
