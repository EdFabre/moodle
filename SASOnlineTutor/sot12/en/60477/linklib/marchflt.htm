SAS Data Set Marchflights

The Marchflights data set contains information about airline flights that are scheduled for the month of March.


Data Set Description
Variable Type Length Description
Date num 8 date of the flight
DepartureTime num 8 time of departure
FlightNumber char 3 flight number
Origin char 3 place of origin
Destination char 3 destination
Distance num 8 distance in miles
Mail num 8 amount of mail on board
Freight num 8 amount of freight on board
Boarded num 8 number of passengers that boarded
Transferred num 8 number of passengers that transferred
NonRevenue num 8 number of nonrevenue passengers
Deplaned num 8 number of passengers that deplaned
PassengerCapacity num 8 passenger capacity
