SAS Data Set Staffchanges

The Staffchanges data set contains identification and contact information about all employees who have had a change in salary or job code.


Data Set Description
Variable Type Length Description
EmpID char 4 employee ID number
LastName char 15 last name
FirstName char 15 first name
City char 15 city
State char 2 state abbreviation
PhoneNumber char 12 phone number
