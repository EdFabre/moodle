SAS Data Set Stress98

The Stress98 data set contains readings from exercise stress tests performed on patients at a wellness clinic in 1998.


Data Set Description

Variable Type Length Description
ID char 4 patient ID number
Name char 20 patient name
RestHR num 8 resting heart rate
MaxHR num 8 maximum heart rate during test
RecHR num 8 recovery heart rate after test
TimeMin num 8 time, complete minutes
TimeSec num 8 time, seconds
Tolerance char 1 comparison of stress test tolerance
between this test and the last test
(I=increased, D=decreased, S=same,
N=no previous test)
Year num 8 year that test was performed
