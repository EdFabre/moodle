SAS Data Set Salcomps

The Salcomps data set contains employee information, including salary, of an airline company.

Data Set Description

Variable Type Length Description
EmpID char 6 employee ID
LastName char 32 employee's last name
Phone char 8 employee's phone number
Location char 25 city where employee is located
Division char 30 division where employee works
HireDate num 8 date of hire
JobCode char 6 abbreviated job description
Salary num 8 employee's salary
