Submitting ODS Statements in SAS Enterprise Guide

By default, SAS Enterprise Guide wraps all the code you submit to SAS with ODS statements to create HTML formatted output. Before you submit your own ODS statements, you must turn off this default behavior. In addition, Text output must be selected or the Listing destination is not available.

Do the following to set up SAS Enterprise Guide so that you can learn to use ODS statements:

  1. Select Tools Options.

  2. In the Options window, click the Code tab.

  3. In the Code area, clear the Automatically Generate SAS Code check box.

  4. Click the Results tab.

  5. In the Result Formats area, select the HTML and Text output check boxes.

  6. Click OK to close the Options window and apply the changes.
Once you have finished submitting your own ODS statements, you must reset the option to automatically generate SAS code. If you do not reset this option, SAS Enterprise Guide will not produce HTML output.
