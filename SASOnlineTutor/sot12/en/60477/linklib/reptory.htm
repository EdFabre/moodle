SAS Data Set Repertory

The Repertory data set contains information about performances by a small repertory theater company.


Data Set Description

Variable Type Length Description
Play char 23 name of a play in the repertory
Role char 24 name of a character in Play
SSN char 11 social security number of player pl
Date num 8 date of performance of Play
