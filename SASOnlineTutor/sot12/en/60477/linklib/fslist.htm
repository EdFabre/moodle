Viewing Raw Data Files Using PROC FSLIST

The FSLIST procedure enables you to browse external files that are not SAS data sets from within a SAS session. Because the files are displayed in an interactive window, the procedure provides a convenient method for examining the structure of a raw data file.


General form, PROC FSLIST:
PROC FSLIST FILE=file-specification;
RUN;

where

file-specification is either 'external file' or fileref:

  • 'external file' is the complete operating environment file specification for the external file. You must enclose the name in quotation marks.

  • fileref is a fileref that has been previously assigned to the external file.



Example

The following program displays the raw data file Month1.dat, which is stored in the C:\data directory in the Windows environment. PROC FSLIST displays the raw data file in browse mode.

     proc fslist file='c:\data\month1.dat';
     run;


Several commands make it easier to obtain the file layout for the fields in the raw data file.

  • To display column numbers in the FSLIST window, enter the COLS command in the command bar or on a command line.

  • To display line numbers in the FSLIST window, enter the NUMS command in the command bar or on a command line.




You can also browse a raw data file by using a fileref for the file specification in a PROC FSLIST statement. The following program displays the raw data file referenced by the fileref Month2:

     proc fslist file=month2;
     run;

You can also use the FSLIST command to browse the contents of a raw data file. To use the FSLIST command, enter the keyword FSLIST, followed by a file specification or a fileref, in the command bar or on a command line. For example, you enter the following code to browse the contents of the raw data file that is referenced by the fileref Month2:
     fslist month2
