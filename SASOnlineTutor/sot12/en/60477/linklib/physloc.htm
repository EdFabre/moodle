Determining the Location of Your Sample Raw Data Files

There are two ways that you can determine the location of your sample raw data files. You can

  • submit a program
  • use the SAS Explorer window.
If you are using z/OS (OS/390), do not use either of the methods above. In the z/OS operating environment, your sample raw data files are stored as userid.filename.dat. For example, if your user ID is myname, the file route1.dat is stored as myname.route1.dat.


Submit a Program to Find Your Raw Data Files

Copy the program shown below and paste it into a code editing window: 

     options nosource;
     data _null_;
        rawdatalocation=trim(pathname("sasuser"));
        put / "Your raw data files are located here:     " 
              rawdatalocation; 
     run;
     options source;
Submit the program. The location of your raw data files appears in the SAS log.


Use the SAS Explorer Window

  1. Locate Sasuser in the SAS Explorer window.

  2. Right-click the Sasuser icon and select Properties.



  3. The physical location appears next to the directory icon in the tree view.
