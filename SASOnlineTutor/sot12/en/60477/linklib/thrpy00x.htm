SAS Data Set Therapy2000

The Therapy2000 data set contains exercise therapy information for patients in a hypothetical wellness clinic for a twelve-month period.


Data Set Description

Variable Type Length Description
Month num 2 month in which data is collected
Year num 8 year in which data is collected
AerClass num 8 number of patients in aerobics class
WalkJogRun num 8 number of patients who walk/jog/run
Swim num 8 number of patients who swim
