SAS Data Set Cargo99

The Cargo99 data set contains information about cargo weights and revenue for airline flights.


Data Set Description
Variable Type Length Description
FlightID char 7 Flight Number
RouteID char 7 Route Number
Origin char 3 Start Point
Dest char 3 Destination
CapCargo num 8 Aircraft Capacity - Total
Date num 8 Scheduled Date of Flight
CargoWgt num 8 Weight of Cargo in Pounds
CargoRev num 8 Revenue from Cargo
