SAS Data View Sashelp.Vstabvw

The Sashelp.Vstabvw data view contains information about all objects that are in currently defined SAS data libraries.

Data View Description

Variable Type Length Description
libname char 8 Library Name
memname char 32 Member Name
memtype char 8 Member Type
