Combine the first five observations in the Sasuser.Empdata data set with a list of lookup values that are not stored in a data set.

  1. Write a program that uses the IF-THEN/ELSE statement to combine the employee birthdates that are recorded in the list below with the first five observations of the Sasuser.Empdata data set. Create a temporary data set named Empdata_new to contain the combined observations. In Empdata_new, record the lookup values in a character variable named Birthdate. Submit the program.

    In the list below, employees' birthdates are listed according to their identification numbers.

    Employee Birthdates
    E0001  24SEP1962
    E0002  09JAN1981
    E0003  17JUL1945
    E0004  30NOV1975
    E0005  06AUG1976


  2. Write a program that uses the FORMAT procedure to combine the employee birthdates that are recorded in the list above with the first five observations of the Sasuser.Empdata data set. Create a temporary data set named Empdata_new2 to contain the combined observations. Create and use a format named $BIRTHDATE to record the lookup values in a character variable named Birthdate. Submit the program.

  3. Write a program that uses the ARRAY statement to combine the employee birthdates that are recorded in the list above with the first five observations of the Sasuser.Empdata data set. Create a temporary data set named Empdata_new3 to contain the combined observations. Create and use a temporary array named birthdates to record the lookup values in a character variable named Birthdate. Be sure to specify that the birthdates array is character and that the values in the array have a length of 9. Submit the program.
