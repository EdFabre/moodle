Create a summary data set, and combine it with detail data from the Sasuser.Contrib data set in one DATA step.

  1. Write a DATA step to create a new data set named Work.Diff2 that contains the overall average employee contribution as stored in Sasuser.Contrib. Diff2 should contain all information from Sasuser.Contrib, as well as a new variable named DiffAvg that is the difference between an individual contribution and the average employee contribution.

    In your DATA step, you will need to calculate the overall average employee contribution. Create a variable named AvgAmt to contain this average. Use the following RETAIN statement at the beginning of your DATA step to initialize the variable AvgAmt and retain its value as the DATA step iterates. 
         retain AvgAmt;
     Do not include missing values in AvgAmt. You will also need to find the difference between AvgAmt and each individual contribution. Also, do not include observations with a missing value for the contribution in Work.Diff2. Submit the program.

  2. Write a step to print the Work.Diff2 data set. Submit the program and examine the output.
