Use the Flight index to combine data from Sasuser.Flighttimes and Sasuser.Newtimes.

  1. Copy the following program and paste it into the code editing window. Submit the program to create an index named Flight on the Sasuser.Flighttimes data set. 

    proc datasets library=sasuser;
   modify flighttimes;
   index create flight;
quit;
  2. Write a DATA step that will create a new data set named Work.Newsched that will combine data from Sasuser.Newtimes and Sasuser.Flighttimes using the Flight index. Also in your DATA step, create a new variable named NewDepart that uses the TimeDiff variable in Newtimes to calculate the new departure time for the flight. In Work.Newsched, display both Depart and NewDepart in the TIME5. format.

    TimeDiff records how long a flight is delayed in minutes. Depart records the scheduled departure time for a flight as a SAS time value (the number of seconds since midnight).

    Add a step to print the output data set. Submit the program and examine the output as well as any messages that are written to the SAS log.

  3. If you find any observations with nonmatching data written to the log, revise the program to use _IORC_ to detect data errors. Write any nonmatching observations to a temporary data set named Errors. Add steps to print both output data sets. Submit the program and examine the output as well as any messages that are written in the log.
