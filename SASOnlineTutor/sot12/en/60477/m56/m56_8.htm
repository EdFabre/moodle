Working with Lookup Values Outside of SAS Data Sets (continued)
Select the statement below that is true.
If the lookup values are not stored in a SAS data set, you must use the IF-THEN/ELSE statement to perform a table lookup operation.
You can use either the IF-THEN/ELSE statement or the ARRAY statement to return multiple lookup values.
A user-defined format can provide easier maintenance than the IF-THEN/ELSE statement for lookup values that change often or that are used in many programs.
