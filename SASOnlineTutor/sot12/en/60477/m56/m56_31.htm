Create a summary data set, and combine it with detail data from the Sasuser.Contrib data set.

  1. Use PROC MEANS to create a data set named Work.Mean that contains the overall average employee contribution as stored in Sasuser.Contrib. Name the variable that holds this average value AvgAmt. Submit the program.

  2. Write a DATA step to create a new SAS data set named Work.Diff that contains all of the data from Sasuser.Contrib, as well as a new variable named DiffAvg.

    DiffAvg     should record the difference between each individual contribution and the overall average contribution (as recorded in Mean.AvgAmt), rounded to the nearest cent. Do not include the _Type_ and _Freq_ variables from the Mean data set in Work.Diff. Submit the program.

  3. Write a step that will print the Work.Diff data set that was created in the last step. Submit the program and examine the output.
