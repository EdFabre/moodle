Merge three SAS data sets that do not have a common BY variable. Combine Sasuser.Empdata, Sasuser.Jcodedat, and Sasuser.Newsals.

  1. Examine Sasuser.Empdata, Sasuser.Jcodedat, and Sasuser.Newsals in order to identify any common variables among them. You can either use the links at the top of this page to view more information about the data sets, or you can examine them in SAS.

    If there is not one variable that appears in all three data sets, you will need to pick two variables. Make sure that the variables that you choose appear in at least two of the three data sets. Also be sure that each of the three data sets contains at least one of the variables that you choose.

  2. Write a program that combines Sasuser.Empdata, Sasuser.Jcodedat, and Sasuser.Newsals into a new data set named Work.Jobdata. The new data set should contain variables that display employee ID numbers, employee job codes, job descriptions, current salaries, and new salaries.

    You should create temporary versions of your input data sets that are sorted by their common variables. Also, use the IN= option and a subsetting IF statement to ensure that only observations that contain data from all input data sets are included in the Work.Jobdata output data set. Submit the program.

  3. Write a step that will print the Work.Jobdata data set. Submit the program and examine the output.
