Query the table Sasuser.Frequentflyers using PROC SQL with an option to control output.

To view the expected PROC SQL output, click the related link in the text for each step.

    proc sql;
   select ffid, name, pointsused
      from sasuser.frequentflyers
      where membertype='GOLD' and pointsused>0
      order by pointsused;
quit;
