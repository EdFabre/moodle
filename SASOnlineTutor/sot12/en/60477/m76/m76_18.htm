Reset PROC SQL options without re-invoking the SQL procedure.

To view the expected PROC SQL output, click the related link in the text for each step.

  1. Copy and paste the following PROC SQL step into the code editing window:
    proc sql inobs=10;   
   select lastname, firstname, state     
      from sasuser.staffmaster    
      where state='NY';
  2. Modify the PROC SQL step by adding a statement to

    • reset the option specified in the PROC SQL statement to its default value

    • list row numbers in the query output.

  3. Modify the PROC SQL step by copying and pasting the following code at the end of the step:

       select lastname, firstname, state
      from sasuser.staffmaster
      where state='CT';
   quit;
  4. Submit this modified PROC SQL step and view the messages in the SAS log. Then view the output.
