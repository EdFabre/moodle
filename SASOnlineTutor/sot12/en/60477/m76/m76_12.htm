Query the table Sasuser.Marchflights using PROC SQL with options to control output.

To view the expected PROC SQL output, click the related link in the text for each step.

  1. Copy and paste the following PROC SQL query into the code editing window:

    proc sql;
   select flightnumber, date, destination,
          sum(boarded, transferred, nonrevenue)
          as Total
      from sasuser.marchflights
      where destination="LAX";
quit;
