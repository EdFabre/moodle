Specifying SQL Options

Remember that PROC SQL options are specified in the PROC SQL statement.


General form, PROC SQL statement:
PROC SQL <option(s)>;

where option(s) names the option(s) to be used.

After you specify an option, it remains in effect until you change it or you re-invoke PROC SQL.


The following tables list the options for controlling processing that are covered in this lesson. A complete description and an example of each option appears on the following pages.
