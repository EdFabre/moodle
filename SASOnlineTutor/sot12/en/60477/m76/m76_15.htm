Compare the timing information for two queries using the table Sasuser.Payrollmaster.

To view the expected PROC SQL output, click the related link in the text for each step.

  1. Copy and paste the following PROC SQL step into the code editing window:
    proc sql;
   select empid, jobcode, dateofbirth
      from sasuser.payrollmaster
      where jobcode in ('FA1','FA2')
            and dateofbirth < any          
               (select dateofbirth
                   from sasuser.payrollmaster
                   where jobcode='FA3');
   select empid, jobcode, dateofbirth      
      from sasuser.payrollmaster
      where jobcode in ('FA1','FA2')
            and dateofbirth < all
               (select dateofbirth
                   from sasuser.payrollmaster
                   where jobcode='FA3');   
quit;

  3. Submit the PROC SQL step and view the messages in the SAS log. Then view the output.
