Use the FILEVAR= option and SAS date functions to concatenate raw data files that contain records from the current year and the previous year.

  1. Create the SAS data set Work.Last2 by concatenating two raw data files. One file contains records from the current year. The other file contains records from the previous year.

    The files use the following naming convention: Yyyyy.dat, for example, Y2002.dat. The raw data files have the same file layout as Y2003.dat.

    Remember that you need to point to the physical location of the sample raw data files in your operating environment.

  2. Print the Work.Last2 data set and view the output.
  1. Create the SAS data set Work.Last2 by concatenating two raw data files. One file contains records from the current year. The other file contains records from the previous year.

    The files use the following naming convention: Yyyyy.dat, for example, Y2002.dat. The raw data files have the same file layout as Y2003.dat.

    Remember that you need to point to the physical location of the sample raw data files in your operating environment.

  2. Print the Work.Last2 data set and view the output.
  1. Submit one of the following programs: 
    data work.last2 (drop=thisyear lastyear);
   thisyear=year(today());
   lastyear=thisyear-1;
   do Year = thisyear, lastyear;
      nextfile="your-sasuser-location\y"
         !!put(year,4.)!!".dat";
      do until (lastobs);
         infile temp filevar=nextfile end=lastobs
            dlm=",";
         input Flight $ Date : date9. Depart $;
      output;
      end;
   end;
   stop;
run;
     
    data work.last2 (drop=thisyear lastyear);
   thisyear=year(today());
   lastyear=year(intnx('year',today(),-1));
   do Year = thisyear, lastyear;
      nextfile="your-sasuser-location\y"
         !!put(year,4.)!!".dat";
      do until (lastobs);
         infile temp filevar=nextfile end=lastobs
            dlm=",";
         input Flight $ Date : date9. Depart $;
      output;
      end;
   end;
   stop;
run;
  2. Submit the following PROC PRINT step: 
    proc print data=work.last2;
run;
