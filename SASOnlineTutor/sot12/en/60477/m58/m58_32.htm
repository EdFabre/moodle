Add codes to a custom format.

You must complete the previous practice before you can do this practice.

  1. Submit a PROC FORMAT step that writes the $JCODES. format out to the data set Sasuser.Jcodedata.

  2. First, close the Jcodedata data grid. Then, submit the following PROC SQL code to add three rows to the data set Sasuser.Jcodedata:
    proc sql;
   insert into sasuser.jcodedata
   set fmtname='$jcodes',
       start='TKTAG1',
       end='TKTAG1',
       label='Ticket Agent Grade 1'
   set fmtname='$jcodes',
       start='TKTAG2',
       end='TKTAG2',
       label='Ticket Agent Grade 2'
   set fmtname='$jcodes',
       start='TKTAG3', 
       end='TKTAG3',
       label='Ticket Agent Grade 3';
quit;
  3. Submit a PROC FORMAT step that reads in the data set Sasuser.Jcodedata and saves a new version of the $JCODES. format in the Sasuser library.

  4. Submit a PROC FORMAT step that documents the $JCODES. format. View the output and verify that the new values were added to the format.
