Create a custom format from the Sasuser.Jcodedat data set.

Be sure to complete this practice. You'll use the format you create here in the next practice.

  1. The Sasuser.Jcodedat data set has two variables, Jobcode and Descript. Write and submit a DATA step that uses this data set to create a data set named Newjcodedat. Be sure to

    • add the variable FmtName with the value $jcodes
    • rename the variable Jobcode to Start
    • rename the variable Descript to Label.

  2. Submit a PROC FORMAT step to read in the data set Newjcodedat and create the custom $JCODES. format. Save the format in the Sasuser library.

  3. Submit a PROC FORMAT step to document the new $JCODES. format. View the output.
