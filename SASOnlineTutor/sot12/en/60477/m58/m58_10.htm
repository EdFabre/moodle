Create a picture format and apply it to the Date variable in Sasuser.Cargo99.

You must complete this practice in order to do the next practice in this lesson.

  1. Write a PROC FORMAT step that creates the following temporary picture formats:

    • the picture format NEWDATE. that uses directives to format a date as mon yyyy (such as Jan 2003)
      • . Use the directive %b for the name of the month and %Y for the year.

    • the picture format CARGO. that labels weights less than or equal to 70000 as nn,nnn Below Capacity. Use nonzero digit selectors.

  2. Write a PROC PRINT step that lists the Sasuser.Cargo99 data set and applies the format NEWDATE. to the Date variable and the format CARGO. to the CargoWgt variable.

  3. Submit the program and view the output.
