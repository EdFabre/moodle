Manage formats using PROC CATALOG.

If you are using the SAS Learning Edition, you will not be able to submit a PROC CATALOG step because the CATALOG procedure is not included in the software. You can practice writing the code in this practice and compare your code to the solution code.

  1. Write and submit a PROC FORMAT step that documents the formats in the Work.Formats catalog.

  2. View the output. You should see the CARGO. and NEWDATE. formats that you created in the previous practice.

  3. Write and submit a PROC CATALOG step that copies only these two formats into the Sasuser.Formats catalog.

  4. Check the log to be sure the formats were copied correctly.
  1. Write the following PROC FORMAT step: 
    proc format lib=work fmtlib;
run;
  2. View the output. You should see the formats you created in the last practice.

  3. Submit the following PROC CATALOG step: 
    proc catalog catalog=work.formats;
   copy out=sasuser.formats;
   select newdate.format cargo.format;
run;
quit;
  4. Check the log to see if the formats were copied to the Sasuser.Formats catalog.
