Create and document a custom format, and then apply it to the Sasuser.Dnunder data set.

  1. Submit the following PROC COPY code to copy the data set Dnunder from the Sasuser library to the Work library:

    proc copy in=sasuser out=work;
   select dnunder;
run;
  2. Write and submit a PROC FORMAT step that creates a permanent character format named $REGIONS., whose values are listed below. Use a keyword to label all other values "Unknown." Save the format in the Sasuser library.


    Value
    Label
    0000102 Region1
    0000103 Region1
    0000104 Region2
    0000105 Region2
    0000106 Region3
    0000107 Region3
    0000108 Region4


  3. Write and submit a PROC PRINT step that lists the Sasuser.Dnunder data set and formats the variable RouteID with the $REGIONS. format. Don't forget to set a SAS system option so that SAS can locate the format.

  4. Write and submit a PROC DATASETS step that assigns the $REGIONS. format to the variable RouteID in the Work.Dnunder data set.

  5. Write and submit a PROC PRINT step that lists the Work.Dnunder data set. View the results.
