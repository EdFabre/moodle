Create a multilabel format.

  1. You have a data set that has a variable, Snowfall, with numeric values that reflect the amount of snow that has fallen overnight at a ski resort. Write a FORMAT procedure that creates a temporary format SKICOND. that labels the values as follows:


    Value
    Label
    0 - 4 Poor
    above 4 - 7 Fair
    above 7 - 10 Good
    above 10 Excellent
    missing values Check Data
    0 - 4 Lift 3 Closed
    above 4 All Lifts Open


  2. Submit the program and check the log to verify that the format was created.
