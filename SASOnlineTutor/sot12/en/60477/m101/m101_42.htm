Creating Queries That Accept Parameters (continued) Create a filter based on the parameter.
  1. In the Query Builder window, click the Filter Data tab.

  2. Select the CategoryID column from the Products table and drag and drop it on the filter area. The Edit Filter Condition window opens.

  3. In the Value box, type &category. For this filter condition, you are comparing the value in the CategoryID column to the value stored in the parameter category. Remember that the value is stored in a macro variable.The ampersand in &category signals that the variable is a macro variable.


Click to view the full window.


  1. Click OK. The filter condition is set.
