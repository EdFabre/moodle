Grouping and Filtering Data in a Query (continued) Open the Profit query and add and delete columns from the results.
  1. In the Project window, double-click the Profit query to open it.

  2. Click the Select and Sort tab.

  3. From the Products table, double-click the CategoryID column to add it to the query results.

  4. While holding down the Ctrl key, select ProductName, UnitCost, QUANTITY, and UnitPrice from the selected columns. Then click Delete. You should have only three columns in the query.
