Using Outer Joins (continued) Begin a query and join the Products and ProdCategories tables.
  1. Select FileOpenFrom My Computer. Navigate to the sample data directory or the directory where you stored the practice data you downloaded for this lesson. Select ProdCategories and click Open to add the data to the project.

  2. Select ToolsQueryCreate Empty.

  3. Click Add Data. In the Open Data window select Project.


  4. In the Add from Project window, press Ctrl and select Products and ProdCategories. Click OK. The two tables are joined automatically.

  5. Click the Tables tab. Notice that the tables are joined on the CategoryID column.
