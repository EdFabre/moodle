Working with Columns in a Query (continued) Add a column to compute the total dollar amount of each product in stock.
  1. On the Select and Sort tab, click New.


  2. In the Alias box, type Total Inventory.

  3. Click the Expression tab. Notice that at the top of the window, there is a white box for the expression that is used to create the new column. You can either type an expression in this box or construct an expression using the operator buttons and the Functions and Values tabs. For the Total Inventory column, you want the product of the values in the UnitPrice and UnitsInStock columns.

  4. Click the Values tab below the row of operator buttons.

  5. In the Data Source box, Products is selected. In the Value box, double-click UnitPrice.

  6. Click the multiplication operator .

  7. In the Value box, double-click UnitsInStock.


Click to view the full window.


  1. In the New window, click the Format tab.

  2. In the Categories box, select Currency.

  3. In the Formats box, select DOLLARw.d. This format displays values with a dollar sign to a particular width and number of decimal places, with commas and a period if necessary. As you see under Attributes, the default width is 6 and the default number of decimal places is 0.


Click to view the full window.


  1. Click OK to close the New window and create the computed column.

  1. On the Select and Sort tab, click New.


  2. In the Alias box, type Total Inventory.

    SAS Enterprise Guide uses the alias CC1 to indicate that the column you are creating is the first computed column in the query. When you type Total Inventory into the Alias box, you are renaming Enterprise Guide's default name. It also adds this column to the left side of the Query Builder under Computed Columns.

  3. In the Expression box, click Change. The Expression Builder opens. Notice that at the top of the window, there is a white box for the expression that is used to create the new column. You can either type an expression in this box or construct an expression using the operator buttons and the Functions and Values tabs. For the Total Inventory column, you want the product of the values in the UnitPrice and UnitsInStock columns.

  4. In the Expression Builder, click the Values tab, located below the row of operator buttons.

  5. In the Data Source box, Products is selected. In the Value box, double-click UnitPrice.

  6. Click the multiplication operator .

  7. In the Value box, double-click UnitsInStock.


Click to view the full window.


  1. Click OK. In the New window, notice that the expression has been added to the Expression box.

  2. In the Format box, click Change.

  3. In the Categories box, select Currency. In the Formats box, select DOLLARw.d. This format displays values with a dollar sign to a particular width and number of decimal places, with commas and a period if necessary. As you see under Attributes, the default width is 6 and the default number of decimal places is 0. Click OK.
