Perform a query on the Listings table.

  1. Begin a query on the Listings data.

    • Select ToolsQueryCreate Empty to open the query window.
    • Click Add Data. Navigate to the location where you saved the downloaded data and double-click Listings.sd2.
  2. Include the following columns (in order) in the query: STYLE, SQFEET, PRICE.

    • Click the Select and Sort tab.
    • Clear the check boxes next to PROPID, BEDROOMS, and BATHS.
  3. Create a calculated column named PricePerSqFoot using PRICE and SQFEET. Format the column using the DOLLARw.d currency format.

    • Click New.
    • In the Name box, type PricePerSqFoot.
    • Click the Expression tab, then click the Values tab.
    • In the Value box, double-click PRICE.
    • Click the division operator button ( / ).
    • In the Value box, double-click SQFEET.
    • Click the Format tab.
    • In the Categories box, select Currency.
    • In the Formats box, select DOLLARw.d.
    • Click OK to add the computed column to the query
  4. Sort the PricePerSqFoot column by descending values.

    • On the Select and Sort tab, click the sort box to the right of the PricePerSqFoot column.
    • Select Descending from the drop-down list.
  5. Run the query, then rename the query Practice1. What is the highest value of PricePerSqFoot?
    • Click Run Query. In the Project window, rename the query Practice1. The highest value of PricePerSqFoot is $63.

If you have not added the sample data to your project, return to the step-by-step page and add the data before you do this practice.

  1. Begin a query on the Listings data.

    • Select DataCreate New Empty Query to open the query window.
    • Click Add Data. In the Open Data window, click Project. In the Add from Project window, select Listings.sd2 and click OK.
  2. Include the following columns (in order) in the query: STYLE, SQFEET, PRICE.

    • Click the Select and Sort tab.
    • Clear the check boxes next to PROPID, BEDROOMS, and BATHS.
  3. Create a calculated column named PricePerSqFoot using PRICE and SQFEET. Format the column using the DOLLARw.d currency format.

    • Click New.
    • In the Alias box, type PricePerSqFoot.
    • In the Expression box, click Change. Then click the Values tab.
    • In the Value box, double-click PRICE.
    • Click the division operator button ( / ).
    • In the Value box, double-click SQFEET.
    • Click OK.
    • In the Format box, click Change.
    • In the Categories box, select Currency.
    • In the Formats box, select DOLLARw.d. Click OK.
  4. Sort the PricePerSqFoot column by descending values.

    • In the Sorting area of the New window, click the Direction drop-down list and select Descending.
    • Click the Priority drop-down box and select 1.
    • Click OK to add the computed column to the query.

  5. Run the query, then rename the query Practice1. What is the highest value of PricePerSqFoot?
    • Click Run Query. In the Process Flow window, rename the query Practice1. The highest value of PricePerSqFoot is $63.
