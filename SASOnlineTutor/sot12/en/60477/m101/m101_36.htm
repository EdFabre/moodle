Running Tasks against Query Output (continued) Lay out the table and select statistics.
If you prefer, you can view an animation of the following steps.
  1. In the Summary Tables for Profit window, click the Table tab.

  2. From the Available variables list, select _Country, then drag and drop it on the white box that represents a row in the table.


  3. Drag ProductName to the cell at the upper right of _Country. Drop ProductName when the arrow points toward the box for _Country.




    If you drop a variable in the wrong location, you can click the Undo last drop button ( ) or drag and drop the variable back in the Available variables list.

  1. Drag All below the row for _Country and drop it. This creates a row that has the sum of all product sales.




  1. Drag All between the two ProductName rows and drop it. This creates a row that has the sum of product sales for each value of _Country.




  1. Drag Profit to the left border of the N box and drop it.




  1. In the Available statistics box, select Mean and drag it to the bottom of the Profit box so that the area scrolls down to the Sum box. Drag Mean to the left border of the Sum box and drop it.
