Subsetting Data in a Query (continued) Add the Discontinued column and run the query.
  1. In the Project window, double-click the Products to Order item to reopen the query for editing.

  2. Click the Select and Sort tab.

  3. In the columns list, double-click Discontinued to add it to the query.

  4. Select the Discontinued column and then click Move Up until the column is listed first in the query.


Click to view the full window.


  1. Click Run Query. When you are prompted about replacing the results of the last query, click Yes.

  2. Take a look at the results of the query. The Discontinued column shows that the items that have a reorder level of 0 have a discontinued value of 1. These items have been discontinued. There is only one item that needs to be reordered at this time.
