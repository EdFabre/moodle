Join the ListingSales and Listings tables. Replace values in the SQFEET column to show houses that qualify for a sales contest. Then create a report to find the winner of the contest.

This practice uses the Practice3 query that was created in the last practice. If you haven't created this query, return to the practice and create it before continuing here.

  1. Reopen Practice3.

    • In the Project window, double-click Practice3 to open the query.
  2. Add the ListingSales table. Join the PROPID column in the Listings table to the PropertyID column in the ListingSales table.

    • Click Add Data. Navigate to the directory where you stored the downloaded data. Double-click ListingSales. Click OK when you are prompted to join the tables manually.
    • In the Listings table, click PROPID and drag the join line to the PropertyID column in the ListingSales table.
  3. Include the following columns (in order) in the query output: AGENT, SalesPrice, SaleDate, SQFEET.

    • Click the Select and Sort tab.
    • While holding down the Ctrl key, select STYLE, PRICE, and PropertyID, then click Delete.
    • Select SQFEET and click Move Down until the column is at the bottom of the list.
  4. For this contest, only sales of properties greater than 1300 square feet qualify. In the SQFEET column, replace values above 1300 with Qualifies and replace all other values with DNQ. Don't forget to click Add to List after you replace values above 1300.

    • Select the SQFEET column, and click Properties.
    • In the Properties window, click Replace Values.
    • Select Replace a range of values.
    • Click the Set a lower limit drop-down box and select 1300. Clear the Set an upper limit check box. In the Replace with box, type Qualifies.
    • Click Add to List.
    • In the Replace all others with box, type DNQ.
    • Click OK.
  5. Set a filter so that the results list only sales that qualify for the contest.

    In real life you would simply create a filter so that only properties > 1300 sq.ft. are included in the output. For this practice, filter the replaced values in SQFEET.


    • Click Filter Data.
    • In the Computed Columns list, drag and drop _SQFEET in the filter area.
    • In the Edit Filter Condition window, type "Qualifies" (with quotation marks) in the Value box.
    • Click OK.
  6. Set the results format to create a data table.

    • Click More Query Options.
    • In the Result set format area, click Data table.
  7. Run the query and do not replace the results of the previous query. View the data.

    • Click Run Query. In the prompt window, click No.
    • Only sales that qualify for the contest are listed in the data.

  8. Rename the query Practice4.

  9. The winner of the contest is the agent with the highest total sales of qualifying houses. Using the query as input, create a summary table that shows the SalesPrice total for each agent. Who won the contest?

    • Select AnalysisDescriptiveSummary Tables.
    • Assign SalesPrice to the Analysis Variables role and AGENT to the Classification variables role.
    • Click the Tables tab.
    • Drag Agent to the row box. Drag SalesPrice to the column box. In the Available statistics box, select Sum and drag it below SalesPrice. Your model should look like this:



    • Click Finish.
    • Len Mizusawa won the contest with $280,000 in sales of qualifying houses.
