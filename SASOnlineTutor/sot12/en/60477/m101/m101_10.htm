Working with Columns in a Query (continued) Select columns for the query results.
  1. In the Query Builder, click the Select and Sort tab.


  2. By default, all columns from the table are included in the query results. Because you are interested in product inventory, you only need to keep the following columns:

    • ProductName
    • UnitsInStock
    • UnitsOnOrder
    • ReorderLevel.

    Remove each unwanted column by clearing the check box next to that column.
