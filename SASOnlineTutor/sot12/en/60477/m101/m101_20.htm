Add a filter to include only homes that are greater than 1300 square feet in the query output.

This practice uses the Practice1 query that was created in the last practice. If you haven't created this query, return to the practice and create it before continuing here.

  1. Reopen Practice1.

    • In the Project window, double-click Practice1 to open it.
  2. Add a filter to include only rows in which the value of SQFEET is greater than or equal to 1300.

    • On the Filter Data tab, drag and drop the SQFEET column in the filter area.
    • In the Edit Filter Condition window, click the Operator drop-down box and select >=, greater than or equal to.
    • In the Value box, type 1300.
    • Click OK.
  3. Remove the sort from the PricePerSqFoot column. Then, sort first by STYLE in ascending order and second by SQFEET in descending order.

    • Click the Select and Sort tab.
    • Click the sort box to the right of PricePerSqFoot and select <none> from the drop-down list.
    • Click the sort box to the right of STYLE and select Ascending from the drop-down list.
    • Click the sort box to the right of SQFEET and select Descending from the drop-down list.
  4. Run the query and do not replace the results from the last query.

    • Click Run Query. When you are prompted to replace the previous results, click No.
  5. Which style of home has the greatest number of listings? Which style home has the largest square footage?

    • The Condo style has the greatest number of listings. Also, the largest home is a condo.
  6. Rename the query Practice2.

    • In the Project window, rename the new query Practice2.
