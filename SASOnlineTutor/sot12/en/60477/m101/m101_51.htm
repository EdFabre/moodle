Grouping and Filtering Data in a Query (continued)

Filtering Grouped Data

At this point, your query displays a group for each category of international products, along with the total profit for that category. But now suppose that you want to see only groups for which profit is greater than or equal to $4000.

You already set a filter to select only rows for international products.




To select groups based on profit, you can now set a group filter. The group filter applies only to groups, not to individual rows of data. When you set the group filter, your query selects only categories where profit is greater than or equal to $4000.
