Grouping and Filtering Data in a Query (continued) Filter by _Country to show only records for International products.
  1. Click the Filter Data tab.

  2. From the Computed Columns list, select _Country and drag it to the filter area.
    3. The Edit Filter Condition window opens.

  3. In the Value box, type "International" (be sure to use quotation marks). Then click OK.


Click to view the full window.


Next, run the query and examine the results.
