Creating Queries That Accept Parameters (continued) Define a parameter in the Profit query.
  1. In the Project window, double-click the Profit query to open it.


  2. Click the Parameters tab, then click New.

  3. Under Type of parameter, click the drop-down list and select Item from a list. This creates a selection list for the user based on the values of the variable you choose. (Note that other types of parameters allow users to enter values or even select multiple values.)

  4. In the Name box, type category.

  5. In the Message for user box, type Select a product category:.

  6. Under Enclose user input with, click the drop-down list and select Parentheses (Numeric style). The selections in this list determine what, if any, characters are used to enclose the user input in the query code that is generated when you run a query. Since the values for your parameter are numeric, you use the numeric style.


Click to view the full window.


  1. Under Parameter validation, click Fill List with Actual Data Values.

  2. In the Load Distinct Values From Column window, select CategoryID from the Products table. Click OK.




  1. The values 1–8 are listed in the Parameter validation box. Click OK. You have created a parameter to prompt the user for a selection and then store the value. The new parameter is listed on the Parameters tab.
