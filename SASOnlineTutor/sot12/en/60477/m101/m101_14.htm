Working with Columns in a Query (continued) Run the query and examine the results. Then rename the query.
  1. In the Query Builder window, click Run Query.


  2. The query results open in an HTML report. Notice that only the columns you selected appear in the report. You also have two new columns that are formatted using the currency format that you selected. The values in the Total Inventory column are sorted in descending order. The original Products data is unchanged.


Click to view the full window.


  1. In the Project window, right-click Query1 and select Rename from the shortcut menu. Name the query Product Inventory.

  2. Close the query results.
