Create a query that groups the rows by Agent and sums rows by SalesPrice. Then set a group filter to select only rows in which total sales is greater than $200,000.

This practice uses the Practice4 query that was created in a previous practice. If you haven't created this query, return to the practice and create it before continuing here.

  1. Reopen Practice4.

    • In the Project window, double-click Practice4.
  2. Remove the filter from the query.

    • Click the Filter Data tab.
    • Right-click the filter and select Delete.
  3. Add a Group by function to the Agent column and a Summary function to the SalesPrice column. Delete the other columns.

    • Click the Select and Sort tab.
    • Click the box to the left of AGENT and select <group> from the drop-down list.
    • Click the box to the left of SalesPrice and select SUM from the drop-down list.
    • Delete the SalesDate and _SQFEET columns.
  4. Set a group filter on the calculated Sum column to show only agents who have total sales greater than $200,000.

    • Click the Group Filters tab.
    • In the Computed Columns list, drag sum_of_SalesPrice to the filter area.
    • Click the Operator drop-down list and select >, greater than.
    • In the Value box, type 200000.
    • Click OK.
  5. Run the query and do not replace the previous results. Examine the data. Which two salespeople have sales greater than $200,000?

    • Click Run Query. In the prompt window, click No.
    • Len Mizusawa and Arthur Roy have total sales greater than $200,000.
  6. Rename the query Practice6.
