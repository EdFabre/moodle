Starting a Query (continued) Start a query on the Products table.
  1. Select ToolsQueryCreate Empty. The Query Builder opens.

    To begin a query on data that is already in your project, select the data in the Project window, and then select ToolsQueryCreate from Active Data.

  2. Click Add Data. In the Add Data window, click the Existing tab. In the sample data directory, double-click Products.sas7bdat. Notice that the columns in the Products table are listed on the left side of the Query Builder.
