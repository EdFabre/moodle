Grouping and Filtering Data in a Query (continued) Open the query and specify a group filter for the Profit column.
  1. In the Project window, double-click Group-Profit to open the query.

  2. Click the Group Filters tab.

  3. In the Computed Columns list, select Profit and drag it to the filter area.
    4. The Edit Filter Condition window opens.

  4. Click the Operator drop-down list and select >, greater than.

  5. In the Value box, type 4000. Then click OK.


Click to view the full window.
Next, run the query and examine the results.
